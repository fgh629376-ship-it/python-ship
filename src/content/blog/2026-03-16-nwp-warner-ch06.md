## 一、这章为什么重要？

前五章建立了 NWP 的物理和数学框架：方程（Ch2）→ 数值解法（Ch3）→ 参数化（Ch4）→ 地表（Ch5）。但再好的模型，初始条件错了就全废。

Ch6 解决的问题是：**如何从稀疏、有噪声、时空不均匀的观测中，定义一个尽可能接近真实大气的初始状态？**

这不是简单的插值——初始条件必须满足动力学约束（近似地转平衡），否则模型启动后会产生虚假的惯性重力波，前 12 小时的预报不可用。

## 二、核心数学：从标量到向量

### 标量最优估计

两个带误差的温度估计：背景 $T_b$（方差 $\sigma_b^2$）和观测 $T_o$（方差 $\sigma_o^2$）。最优线性组合：

$$T_a = T_b + k(T_o - T_b), \quad k = \frac{\sigma_b^2}{\sigma_b^2 + \sigma_o^2}$$

- 背景误差大 → $k$ 大 → 更信任观测
- 观测误差大 → $k$ 小 → 更信任背景

**分析精度 = 观测精度 + 背景精度**（精度 = 方差的倒数）。这意味着分析永远比两个输入都好。

### 向量推广

$$\mathbf{x}_a = \mathbf{x}_b + \mathbf{K}(\mathbf{y} - H(\mathbf{x}_b))$$

$$\mathbf{K} = \mathbf{B}\mathbf{H}^T(\mathbf{H}\mathbf{B}\mathbf{H}^T + \mathbf{R})^{-1}$$

**B 矩阵**（背景误差协方差）是关键中的关键——它决定了一个观测对周围区域的影响范围和形状。

## 三、同化方法的演进

| 方法 | B 矩阵 | 时间 | 伴随 | 特点 |
|------|--------|------|------|------|
| SC (Cressman/Barnes) | 简单距离权重 | 无 | 否 | 最简单，各向同性 |
| OI | 预计算，静态 | 无 | 否 | 用 B 但只选附近观测 |
| 3DVAR | NMC 法，静态 | 无 | 否 | 用所有观测，可同化卫星辐射 |
| 4DVAR | 隐式演化 | 有 | 是 | ECMWF 运营使用 |
| EnKF | 集合估计，动态 | 有 | 否 | regime-dependent |
| 混合 | $(1-\alpha)\mathbf{P}_f + \alpha\mathbf{B}$ | 有 | 否 | 最优 $\alpha \in [0.1, 0.4]$ |

**关键认识**：从 SC 到混合方法，本质上都是在解同一个问题——**如何让 B 矩阵更真实**。静态 B 不随天气型变化（晴天和暴风雨用同一个 B），这是 3DVAR 的根本缺陷。EnKF 用集合估计让 B 随流而变。4DVAR 通过时间窗口隐式实现同样的效果。

## 四、实践中的关键问题

### Spin-up 问题

模型需要时间发展锋面梯度、对流、云——这些无法直接写进初始条件。冷启动 vs 暖启动 vs 热启动，各有场景。

### 谱 nudging

传统 nudging 会破坏模型已发展的细尺度特征。谱 nudging 的巧妙之处：只修正大尺度（如波数 < 3 的分量），让模型自由发展细尺度特征。

### "Banana function"

在弯曲气流中（如锋面），一个观测对周围的影响范围应该是弯曲的椭圆——沿气流拉伸，垂直气流压缩。各向同性的 B 矩阵做不到这一点。

### 自适应观测

飓风预报中用伴随方法/ETKF 确定对预报最敏感的区域，然后定向投放探空仪。少量精准放置的观测 >> 大量随机分布的观测。

## 五、对光伏预测的启示

NWP 辐照度预报质量 = 初始条件质量 × 模型质量。

- **云场初始化**是短期（<6h）辐照度预报的关键——如果初始条件中的云位置错了，前几小时就废了
- 卫星辐射的直接同化（不需要先反演成温度/湿度）是 3DVAR 相对于 OI 的重要进步
- ERA5 的辐照度数据 = 同化系统的产物，辐射本身通常不被同化 → 有系统偏差
