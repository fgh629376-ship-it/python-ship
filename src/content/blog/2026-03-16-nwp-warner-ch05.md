## 一、这章为什么重要？

Ch4 建立了参数化的循环系统：**辐射→地表→PBL→对流→微物理→云→辐射**。

Ch5 聚焦这个循环的**起点**——地表过程。地表能量平衡决定了多少能量进入大气（感热 H）、多少水汽进入大气（潜热 LE）、多少热量存入土壤（G）。这三个分量的比例直接控制边界层发展和对流触发。

**对光伏的直接意义**：pvlib 需要的输入之一是反照率（albedo），而反照率不是常数——它取决于土壤湿度、植被类型、太阳角度。如果 NWP 的地表模型用错了反照率，辐照度预测就会有系统偏差。

## 二、地表能量平衡：一切的起点

### 核心方程

$$R = LE + H + G \quad (5.1)$$

- $R$ = 净辐射（所有辐射收支之和）
- $LE$ = 潜热通量（蒸发/蒸腾消耗的能量）
- $H$ = 感热通量（加热大气的能量）
- $G$ = 地热通量（传导入土壤的能量）

### 净辐射的分解

$$R = (Q + q)(1 - \alpha) - I_{\uparrow} + I_{\downarrow} \quad (5.2)$$

- $Q$ = 直射太阳辐射，$q$ = 散射太阳辐射
- $\alpha$ = 地表反照率
- $I_{\uparrow}$ = 地表发射的长波辐射（$= \varepsilon\sigma T_g^4$）
- $I_{\downarrow}$ = 大气下行长波辐射

**关键认识**：这个方程把 Ch4 的辐射参数化（提供 $Q, q, I_{\downarrow}$）和地表参数化连接起来。辐射方案的输出是地表模型的输入。

### 水平衡方程

$$\frac{\partial\theta}{\partial t} = P - ET - RO - D \quad (5.3)$$

$\theta$ 是体积含水量，$P$ 降水输入，$ET$ 蒸散发损失，$RO$ 径流，$D$ 深层渗漏。

## 三、土壤热传导：温度波的传播

### 热扩散方程

$$\frac{\partial T_s}{\partial t} = \frac{k_s}{C_s}\frac{\partial^2 T_s}{\partial z^2} \quad (5.6)$$

热扩散率 $\kappa = k_s/C_s$ 决定温度波传播速度：
- $k_s$ = 热传导率（湿土 >> 干土 >> 空气）
- $C_s$ = 热容量（湿土 > 干土）

### 温度波随深度衰减

$$(\Delta T)_z = (\Delta T)_0 \exp\left(-z\sqrt{\frac{\pi}{\kappa P}}\right) \quad (5.7)$$

- 日变化波穿透约 0.5m
- 年变化波穿透约 7m（$\sqrt{365} \approx 19$ 倍深度）

**对光伏的意义**：土壤热惯量（thermal admittance $\mu_s = \sqrt{k_s C_s}$）决定地表温度的日变化幅度。高热惯量（湿土）→ 地表温度变化小 → 感热通量小 → 边界层发展弱 → 对流不易触发。这就是为什么灌溉区和干旱区上方的云有显著差异。

## 四、蒸散发：连接水循环和能量循环

### 两种表面通量表达

感热：$H = \rho C_p D_H (T_g - T_a) \quad (5.14)$

潜热：$LE = \rho L D_W (q_{sat}(T_g) - q_a) \quad (5.15)$

$D_H, D_W$ 是传输系数，取决于：
- **地表粗糙度**（森林 >> 草地 >> 水面）
- **风切变**（风速越大 → 湍流越强 → 传输越快）
- **稳定度**（不稳定 → 浮力湍流 → 传输增强）

### 植被蒸腾的特殊性

植被通过根系从深层土壤吸水 → 通过木质部输送到叶片 → 通过气孔释放水汽。这个过程：
- 消耗的潜热来自**叶面**（不是土壤表面）
- 受土壤含水量控制（低于萎蔫点 → 蒸腾停止）
- 受气孔开闭控制（干旱/高温 → 气孔关闭 → 蒸腾减少 → 感热增大）

**对光伏的连锁反应**：干旱 → 气孔关闭 → 蒸腾↓ → 感热↑ → 边界层更深 → 对流更强 → 午后积云更多 → GHI 下午 ramp down 更剧烈

## 五、陆面模型（LSM）：工程实现

### 层级结构

23 个 LSM 参加了 PILPS（陆面参数化方案对比项目）。关键发现：**复杂的 LSM 并不总是优于简单的**——因为复杂模型需要大量难以准确获取的参数。

### Noah LSM（WRF 默认）

- 4 层土壤（10cm, 30cm, 60cm, 100cm）
- 求解方程：5.1（能量平衡）+ 5.3（水平衡）+ 5.6（热扩散）+ 5.9（水分扩散）+ 5.14/5.15（表面通量）
- 皮温 $T_g$ 通过迭代求解

### 陆面数据同化系统（LDAS）

独立运行 LSM，用观测数据（而非模型输出）驱动，"预热"土壤温湿度场。需要数月到数年的 spin-up 才能达到平衡。

**工程教训**：LDAS 和大气模型必须用**同一个 LSM**。不同 LSM 之间的土壤状态转换几乎不可能准确——不同的假设、不同的参数化、不同的网格。

## 六、其他地表过程

### 海洋过程
- 短期预报（<1-2周）：SST 可假设不变
- 例外：飓风——强风搅拌使 SST 骤降，必须耦合海洋模型
- 长期/气候预测：必须耦合海洋环流+海冰模型

### 地形强迫
- 平滑地形会降低山脉高度 → 低估山脉阻挡和重力波拖曳
- 包络地形（envelope topography）：用"山峰高度"而非"网格平均高度"
- 分辨率影响巨大：30km vs 3.3km 网格的地形差异触目惊心

### 城市冠层模型（UCM）
- 简单方法：调整 LSM 参数（粗糙度↑、反照率↓、热容量↑、透水面积↓）
- 复杂方法：UCM 参数化建筑几何（街道峡谷辐射捕获、阴影效应）
- 城市热岛效应对光伏：城市上方边界层更深 → 对流更强 → 云模式不同

## 八、验证 Ch4 的假设

Ch4 中我提出的假设 1：**"地表参数化的误差贡献有多大？"**

Warner 的回答（部分验证）：
- PILPS 23 个 LSM 对比显示，**复杂模型不一定更好**——参数不确定性可能比模型复杂度更重要
- 土壤含水量对热传导率的影响是**两个数量级**（干土 vs 湿土）——含水量误差会被放大
- LDAS 需要数月 spin-up，说明土壤状态的初始误差**衰减极慢**

结论：地表确实是被低估的误差源，但不如云那么致命。因为地表过程的时间尺度（小时-天）远慢于云（分钟-小时），所以误差传播较慢。
