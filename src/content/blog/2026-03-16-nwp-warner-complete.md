## 一、全书知识链路

Warner 全书可以浓缩为一条**从物理到工程的链路**：

```
物理定律（原始方程 7 个）
    ↓ 离散化
数值方法（有限差分/谱方法 + CFL 约束）
    ↓ 闭合
参数化（云微物理 + 对流 + PBL + 辐射 + 地表）
    ↓ 初始化
数据同化（OI → 3DVAR → 4DVAR/EnKF）
    ↓ 不确定性量化
集成方法（初始条件扰动 + 模型不确定性 + 多模型）
    ↓ 验证
技巧评估（偏差/RMSE/AC + 概率验证 + 对象验证）
    ↓ 应用
业务预报 + 后处理(MOS) + 耦合应用(空气质量/能源/疾病)
```

**关键认识**：NWP 不是"解方程"——它是一个**工程系统**，每一环都有误差，每一环都影响下一环。

## 二、六大模块的核心收获

### 模块一：数学基础（Ch2-3）

**Ch2 控制方程**：7 个原始方程（3 动量 + 连续 + 热力学 + 水汽 + 状态方程）。Reynolds 平均引入闭合问题——这是所有参数化的数学起源。静力近似将垂直动量方程简化为诊断方程，消除声波但限制了模型能力。

**Ch3 数值方法**：CFL 条件 $\Delta t \leq \Delta x / c$ 是高分辨率 NWP 计算量爆炸的根本原因。1km 网格 → 3s 时间步 → 一天需要 28800 步。半隐式方法放宽 CFL 但牺牲精度。数值色散（短波太慢）和数值耗散（短波被抹平）是所有有限差分格式的固有缺陷。

### 模块二：物理表示（Ch4-5）

**Ch4 参数化**——全书最核心的章节。参数化不是独立模块，而是**循环系统**：辐射→地表→PBL→对流→微物理→云→辐射。辐射方案物理已成熟，瓶颈在输入（云的三维分布）。灰色地带（3-9km）的对流参数化可能"双重计算"。

**Ch5 地表过程**：$R = LE + H + G$ 是循环系统的起点。Bowen ratio 决定能量去向——干旱地感热↑→边界层深→对流强→积云多→GHI 下降。PILPS 的教训：复杂 LSM 不一定比简单 LSM 好（参数不确定性 > 模型复杂度）。

### 模块三：初始化与不确定性（Ch6-8）

**Ch6 数据同化**——全书数学密度最高的章节。核心公式：$x_a = x_b + K(y - H(x_b))$。

| 方法 | B 矩阵 | 时间维度 | 需要伴随 | 运营使用 |
|------|--------|---------|---------|---------|
| OI | 预计算，静态 | 无 | 否 | 早期 |
| 3DVAR | NMC法，静态 | 无 | 否 | 广泛 |
| 4DVAR | 隐式演化 | 有 | 是 | ECMWF |
| EnKF | 集合估计，动态 | 有 | 否 | 实验→运营 |

**Ch7 集成方法**：集成平均优于单一预报（非线性滤波效应）。离散度↔不确定性（需校准）。初始条件扰动三法：EnKF、Bred vectors、奇异向量。随机参数化（Ch4 的延伸）增大离散度。

**Ch8 可预测性**：Lorenz 蝴蝶效应。误差增长三阶段。地表强迫提供"免费"可预测性（日变化/季节变化的确定性成分）。阻塞事件→可预测性骤降。

### 模块四：评估（Ch9-11）

**Ch9 验证方法**：RMSE/Bias/AC 是基础。概率验证：可靠性图、rank histogram、Brier 评分。对象验证（MODE/SAL）解决"双惩罚"问题。有效分辨率 > 2Δx。

**Ch10 实验设计**：OSSE 评估新观测系统价值。伴随敏感性定义最重要的初始条件区域。Reforecasts 提供大样本验证。

**Ch11 输出分析**：EOF/PCA 降维提取主要变率模态。SOM 自动分类天气型。这些方法直接适用于辐照度时空分析。

### 模块五：运营（Ch12-13）

**Ch12 业务 NWP**：ECMWF IFS 是全球金标准（~9km）。HRRR 逐小时更新（3km）——最适合光伏日内预测。MOS/PPM 是后处理标配。

**Ch13 后处理**：BMA/EMOS 将集成输出转化为校准概率分布。对光伏：Yang 教材 Ch8 的方法与此完全对应。

### 模块六：扩展（Ch14-16）

**Ch14 耦合应用**：NWP 耦合空气质量、风能/太阳能、传染病预测。WRF-Solar = NWP + 辐射模型 → GHI/DNI 预报。

**Ch15 CFD**：DNS/LES 的计算极限。"terra incognita"（100m-1km）与 Ch4 灰色地带呼应。

**Ch16 气候建模**：AOGCM + CMIP 多模型集成。ERA5 再分析是辐照度长期数据的重要来源。气候降尺度为区域光伏规划提供高分辨率数据。

## 三、对光伏预测的核心启示

学完全书，对光伏功率预测框架的理解可以总结为：

### 1. NWP 辐照度预报的误差来源

```
初始条件误差（Ch6）
    + 云微物理误差（Ch4）← 最大贡献
    + 气溶胶不确定性（Ch4）
    + 地表反照率误差（Ch5）
    + 数值色散/耗散（Ch3）
    + 分辨率限制（Ch3 CFL + Ch4 灰色地带）
    = NWP GHI 预报误差
```

### 2. 后处理是必须的

NWP 输出不能直接用于光伏预测——系统偏差、分辨率不足、概率信息缺失。MOS/BMA/EMOS（Ch12-13）+ Yang 教材 Ch8 的统计方法 → 校准后的概率预报。

### 3. 集成优于单一

单一确定性预报不可靠。集成方法（Ch7）+ 概率校准（Ch13）→ 可靠的不确定性估计 → 电网调度决策支持。

### 4. 时间尺度决定方法

| 时间尺度 | 最优方法 | Warner 对应 |
|---------|---------|-----------|
| 0-6h | 统计/遥感 | Ch8 亚日可预测性 |
| 6-72h | NWP + MOS | Ch12 + Ch13 |
| 1-2周 | 集成 NWP | Ch7 |
| 季节 | 气候模型 | Ch16 |
