## II. 因果推論チェーン

### 推論チェーン 1：なぜ NWP は太陽光発電の前日予測における唯一の信頼できる情報源なのか？

```
前提 1（Yang Ch4.5.4）：日射量の時空間依存は移動する雲からすべて生じる
    ↓
前提 2（Yang Ch4.5.4）：全天カメラは雲が「凍結」していると仮定 → 30 分未満で破綻
前提 3（Yang Ch4.5.4）：衛星 CMV は雲の等速直線運動を仮定 → 4 時間未満で破綻
    ↓
前提 4（Warner Ch2）：雲の力学的発展を解けるのは原始方程式のみ
    ↓
前提 5（Yang Ch5.6.2）：電力先物市場（DAM）は 36 時間超のリードタイムが必要
    ↓
結論：NWP は前日予測の唯一の信頼できる情報源
```

この推論チェーンは Warner Ch2 の原始方程式から出発し、カメラと衛星が前日スケールで破綻する理由を説明する——物理的仮定（凍結場・等速移流）が長時間後には成立しなくなるためであり、NWP の原始方程式だけが雲の生成・発達・消散を模擬できる。

### 推論チェーン 2：NWP 予報誤差はどこから来るのか？

```
Warner Ch2 原始方程式 → 7 変数を解く必要がある
    ↓
問題 1：方程式が非線形結合 → 解析解なし → 離散化が必須（Ch3）
    → 離散化が打ち切り誤差を生む
    ↓
問題 2：サブグリッド過程（乱流・雲微物理）は陽的に解けない
    → パラメタリゼーションが必須（Ch4-5）
    → パラメタリゼーションは真の物理の近似 → パラメタリゼーション誤差を生む
    ↓
問題 3：初期条件は限られた観測から得られる
    → 観測には計測誤差 + 空間被覆の不足がある
    → 不完全な初期条件 → カオス的誤差増幅（Ch8）
    ↓
総誤差 = 打ち切り誤差 + パラメタリゼーション誤差 + 初期条件誤差
```

**Yang 教材との接続**：Yang Ch3.1 は「予測誤差の発生源は二つ——不完全な初期条件と不完全な力学法則」と述べる。Warner Ch2 はさらに「不完全な力学法則」の具体的な意味を明らかにする——方程式自体に問題があるわけではなく（原始方程式は厳密）、**解法**が誤差を生む（離散化＋パラメタリゼーション）ということだ。

### 推論チェーン 3：Reynolds 平均はどのように日射量予測と結びつくのか？

```
Warner Ch2：大気にはサブグリッド乱流がある → 直接解析できない
    ↓
Reynolds 分解：u = ū + u' → 平均をとると共分散項 -ρ(u'w') が現れる
    ↓
共分散項 = Reynolds 応力 = 乱流が平均流に与える影響
    ↓
パラメタリゼーションが必須（Ch4）→ 境界層（PBL）パラメタリゼーション方式
    ↓
PBL 方式が鉛直混合強度を決定する
    ↓
分岐 A：強い混合 → 水蒸気が上昇 → 雲生成 → GHI 減少
分岐 B：弱い混合 → 安定成層 → 雲が少ない → GHI 増加
    ↓
Yang Ch7.1.2：PBL 方式は WRF-Solar で最も感度が高い選択の一つ
```

**核心的洞察**：Reynolds 平均という純粋な数学的操作から、4 ステップの推論を経て、太陽光発電の日射量予測精度に直結する。「知識は孤島ではない」ことの体現だ——1895 年の乱流分解手法が、2024 年の太陽光発電予測品質を左右している。

### 推論チェーン 4：静水圧近似の選択が太陽光発電予測に与える影響（Ch2）

```
Warner Ch2.3.1：静水圧近似が音波をフィルタリング → より大きな時間ステップが可能 → 計算が高速化
    ↓
ただし：Δx < 10 km では静水圧近似が破綻する
    ↓
GFS（全球モデル，~13 km）→ 静水圧近似が使える → 粗い解像度だが全球カバレッジ
HRRR（領域モデル，3 km）→ 非静水圧方程式が必須 → 高解像度だが CONUS のみ
    ↓
Yang Ch6.6 の比較：HRRR の日射量予測は GFS を上回る
    ↓
理由の一つ：非静水圧方程式は対流セルを解像できる → より正確な雲の模擬
    ↓
しかし HRRR はアメリカのみをカバー → 他地域では GFS または ECMWF しか選択肢がない
    ↓
WRF-Solar の価値：ユーザーが自ら非静水圧領域モデルを実行できる
    ただし気象学の知識が必要（Yang Ch4.6.1 の学際的挑戦）
```

**実際の意思決定における意義**：アメリカで太陽光発電予測をするなら HRRR（3 km 非静水圧）を使う。中国では GFS/ECMWF（>10 km 静水圧）か自分で WRF-Solar を走らせるしかない。この意思決定の理論的根拠が Warner Ch2.3 の静水圧近似だ。

### 推論チェーン 5：解像度を 2 倍にするとなぜコストが 8 倍になるのか？（Ch3 追加）

```
CFL 条件：U·Δt/Δx < 1
    ↓
Δx を半分にする → Δt も半分にしなければならない（でないと CFL 違反）
    ↓
2D グリッド点数 ×4（面積一定，各次元 ×2）
    ↓
総計算量 = グリッド点数 × 時間ステップ数 = 4 × 2 = 8
    ↓
実際にはさらに多い：鉛直層も細分化される可能性 → ×16 以上
```

**太陽光発電予測への意思決定的意義**：RTX 4050 で WRF-Solar を実行する場合，9 km → 3 km には約 27 倍の計算量が必要だ。精度向上がそれに見合うかどうかを慎重に検討しなければならない。

### 推論チェーン 6：打ち切り誤差はどのように日射量まで伝播するのか？（Ch3 追加）

```
有限差分による微分計算 → 打ち切り誤差（短波 6Δx で 17% 誤差）
    ↓
気圧傾度力が不正確 → 地衡風のバイアス
    ↓
発散が不正確 → 鉛直運動のバイアス
    ↓
鉛直運動が凝結・蒸発を駆動 → 雲液水量のバイアス
    ↓
放射方式が雲水を用いて日射量を計算 → GHI 出力バイアス
```

**核心的洞察**：日射量予測誤差は放射パラメタリゼーションだけから来るわけではない——最下層の微分近似の段階から誤差が積み上がる。だから Yang が「雲微物理がボトルネック」と言いつつ Warner が「数値手法も重要」と強調する——両者は加法関係ではなく乗法関係だ。

### 推論チェーン 7：なぜ「雲が日射量予測の最大のボトルネック」なのか？（Ch4 追加）

```
放射パラメタリゼーションの物理はすでに成熟している（Beer の法則 + Planck 関数）
    ↓
放射方式への入力 = 雲の三次元分布と光学特性
    ↓
雲の光学特性 ← 雲微物理（雲粒径分布）
    ↓
雲の存在 ← 対流パラメタリゼーション（いつどこで凝結が起きるか）
    ↓
対流への水蒸気供給 ← PBL パラメタリゼーション（鉛直混合強度）
    ↓
PBL の熱力学的駆動 ← 地表エネルギー収支（日射量の関数！）
    ↓
結論：日射量予測誤差の根本は放射方式自体ではなく，
「放射→地表→PBL→対流→微物理→雲→放射」の循環系全体にある
```

**核心的洞察**：Yang は「雲微物理がボトルネック」と言い，Warner は「すべてのパラメタリゼーション方式はひとつの結合系を形成している」と付け加える——単一点の修正では解決できない。

### 推論チェーン 8：エアロゾルは太陽光発電予測にどう影響するのか？（Ch4 追加）

```
人為的・自然的エアロゾル → CCN 濃度の変化
    ↓
CCN が多い → 雲粒が多く小さい → 光学的厚さ↑ → アルベド↑ → GHI↓
CCN が少ない → 雲粒が少なく大きい → 光学的厚さ↓ → 降水効率↑ → 雲が短命 → GHI↑
    ↓
業務 NWP は通常エアロゾルを模擬しない（コストが高すぎる）
ただし ECMWF は CAMS エアロゾルを同化 → 日射量予測が改善
    ↓
Yang Ch5.4 の REST2 は AOD(τ₅₅₀) を入力として必要とする
Yang Ch4.5.1 の晴天モデルは 6 つの透過率の積で，そのうち Tₐ = エアロゾル減衰
    ↓
結論：エアロゾルの不確かさ → 晴天モデルの不確かさ + NWP 雲模擬の不確かさ
```

### 推論チェーン 9：地表エネルギー分配は太陽光発電予測にどう影響するのか？（Ch5 追加）

```
地表に到達する太陽放射：R = (Q+q)(1-α) - I↑ + I↓
    ↓
R は 3 成分に配分される：LE（潜熱）+ H（顕熱）+ G（地中熱）
    ↓
Bowen 比 β = H/LE がエネルギーの行き先を決める：
  - 湿潤地（β が小）→ エネルギーの大部分が蒸発に → PBL は涼しく湿潤
  - 乾燥地（β が大）→ エネルギーの大部分が大気加熱に → PBL は深く高温
    ↓
乾燥：PBL が深い → 対流が強い → 午後の積雲が多い → GHI が急減（ランプダウン）
湿潤：PBL が浅い → 対流が弱い → 積雲が少ない → GHI がより安定
    ↓
結論：土壌水分 → Bowen 比 → PBL 深度 → 対流発生 → 雲 → 日射量
```

**太陽光発電実践の意義**：灌漑期と非灌漑期では太陽光発電の出力カーブに系統的な差異が生じる——灌漑は Bowen 比を下げ，午後のランプイベントを減少させる。

### 推論チェーン 10：PILPS はなぜ複雑な LSM が必ずしも優れていないと示したのか？（Ch5 追加）

```
複雑な LSM はより多くのパラメータを持つ（植生・根系・土壌水理学…）
    ↓
各パラメータの定義には観測データが必要——しかし直接観測できないものがほとんど
    ↓
不確かなパラメータ × 多数 = 誤差空間の膨張
    ↓
単純な LSM はパラメータが少ない → キャリブレーション可能 → 実際の精度が高い場合がある
    ↓
類比：Yang 教材 Ch3 のバイアス・分散トレードオフ
  - 複雑モデル = 低バイアス・高分散
  - 単純モデル = 高バイアス・低分散
```

**示唆**：これは機械学習の過学習問題と完全に同構だ——モデルの複雑さはデータの情報量と一致させなければならない。

### 推論チェーン 11：データ同化の本質は何か？（Ch6 追加）

```
真の大気状態 x_true（未知）
    ↓
2 つの不完全な推定：
  背景場 x_b（前回の予報，系統誤差 B）
  観測 y（疎，計器誤差 R を持つ）
    ↓
最適な組み合わせ：x_a = x_b + K(y - H(x_b))
  K = BH^T(HBH^T + R)^{-1}  （カルマンゲイン行列）
    ↓
鍵：B 行列がすべてを決める
  - 静的 B（3DVAR）→ 天気パターンによらず一定 → 解析品質を制限
  - 動的 B（EnKF）→ アンサンブルで推定 → レジーム依存
  - 暗黙的 B（4DVAR）→ 時間窓を通じて伝播 → 最適だが随伴モデルが必要
```

**太陽光発電への示唆**：NWP の日射量予報品質 = 初期条件品質 × モデル品質。同化品質が低い → 雲場の初期化が誤る → 短期予報が直接無効化される。

### 推論チェーン 12：なぜアンサンブル平均が単一予報より優れているのか？（Ch7 追加）

```
非線形方程式 → f(x̄) ≠ f̄(x)
    ↓
単一の「最適」初期条件による予報 ≠ 最適予報
    ↓
複数メンバーの誤差が予測不可能成分で互いに打ち消し合う
    ↓
アンサンブル平均 → 非線形フィルタリング効果 → 真値に近づく
    ↓
ただし！分岐点ではアンサンブル平均が二分岐の間に落ちる → 物理的な状態を表さない
    ↓
結論：アンサンブルが提供するのは「最良の予報」ではなく「不確かさの推定」
```

**太陽光発電への示唆**：確率予報（「明日の GHI が 500 W/m² を超える確率は 70%」）は，決定論的予報（「明日の GHI = 520 W/m²」）よりも電力系統の調整に大きな価値を持つ。

### 推論チェーン 13：予測可能性の上限はどこにあるのか？（Ch8 追加）

```
Lorenz のバタフライ効果 → 微小な初期差異 → 数週間後に完全に発散
    ↓
誤差増幅の 3 段階：誘導期（10〜15 日）→ 線形増幅（〜20 日）→ 飽和
    ↓
ただし！地表強制が「無料の」予測可能性を提供する：
  - 日変化：海陸風・山谷風・低層ジェット
  - 季節変化：太陽高度角・植生状態
    ↓
太陽光発電予測の良い知らせ：日射量の日変化は決定論的な太陽強制が主に駆動する
太陽光発電予測の悪い知らせ：雲の発生・消散はカオス的で，数時間を超える予測は困難
```

### 推論チェーン 14：代表性誤差はなぜ解消できないのか？（Ch9 追加）

```
観測 = ある地点・ある瞬間の値
モデル = グリッドボックス体積の時空間平均値
    ↓
完璧なモデル + 完璧な計器でも，両者は等しくならない
    ↓
代表性誤差 ∝ グリッドボックス内のサブグリッド変率
    ↓
解像度が高いほど代表性誤差は小さくなるが，ゼロにはならない
    ↓
有効解像度 > 2Δx（数値散逸がさらに場を平滑化するため）
```

**太陽光発電への示唆**：地面日射量計（点計測）で NWP 出力（グリッド平均）を検証する際は，代表性誤差を考慮しなければならない——特に破れた雲の天気では。

### 推論チェーン 15：MOS 後処理のジレンマ（Ch12-13 追加）

```
MOS = 過去の予報-観測ペアから統計的回帰を構築する手法
    ↓
利点：系統的バイアスを除去；技術向上は「数年分のモデル改善」に相当
    ↓
ジレンマ：モデルを改良するたびに → 統計的関係が無効化 → 再訓練が必要
    ↓
解決策 1：PP 法（予報ではなく解析場で回帰を構築）→ モデルバージョンに依存しないがモデル誤差を修正しない
解決策 2：動的 MOS（短いローリング訓練窓）→ 素早く適応するがサンプルが少ない
解決策 3：Reforecasts（現行モデルで過去事例を再予報）→ 大サンプルだが計算コストが高い
```

**Yang Ch8 との対応**：Yang の EMOS/BMA 手法 = Warner Ch13 のアンサンブル後処理——まったく同じフレームワーク。

### 推論チェーン 16：再解析データの二重性（Ch16 追加）

```
再解析 = 固定バージョンの同化システムで過去の観測を処理したもの
    ↓
利点：長期的に一貫したグリッドデータ，生の観測より完全
    ↓
落とし穴：降水と地表フラックスは通常同化されない → 純粋なモデル産物
    ↓
ERA5 の日射量データ = モデル放射方式の出力であり，観測ではない！
    ↓
再解析を使って NWP を検証する → あるモデルで別のモデルを検証 → バイアスが生じる
```

**太陽光発電への示唆**：ERA5 の GHI データは便利だが系統的バイアスがある。地上観測や衛星推定データとの相互検証が必須だ。

## V. 解決済みの疑問

### ❓ Yang Ch7.1.1 は「7 本の原始方程式」と言うが，方程式が非常に抽象的で各項の物理的意味がわからない

✅ **Warner Ch2 がこの疑問を完全に解決した。** 各方程式の各項には明確な物理的解釈がある：気圧傾度力が風を駆動し，コリオリ力が風を偏向させ，加熱率 $H$ は放射＋潜熱＋顕熱を含み，水蒸気源汇項 $Q_v$ は蒸発と凝結を含む。

### ❓ Yang Ch7.1.2 は「パラメタリゼーションは NWP で最も議論の的になるテーマ」と言うが，なぜか？

✅ **Warner Ch2 の Reynolds 平均が答えを与える。** パラメタリゼーションの数学的起源は Reynolds 応力項だ——本質的には**閉合問題**（方程式数が未知数より少ない）である。どんなパラメタリゼーション方式も，この閉合問題に対する一つの近似にすぎず，「唯一の正解」は存在しない。だから議論は永遠に続く。

### ❓ CFL 条件の具体的な導出は何か？時間ステップ制約は NWP のリアルタイム性にどう影響するか？

✅ **Warner Ch3 が詳細に導出している。** CFL 条件：$\Delta t \leq \Delta x / c$，ここで $c$ は最速信号伝播速度（音波 ~340 m/s）。これは $\Delta x = 1$ km のとき $\Delta t \leq 3$ s を意味し，1 日の模擬に 28,800 ステップが必要になる——これが高解像度 NWP の計算量爆発の根本的な理由だ。半陰的手法で CFL 制約を緩和できるが，精度の損失を伴う。

### ❓ 雲微物理パラメタリゼーションは日射量予測の最大の誤差源か？

✅ **Warner Ch4 がこれを確認し，より深く掘り下げた。** 雲微物理そのものだけでなく，パラメタリゼーション循環系全体（放射→地表→PBL→対流→微物理→雲→放射）が結合している。放射方式の物理はすでに成熟している；ボトルネックはその**入力**——雲の三次元分布にある。雲の品質は上流のすべてのパラメタリゼーション方式の協調に依存する。エアロゾル（CCN 濃度）の不確かさは，雲粒径分布に影響することでさらに誤差を増幅させる。

### ❓ 陸面パラメタリゼーションの誤差寄与はどのくらいか？植生・土壌パラメータの不確かさは過小評価されていないか？

✅ **Warner Ch5 が部分的に検証した。** PILPS の 23 LSM 比較では，複雑な LSM が単純な LSM を必ずしも上回らないことが示された——パラメータの不確かさが膨張するためだ。土壌水分が熱伝導率に与える影響は 2 桁に及ぶ（乾燥土 vs 湿潤土）ため，水分誤差が増幅される。LDAS のスピンアップに数ヶ月かかることは，土壌状態の初期誤差の減衰が極めて遅いことを示す。**結論**：陸面は確かに過小評価されている誤差源だが，雲ほど致命的ではない——陸面過程の時間スケール（時間〜日）は雲（分〜時間）より遅く，誤差伝播も遅い。

### ❓ ECMWF の 4D-Var は 3D-Var よりどれだけ優れているのか？

✅ **Warner Ch6 が答える。** 4DVAR は時間窓内のすべての観測を使って初期条件を制約し，B 行列を暗黙的に時間発展させる（レジーム依存）。3DVAR の B は静的だ（NMC 法で推定；天気パターンによらず不変）。しかし 4DVAR は随伴モデルが必要（構築・維持は巨大な工学的努力）で，モデルが完璧であると仮定する（Q=0）。EnKF は随伴不要だが十分なメンバー数が必要。ハイブリッド法 $B_{hybrid}$ は両者単独より優れる。

### ❓ Warner の気象アンサンブルと Yang の統計アンサンブルは対応するか？確率的パラメタリゼーションとモデル不確かさ摂動は同じものか？

✅ **Warner Ch7 が答える。** 気象アンサンブルの 3 種の不確かさ源——初期条件（Bred vectors／特異ベクトル／EnKF）・モデル物理（マルチ方式／確率的パラメタリゼーション）・境界条件——は Yang Ch3.3 のフレームワークと部分的に対応するが，完全に同構ではない。確率的パラメタリゼーション（Ch4 Grell & Devenyi 2002）は確かにアンサンブル手法の特殊な形態だ：パラメタリゼーション傾向に [0.5, 1.5] の乱数を乗じる → 「モデルにノイズを加える」のと等価。

### ❓ Warner の NWP 検証と Yang の予測検証フレームワークはどれだけ重複するか？

✅ **Warner Ch9 が答える。** 重複は大きい。MAE/RMSE/Bias/AC/POD/FAR/CSI/HSS は完全に一致する。Yang がさらに強調する点：①日射量の検証には日変化周期を考慮しなければならない（持続性ではなく日変化持続性を参照基準に使う）；②GHI の空間変動は大きく（破れた雲では 1 km 以内で 50% 変動しうる），代表性誤差が特に深刻だ。

### ❓ 日射量予測の検証には何か特別な点があるか？

✅ **Warner Ch9 の代表性誤差の概念が直接適用できる。** 地面日射量計（点計測）vs NWP グリッド平均——積雲対流の天気下では，1 km² 内で GHI が 200〜900 W/m² に変化することもあり，代表性誤差がモデル誤差より大きくなる可能性がある。オブジェクト検証手法（MODE/SAL）は，日射量予報の空間パターンを評価するのに，格子点ごとの検証より適している場合がある。

### ❓ 実験設計では個々の因子の寄与をどう分離するか？

✅ **Warner Ch10 が完全な方法論を示す。** 因子分離法（Stein & Alpert 1993）：N 個の因子に対して 2^N 回のシミュレーションが必要；各因子の純寄与 = その因子を含むシミュレーション − 含まないシミュレーション − 交互作用項。OSSE（観測システム模擬実験）は「仮想大気」で新しい観測システムの価値を検証する——衛星を打ち上げる前に効果を評価できる。随伴感度法は，コスト関数の初期条件に対する勾配を用いて「予報に最も影響を与える領域」を特定する——Ch6 の適応観測（定点ゾンデ投入）と同じツールだ。

### ❓ モデル出力にはどのような高度な解析手法があるか？

✅ **Warner Ch11 が体系的に整理する。** EOF/PCA は主要な変動モードを抽出する——日射量時空間データの第 1 モードは通常，大規模な雲被覆の日変化に対応する。SOM（自己組織化マップ）は天気パターンを自動分類する——日射量の日型を「晴天／薄雲／破れ積雲／全曇り」などに分類し，各カテゴリに異なる予測戦略を適用できる。CCA/SVD は結合モードを見つける——大規模循環パターンと局地日射量の統計的関係は，統計的ダウンスケーリングの数学的基礎だ。

### ❓ 業務 NWP システムはどのような工学的制約に直面するか？

✅ **Warner Ch12 が運用の視点を与える。** CFL 違反は業務停止の最も一般的な原因だ——極端な風速が不安定性を生む。解決策には適応的時間ステップ，半陰的手法，そして最も素朴な「再起動して時間ステップを小さくする」がある。予報は天気より速く走らなければならない——ECMWF の IFS 10 日全球予報は約 1 時間以内に完了しなければならない。モデル改良は漸進的だ——頻繁な変更はできない，なぜなら下流の MOS 統計的関係が無効化されるから。予報官の役割は「モデルを修正する」から「不確かさを解釈する」と「影響の大きな天気に集中する」へと移行した。

### ❓ 統計的後処理はどれだけ改善できるか？MOS の限界はどこにあるか？

✅ **Warner Ch13 が明確に答える。** MOS の技術向上は「数年分のモデル物理改善」に相当する——これは統計的手法の ROI が非常に高いことを意味する。しかし MOS のジレンマ：モデルを改良するたびに → 統計的関係が無効化 → 新たな予報-観測ペアを蓄積して再訓練しなければならない。三つの対応策：①Perfect-Prog（予報ではなく解析場で回帰を構築；モデルバージョンに依存しないがモデル誤差を修正しない）②動的 MOS（短いローリング訓練窓；素早く適応するがサンプルが少ない）③Reforecasts（現行モデルで過去事例を再予報；大サンプルだが計算コストが高い）。BMA/EMOS はアンサンブル出力を較正された確率密度関数に変換する——Yang 教材 Ch8 の手法がこれと完全に対応する。

### ❓ NWP には太陽光発電以外にどのような結合応用があるか？

✅ **Warner Ch14 が NWP の広範な下流応用を示す。** 風力ランプイベントの 4 つの物理的原因：前線通過，低層ジェット高度振動，地形重力波，対流流出——日射量にもランプイベントがあり（雲の急激な遮蔽・消散），物理メカニズムは異なるが予測の困難さは同様だ。大気質予測には NWP による風場＋境界層高度＋降水（湿性沈着）が必要。感染症予測（マラリア・デング熱）は NWP の気温＋湿度で蚊の個体群モデルを駆動する——遅れた相関があるため事前予測が可能だ。電力需要モデル（Taylor & Buizza 2003）はアンサンブル予報を使って 10 日確率予測を提供する。

### ❓ DNS/LES/RANS の適用範囲は何か？テラ・インコグニタはどう扱うか？

✅ **Warner Ch15 が完全な CFD 分類を示す。** DNS はすべての乱流スケールを解析する（Re ~O(10³) 以下で可能；大気の Re ~O(10⁸) は完全に不可能）。LES は大渦を陽的に解析し，小渦をパラメタリゼーションする（格子 ~O(10 m)；SGS モデルが必要）。RANS は Reynolds 平均後にすべての乱流をパラメタリゼーションする（古典的な NWP の手法）。**テラ・インコグニタ**（Wyngaard 2004）：Δx = 100 m〜1 km では，乱流を完全にパラメタリゼーションすることも完全に解析することもできない——Ch4 のグレーゾーン（Δx = 1〜10 km，対流が半パラメタリゼーション・半解析）と同構の問題で，スケールは異なるが哲学的困難は同じだ。将来の超高解像度 NWP はこの問題に必ず直面する。

### ❓ 再解析データを観測として直接使えるか？気候ダウンスケーリングの 2 種の手法はそれぞれどんな長所と短所があるか？

✅ **Warner Ch16 が詳細に答える。** 再解析 ≠ 観測。降水と地表フラックスは通常同化されない（純粋なモデル産物）。ERA5 の日射量 = 放射パラメタリゼーション方式の出力 → 系統バイアスがある。同化された変数（気温・風・湿度）でさえ，解析結果はモデル力学と観測の混合物だ——再解析間の差異が不確かさを定量化できる（例：ERA5 vs JRA-55 vs MERRA-2）。統計的ダウンスケーリングは計算コストが安いが，関係が定常であることを仮定する（気候変化下では破綻する可能性があり危険）；極値を過小評価する。力学的ダウンスケーリング（領域気候モデル RCM）は物理的に一貫していて極値を捉えられるが，計算コストが高く GCM バイアスの伝播を受けやすい。どちらも GCM の大規模バイアスに敏感だ。太陽光発電への示唆：領域気候モデルは高解像度の長期日射量データを提供でき，発電所の立地選定や長期発電量評価に使える——ただし地上観測や衛星データとの相互検証が必須だ。
