## 二、因果推論チェーン

### 推論チェーン1：なぜ NWP は太陽光発電の前日予測における唯一信頼できる情報源なのか？

```
前提1（Yang Ch4.5.4）：日射量の時空間依存性は完全に移動する雲に由来する
    ↓
前提2（Yang Ch4.5.4）：全天カメラは雲を「凍結」と仮定 → 30分未満で無効
前提3（Yang Ch4.5.4）：衛星 CMV は雲が等速直線運動すると仮定 → 4時間未満で無効
    ↓
前提4（Warner Ch2）：雲の力学的進化を解くには原始方程式のみが有効
    ↓
前提5（Yang Ch5.6.2）：電力系統の DAM は 36時間以上のリードタイムを必要とする
    ↓
結論：NWP は前日予測における唯一信頼できる情報源である
```

この推論チェーンは Warner Ch2 の原始方程式から出発し、なぜカメラと衛星が前日スケールで失敗するかを説明しています——それらの物理的仮定（凍結場/等速平流）が長時間後に成立しなくなるためであり、NWP の原始方程式のみが雲の生成・発達・消散をシミュレートできます。

### 推論チェーン2：NWP 予報誤差はどこから来るのか？

```
Warner Ch2 原始方程式 → 7 個の変数を解く必要がある
    ↓
問題1：方程式は非線形結合 → 解析解なし → 離散化が必須（Ch3）
    → 離散化が打ち切り誤差を生む
    ↓
問題2：サブグリッドプロセス（乱流、雲微物理）は陽的に解けない
    → パラメータ化が必須（Ch4-5）
    → パラメータ化は実際の物理の近似 → パラメータ化誤差を生む
    ↓
問題3：初期条件は限られた観測から得られる
    → 観測には測定誤差 + 空間カバレッジ不足がある
    → 不完全な初期条件 → カオス増幅（Ch8）
    ↓
総誤差 = 打ち切り誤差 + パラメータ化誤差 + 初期条件誤差
```

**Yang 教材との接続**：Yang Ch3.1 は「予報誤差には2つの源がある：不完全な初期条件 + 不完全な力学法則」と述べています。Warner Ch2 はさらに「不完全な力学法則」の具体的な意味を説明します——方程式自体に問題があるのではなく（原始方程式は正確です）、**解法**が誤差を引き込む（離散化 + パラメータ化）のです。

### 推論チェーン3：Reynolds 平均は日射量予測にどう繋がるのか？

```
Warner Ch2：大気にはサブグリッド乱流がある → 直接解析できない
    ↓
Reynolds 分解：u = ū + u' → 平均後に共分散項 -ρ(u'w') が現れる
    ↓
共分散項 = Reynolds 応力 = 乱流が平均流に与える影響
    ↓
パラメータ化が必須（Ch4）→ 境界層（PBL）パラメータ化スキーム
    ↓
PBL スキームが鉛直混合強度を決定
    ↓
分岐A：強い混合 → 水蒸気上昇 → 雲形成 → GHI 減少
分岐B：弱い混合 → 安定成層 → 雲少 → GHI 増加
    ↓
Yang Ch7.1.2：PBL スキームは WRF-Solar で最も感度の高い選択の一つ
```

**核心的洞察**：Reynolds 平均というまったくの数学的操作から、4ステップの推論を経て、太陽光発電日射量予測の精度に直結します。これが「知識は孤島ではない」の体現です——1895年の乱流分解手法が、2024年の太陽光発電予測品質を決めています。

### 推論チェーン4：静水圧近似の選択が太陽光発電予測にどう影響するか？

```
Warner Ch2.3.1：静水圧近似は音波をフィルタリング → より大きな時間ステップを許容 → 計算が速い
    ↓
しかし：Δx < 10km では静水圧近似が破綻
    ↓
GFS（全球モデル、~13km）→ 静水圧近似を使用可 → 粗い解像度だが全球カバレッジ
HRRR（領域モデル、3km）→ 非静水圧方程式が必須 → 高解像度だが CONUS のみ
    ↓
Yang Ch6.6 の比較：HRRR の日射量予報が GFS を上回る
    ↓
理由の一つ：非静水圧方程式は対流セルを解像できる → より正確な雲シミュレーション
    ↓
しかし HRRR は米国のみをカバー → 他の地域は GFS や ECMWF に頼るしかない
    ↓
WRF-Solar の価値：ユーザー自身が非静水圧領域モデルを実行できる
    ただし気象学の知識が必要（Yang Ch4.6.1 の学際的挑戦）
```

**実際の意思決定の意義**：米国で太陽光発電予測を行うなら HRRR（3km 非静水圧）を使用。中国では GFS/ECMWF（>10km 静水圧）しかないか、自分で WRF-Solar を走らせる必要があります。この意思決定の理論的根拠が Warner Ch2.3 の静水圧近似です。

### 推論チェーン5：解像度を2倍にするとコストが×8になるのはなぜか？（Ch3 追加）

```
CFL 条件：U·Δt/Δx < 1
    ↓
Δx を半分に → Δt も半分にしなければならない（さもなくば CFL 違反）
    ↓
2D グリッド点数 ×4（面積は同じで各次元 ×2）
    ↓
総計算コスト = グリッド点数 × 時間ステップ数 = 4 × 2 = 8
    ↓
実際はさらに多い：鉛直層も細分化される場合 → ×16 以上
```

**太陽光発電予測への意思決定の意義**：BOSS の RTX 4050 で WRF-Solar を走らせる場合、9km → 3km は約 27 倍の計算コストが必要です。精度向上がそれに見合うかどうかを慎重に検討しなければなりません。

### 推論チェーン6：打ち切り誤差はどのように日射量まで伝播するのか？（Ch3 追加）

```
有限差分での微分計算 → 打ち切り誤差（短波 6Δx：17% 誤差）
    ↓
気圧傾度力が不正確 → 地衡風のバイアス
    ↓
発散が不正確 → 鉛直運動のバイアス
    ↓
鉛直運動が凝結/蒸発を駆動 → 雲液水量のバイアス
    ↓
放射スキームが雲水で日射量を計算 → GHI 出力のバイアス
```

**核心的洞察**：日射量予報誤差は放射パラメータ化だけから来るのではなく、最低レベルの微分近似から積み重なっていきます。これが Yang が「雲微物理がボトルネック」と言い、Warner が「数値手法も重要」と強調する理由です——両者は加法的でなく乗法的な関係です。

### 推論チェーン7：なぜ「雲が日射量予測最大のボトルネック」なのか？（Ch4 追加）

```
放射パラメータ化の物理は成熟している（ベール則 + プランク関数）
    ↓
放射スキームへの入力 = 雲の3次元分布と光学特性
    ↓
雲の光学特性 ← 雲微物理（雲粒径分布）
    ↓
雲の存在 ← 対流パラメータ化（いつどこで凝結が引き起こされるか）
    ↓
対流への水蒸気供給 ← 境界層パラメータ化（鉛直混合強度）
    ↓
境界層の熱力学的駆動 ← 地表エネルギー収支（日射量の関数！）
    ↓
結論：日射量予報誤差の根源は放射スキーム自体にあるのではなく、
「放射→地表→PBL→対流→微物理→雲→放射」というサイクル全体にある
```

**核心的洞察**：Yang は「雲微物理がボトルネック」と言い、Warner は「すべてのパラメータ化スキームは結合サイクルを形成している」と補足します——単一点の修正では解決できません。

### 推論チェーン8：エアロゾルは太陽光発電予測にどう影響するか？（Ch4 追加）

```
人為的/自然起源エアロゾル → CCN 濃度の変化
    ↓
CCN 多 → 雲粒多く小さい → 光学的厚さ増加 → アルベド上昇 → GHI 低下
CCN 少 → 雲粒少なく大きい → 光学的厚さ低下 → 降水効率上昇 → 雲が短命 → GHI 上昇
    ↓
業務 NWP は通常エアロゾルをシミュレートしない（コスト大）
しかし ECMWF は CAMS エアロゾルを同化 → 日射量予報がより優れる
    ↓
Yang Ch5.4 REST2 は AOD(τ₅₅₀) を入力として必要とする
Yang Ch4.5.1 晴天モデルは6つの透過率を掛け合わせ、うち Tₐ = エアロゾル減衰
    ↓
結論：エアロゾル不確実性 → 晴天モデル不確実性 + NWP 雲シミュレーション不確実性
```

### 推論チェーン9：地表エネルギー分配は太陽光発電予測にどう影響するか？（Ch5 追加）

```
地面に到達する太陽放射：R = (Q+q)(1-α) - I↑ + I↓
    ↓
R は3つに分配：LE（潜熱）+ H（顕熱）+ G（地中熱）
    ↓
ボーエン比 β = H/LE がエネルギーの行き先を決定：
  - 湿潤地（β小）→ ほとんどのエネルギーが蒸発に使われる → 境界層が涼しく湿潤
  - 乾燥地（β大）→ ほとんどのエネルギーが大気を加熱 → 境界層が熱く深い
    ↓
乾燥：より深い PBL → より強い対流 → 午後の積雲増加 → GHI 急降下
湿潤：浅い PBL → 弱い対流 → 積雲少 → GHI がより安定
    ↓
結論：土壌水分 → ボーエン比 → PBL 深さ → 対流トリガー → 雲 → 日射量
```

**太陽光発電実践の意義**：灌漑期と非灌漑期の太陽光発電出力曲線には系統的な差異があります——灌漑によりボーエン比が低下し → 午後の急変イベントが減少します。

### 推論チェーン10：なぜ PILPS は複雑な LSM が必ずしも優れていないと発見したのか？（Ch5 追加）

```
複雑な LSM はより多くのパラメータを持つ（植生、根系、土壌水力学...）
    ↓
各パラメータを定義するには観測データが必要——しかしほとんどのパラメータは直接観測できない
    ↓
不確実なパラメータ × 多数 = 誤差空間の膨張
    ↓
シンプルな LSM はパラメータが少ない → キャリブレーション可能 → 実際の精度が高い場合がある
    ↓
類比：Yang Ch3 のバイアス-バリアンストレードオフ
  - 複雑モデル = 低バイアス高バリアンス
  - シンプルモデル = 高バイアス低バリアンス
```

**示唆**：これは機械学習の過学習問題と完全に同型です——モデルの複雑さはデータの情報量に一致しなければなりません。

### 推論チェーン11：データ同化の本質とは何か？（Ch6 追加）

```
真の大気状態 x_true（未知）
    ↓
2つの不完全な推定：
  背景場 x_b（前回の予報、系統誤差 B）
  観測 y（疎ら、機器誤差 R を持つ）
    ↓
最適な結合：x_a = x_b + K(y - H(x_b))
  K = BH^T(HBH^T + R)^{-1}（カルマンゲイン行列）
    ↓
鍵：B 行列がすべてを決める
  - 静的 B（3DVAR）→ 天気パターンによって変化しない → 解析品質を制限
  - 動的 B（EnKF）→ アンサンブルで推定 → レジーム依存
  - 陰的 B（4DVAR）→ 時間窓で伝播 → 最適だが伴随モデルが必要
```

**太陽光発電への意味**：NWP 日射量予報品質 = 初期条件品質 × モデル品質。同化品質が低い → 雲場の初期化が誤り → 短期予報が直接破綻します。

### 推論チェーン12：なぜアンサンブル平均は単一予報より優れるのか？（Ch7 追加）

```
非線形方程式 → f(x̄) ≠ f̄(x)
    ↓
単一の「最適」初期条件からの予報 ≠ 最適予報
    ↓
複数メンバーの誤差は予測不可能な成分で互いに打ち消し合う
    ↓
アンサンブル平均 → 非線形フィルタリング効果 → 真値に近い
    ↓
しかし！分岐点では、アンサンブル平均が2つの分岐の間に落ちる場合がある → 物理的状態を表さない
    ↓
結論：アンサンブルは「最良の予報」を提供するのではなく、「不確実性の推定」を提供する
```

**太陽光発電への意味**：確率的予報（「明日の GHI が 500 W/m² を超える確率は 70%」）は、確定的予報（「明日の GHI = 520 W/m²」）よりも電力系統の運用管理に大きな価値があります。

### 推論チェーン13：予測可能性の上限はどこか？（Ch8 追加）

```
Lorenz のバタフライ効果 → わずかな初期差異 → 数週間後に完全発散
    ↓
誤差成長の3段階：誘導期（10-15日）→ 線形成長（~20日）→ 飽和
    ↓
しかし！地表強制が「タダ」の予測可能性を提供する：
  - 日変化：海陸風、山谷風、下層ジェット
  - 季節変化：太陽高度角、植生状態
    ↓
太陽光発電予測の良いニュース：日射量の日変化は主に確定的な太陽強制で駆動される
太陽光発電予測の悪いニュース：雲の出現/消散はカオス的で、数時間以上の予測は困難
```

### 推論チェーン14：代表性誤差はなぜ排除できないのか？（Ch9 追加）

```
観測 = 特定の時刻・場所での瞬時の点値
モデル = グリッドボックス体積の時空間平均値
    ↓
完全なモデル + 完全な機器があっても、両者は等しくない
    ↓
代表性誤差 ∝ グリッドボックス内のサブグリッド変動
    ↓
解像度が高い → 代表性誤差が小さい、しかしゼロにはならない
    ↓
有効解像度 > 2Δx（数値散逸がさらに場を平滑化する）
```

**太陽光発電への意味**：NWP 出力（グリッド平均）を地面日射量ステーション（点測定）で検証する際、特に破断雲の天気では代表性誤差を考慮しなければなりません。

### 推論チェーン15：MOS 後処理のジレンマ（Ch12-13 追加）

```
MOS = 過去の予報-観測ペアから統計回帰を構築
    ↓
利点：系統的バイアスを排除、スキルは数年のモデル改良と同等
    ↓
ジレンマ：モデル更新のたびに → 統計関係が無効 → 再訓練が必要
    ↓
解決策1：PP 法（予報でなく解析場から回帰を構築）→ モデルバージョンに依存しないがモデル誤差を修正しない
解決策2：動的 MOS（短いローリング訓練期間）→ 素早く適応するがサンプルが小さい
解決策3：Reforecasts（現在のモデルで過去ケースを再予報）→ 大きなサンプルだが計算コスト高
```

**Yang Ch8 との対応**：Yang の EMOS/BMA 手法 = Warner Ch13 のアンサンブル後処理——まったく同じフレームワーク。

### 推論チェーン16：再解析データの二重性（Ch16 追加）

```
再解析 = 固定バージョンの同化システムで過去の観測を処理
    ↓
利点：長期的に一貫したグリッドデータ、生観測より完全
    ↓
落とし穴：降水と地表フラックスは通常同化されない → 純粋にモデルの産物
    ↓
ERA5 日射量データ = モデル放射スキームの出力であり、観測ではない！
    ↓
再解析で NWP を検証する → あるモデルで別のモデルを検証 → バイアスがある
```

**太陽光発電への意味**：ERA5 GHI データは便利ですが系統的バイアスがあり、地面ステーションや衛星導出データとのクロス検証が必須です。

## 五、解決済みの疑問

### ❓ Yang Ch7.1.1 は「7 個の原始方程式」と述べていますが、方程式が抽象的に見え、各項の物理的意味が不明でした

✅ **Warner Ch2 がこの疑問を完全に解決しました。** すべての方程式のすべての項に明確な物理的解釈があります：気圧傾度力が風を駆動し、コリオリ力が風を偏向させ、加熱率 $H$ には放射 + 潜熱 + 顕熱が含まれ、水蒸気源/シンク項 $Q_v$ には蒸発と凝結が含まれます。

### ❓ Yang Ch7.1.2 は「パラメータ化は NWP で最も議論の多いトピック」と述べていますが、なぜかわかりませんでした

✅ **Warner Ch2 の Reynolds 平均が答えを示しています。** パラメータ化の数学的起源は Reynolds 応力項です——これは本質的に**閉合問題**（方程式の数より未知数が多い）です。あらゆるパラメータ化スキームはこの閉合問題への近似であり、「唯一正しい答え」は存在しません。だからこそ議論は常に続きます。

### ❓ CFL 条件の具体的な導出は？時間ステップの制約は NWP のリアルタイム性にどう影響するか？

✅ **Warner Ch3 に詳細な導出があります。** CFL 条件：$\Delta t \leq \Delta x / c$、ここで $c$ は最速信号伝播速度（音波 ~340 m/s）。これは $\Delta x = 1$ km の場合 $\Delta t \leq 3$ s を意味し、1日をシミュレートするには 28,800 ステップが必要です——これが高解像度 NWP の計算コストが爆発的になる根本的な理由です。半陰的手法は CFL 制約を緩和できますが精度の損失を伴います。

### ❓ 雲微物理パラメータ化は日射量予測の最大誤差源なのか？

✅ **Warner Ch4 が確認・深化させています。** 雲微物理自体がボトルネックであるだけでなく、パラメータ化フィードバックサイクル全体（放射→地表→PBL→対流→微物理→雲→放射）が結合しています。放射スキームの物理は成熟していますが、ボトルネックはその**入力**——雲の3次元分布にあります。そして雲の品質は上流のすべてのパラメータ化スキームの協調的な性能に依存します。エアロゾル（CCN 濃度）の不確実性は、雲粒径分布に影響することでさらに誤差を増幅させます。

### ❓ 地表パラメータ化の誤差寄与はどの程度か？植生/土壌パラメータの不確実性は過小評価されていないか？

✅ **Warner Ch5 が部分的に検証しています。** 23 個の LSM の PILPS 比較では、複雑な LSM が単純な LSM より優れるとは限らないことが示されています——パラメータの不確実性が誤差空間を膨張させるためです。土壌の熱伝導率は乾燥土壌と湿潤土壌の間で 2 桁異なり、含水量の誤差が増幅されます。LDAS が数ヶ月のスピンアップを必要とすることは、土壌状態の初期誤差が極めてゆっくりと減衰することを示しています。**結論**：地表は確かに過小評価された誤差源ですが、雲ほど致命的ではありません——地表過程の時間スケール（時間～日）は雲（分～時間）より遅く、誤差の伝播も遅いです。

### ❓ ECMWF の 4D-Var は 3D-Var よりどれほど優れているか？

✅ **Warner Ch6 が答えています。** 4DVAR は時間窓内のすべての観測を使って初期条件を拘束し、暗黙的に B 行列を時間とともに進化させます（レジーム依存）。3DVAR の B は静的です（NMC 法で推定、天気パターンによって変化しない）。しかし 4DVAR は伴随モデルを必要とし（構築・維持は巨大なエンジニアリング作業）、完全なモデルを仮定します（Q=0）。EnKF は伴随を必要としませんが、十分な数のメンバーが必要です。ハイブリッド B_hybrid はどちらか単独を上回ります。

### ❓ Warner の気象アンサンブルと Yang の統計的アンサンブルは対応しているか？確率的パラメータ化とモデル不確実性の摂動は同じものか？

✅ **Warner Ch7 が答えています。** 気象アンサンブルの3種類の不確実性源——初期条件（Bred vectors/特異ベクトル/EnKF）、モデル物理（マルチスキーム/確率的パラメータ化）、境界条件——は Yang Ch3.3 のフレームワークと部分的に対応しますが完全に同型ではありません。確率的パラメータ化（Ch4、Grell & Devenyi 2002）はアンサンブル手法の特殊な形態です：パラメータ化トレンドに [0.5, 1.5] のランダム因子を掛ける → 「モデルにノイズを加える」と等価。

### ❓ Warner の NWP 検証と Yang の予測検証フレームワークはどの程度重複しているか？

✅ **Warner Ch9 が答えています。** 非常に高い重複度。MAE/RMSE/Bias/AC/POD/FAR/CSI/HSS は完全に一致しています。Yang がさらに強調している点：①日射量検証では日変化サイクルを考慮しなければならない（持続性だけを参照基準にしてはならず、日変化持続性を使う必要がある）②GHI の空間変動が大きい（破断雲の状況では 1km 以内で 50% 変化する場合がある）ため、代表性誤差が特に深刻。

### ❓ 日射量予測の検証には何か特別なことがあるか？

✅ **Warner Ch9 の代表性誤差の概念が直接適用されます。** 地面日射量ステーション（点測定）vs NWP グリッド平均——積雲対流天気では、1km² 内の GHI が 200 から 900 W/m² の範囲で変動し、代表性誤差がモデル誤差より大きい場合があります。オブジェクトベース検証手法（MODE/SAL）は、日射量予報の空間パターンの評価に逐点検証より適している可能性があります。
