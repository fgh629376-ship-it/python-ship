## 二、因果推論チェーン

### 推論チェーン 1：なぜ NWP は太陽光の翌日予測における唯一の信頼できる情報源なのか？

```
前提1（Yang Ch4.5.4）：日射量の時空間依存性は完全に移動する雲に由来する
    ↓
前提2（Yang Ch4.5.4）：全天カメラは雲が「凍結」していると仮定
                       → 30分以内に失効
前提3（Yang Ch4.5.4）：衛星 CMV は雲が等速直線運動すると仮定
                       → 4時間以内に失効
    ↓
前提4（Warner Ch2）：原始方程式のみが雲の力学的発展を解くことができる
    ↓
前提5（Yang Ch5.6.2）：系統 DAM は 36時間以上の先行時間を必要とする
    ↓
結論：NWP は翌日予測における唯一の信頼できる情報源
```

この推論チェーンは Warner Ch2 の原始方程式から出発し、カメラと衛星が翌日スケールで失効する理由を説明する——それらの物理的仮定（凍結場／等速移流）が長時間後には成立しなくなるためであり、雲の生成・発達・消散をシミュレートできるのは NWP の原始方程式だけだ。

### 推論チェーン 2：NWP 予測誤差はどこから来るのか？

```
Warner Ch2 原始方程式 → 7つの変数を解く必要がある
    ↓
問題1：方程式は非線形結合 → 解析解なし → 離散化が必要（Ch3）
       → 離散化により打ち切り誤差が発生
    ↓
問題2：サブグリッド過程（乱流、雲微物理）を陽的に解くことができない
       → パラメータ化が必要（Ch4-5）
       → パラメータ化は実物理の近似 → パラメータ化誤差が発生
    ↓
問題3：初期条件は限られた観測から得られる
       → 観測には測定誤差 + 空間的カバレッジ不足がある
       → 不完全な初期条件 → カオス的増幅（Ch8）
    ↓
総誤差 = 打ち切り誤差 + パラメータ化誤差 + 初期条件誤差
```

**Yang 教材との接続**：Yang Ch3.1 は「予測誤差には2つの源がある：不完全な初期条件＋不完全な力学法則」と述べる。Warner Ch2 は「不完全な力学法則」の具体的な意味をさらに解き明かす——方程式自体に問題があるわけではなく（原始方程式は厳密）、**解法**が誤差を導入している（離散化＋パラメータ化）。

### 推論チェーン 3：Reynolds 平均はどのように日射量予測へつながるのか？

```
Warner Ch2：大気にはサブグリッド乱流がある → 直接解析できない
    ↓
Reynolds 分解：u = ū + u'
    → 平均後に共分散項 -ρ(u'w') が現れる
    ↓
共分散項 = Reynolds 応力 = 乱流が平均流に与える影響
    ↓
パラメータ化が必要（Ch4）→ 境界層（PBL）スキーム
    ↓
PBL スキームが鉛直混合の強度を決定する
    ↓
分岐A：強い混合 → 水蒸気上昇 → 雲の生成 → GHI 減少
分岐B：弱い混合 → 安定成層  → 少ない雲 → GHI 増加
    ↓
Yang Ch7.1.2：PBL スキームは WRF-Solar で最も敏感な選択の一つ
```

**核心的洞察**：純粋に数学的な操作である Reynolds 平均から出発し、4ステップの推論を経て、太陽光日射量予測の精度に直接到達する。これが「知識は孤島ではない」の体現だ——1895年の乱流分解手法が、2024年の太陽光発電予測の品質を左右している。

### 推論チェーン 4：静力近似の選択が太陽光予測にどう影響するか？

```
Warner Ch2.3.1：静力近似が音波をフィルタリング
                → より大きな時間刻みが可能 → 計算が速い
    ↓
しかし：Δx < 10km では静力近似が成立しなくなる
    ↓
GFS （全球モデル、~13km）→ 静力近似を使用可 → 粗いが全球カバー
HRRR（地域モデル、   3km）→ 非静力方程式が必要 → 高解像度だが CONUS のみ
    ↓
Yang Ch6.6 の比較：HRRR の日射量予測は GFS を上回る
    ↓
理由の一つ：非静力方程式は対流セルを解析できる
            → より正確な雲シミュレーション
    ↓
しかし HRRR は米国のみをカバー → 他地域では GFS か ECMWF を使うしかない
    ↓
WRF-Solar の価値：ユーザーが自分で非静力の地域モデルを実行できる
    ただし気象学の知識が必要（Yang Ch4.6.1 の学際的課題）
```

**実際の意思決定の意味**：米国で太陽光予測をするなら HRRR（3km、非静力）を使う；中国では GFS/ECMWF（>10km、静力）しか入手できないか、自分で WRF-Solar を走らせる。この意思決定の理論的根拠は Warner Ch2.3 の静力近似にある。

### 推論チェーン 5：解像度を2倍にするとなぜコストが×8になるのか？（Ch3 新規追加）

```
CFL 条件：U·Δt/Δx < 1
    ↓
Δx を半分にする → Δt も半分にしなければならない（でないと CFL を破る）
    ↓
2D グリッド点数 ×4（領域面積は同じ、各次元で×2）
    ↓
総計算量 = グリッド点数 × 時間ステップ数 = 4 × 2 = 8
    ↓
実際にはさらに多い：鉛直層も細かくなる可能性がある
    → ×16 あるいはそれ以上
```

**太陽光予測への意思決定の意味**：RTX 4050 で WRF-Solar を実行する場合、9km → 3km への変更には約 27 倍の計算量が必要になる。精度向上がそのコストに見合うかどうかを慎重に検討しなければならない。

### 推論チェーン 6：打ち切り誤差はどのように日射量まで伝播するのか？（Ch3 新規追加）

```
有限差分による微分の近似 → 打ち切り誤差
    （短波：6Δx で約 17% の誤差）
    ↓
不正確な気圧傾度力 → 地衡風バイアス
    ↓
不正確な発散 → 鉛直運動バイアス
    ↓
鉛直運動が凝結・蒸発を駆動する
    → 雲の液体水含量バイアス
    ↓
放射スキームが雲水量から日射量を計算する
    → GHI 出力バイアス
```

**核心的洞察**：日射量予測誤差は放射パラメータ化からだけ生じるわけではない——最下層の微分近似の段階からすでに誤差が蓄積し始めている。Yang が「雲微物理がボトルネック」と言いつつ、Warner が「数値手法も重要」と強調する理由はここにある——両者は加法関係ではなく、乗法関係にある。

## 五、解決済みの疑問

### ❓ Yang Ch7.1.1 が「7つの原始方程式」と述べるが、各項の物理的意味が抽象的でわかりにくかった

✅ **Warner Ch2 がこの疑問を完全に解決した。** 各方程式のすべての項に明確な物理的解釈がある：気圧傾度力が風を駆動し、コリオリ力が風を偏向させ、加熱率 $H$ は放射＋潜熱＋顕熱を含み、水蒸気の源汇項 $Q_v$ は蒸発と凝結を含む。

### ❓ Yang Ch7.1.2 が「パラメータ化は NWP で最も議論を呼ぶテーマ」と言うが、その理由がわからなかった

✅ **Warner Ch2 の Reynolds 平均が答えを与えてくれる。** パラメータ化の数学的起源は Reynolds 応力項——本質的には**閉合問題**（方程式数が未知数より少ない）だ。どんなパラメータ化スキームもこの閉合問題への近似であり、「唯一正解」は存在しない。これが議論が終わらない理由だ。
