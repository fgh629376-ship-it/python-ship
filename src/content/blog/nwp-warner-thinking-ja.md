## 二、因果推論連鎖

### 推論連鎖 1：NWP はなぜ翌日太陽光予測の唯一の信頼できる手段なのか？

```
前提1（Yang Ch4.5.4）：日射量の時空間依存は移動する雲から完全に生じる
    ↓
前提2（Yang Ch4.5.4）：全天カメラは雲が「凍結」していると仮定 → 30分未満で失効
前提3（Yang Ch4.5.4）：衛星 CMV は雲の等速直線運動を仮定 → 4時間未満で失効
    ↓
前提4（Warner Ch2）：原始方程式のみが雲の動力学的発展を解くことができる
    ↓
前提5（Yang Ch5.6.2）：電力市場 DAM は >36h のリードタイムを必要とする
    ↓
結論：NWP は翌日予測の唯一の信頼できる手段である
```

この推論連鎖は Warner Ch2 の原始方程式から出発し、カメラと衛星が翌日スケールで失敗する理由を説明します——物理的な仮定（凍結場／等速移動）が長時間後に成立しなくなるからであり、NWP の原始方程式だけが雲の生成・発達・消散をシミュレートできます。

### 推論連鎖 2：NWP の予測誤差はどこから来るのか？

```
Warner Ch2 原始方程式 → 7 つの変数を解く必要がある
    ↓
問題1：方程式は非線形結合 → 解析解なし → 離散化が必要（Ch3）
    → 離散化は打ち切り誤差を導入する
    ↓
問題2：サブグリッド過程（乱流、雲微物理）は陽的に解けない
    → パラメータ化が必要（Ch4-5）
    → パラメータ化は実際の物理の近似 → パラメータ化誤差を導入
    ↓
問題3：初期条件は限られた観測から得られる
    → 観測には測定誤差 + 空間カバレッジ不足がある
    → 不完全な初期条件 → カオス的増幅（Ch8）
    ↓
総誤差 = 打ち切り誤差 + パラメータ化誤差 + 初期条件誤差
```

**Yang 教材との接続**：Yang Ch3.1 は「予測誤差には2つの発生源がある：不完全な初期条件 + 不完全な力学法則」と述べています。Warner Ch2 は「不完全な力学法則」の具体的な意味をさらに解説します——方程式自体に問題があるのではなく（原始方程式は正確です）、**解法**が誤差を導入する（離散化 + パラメータ化）のです。

### 推論連鎖 3：Reynolds 平均は日射量予測とどう繋がるのか？

```
Warner Ch2：大気にはサブグリッド乱流がある → 直接解析できない
    ↓
Reynolds 分解：u = ū + u' → 平均後に共分散項 -ρ(u'w') が出現する
    ↓
共分散項 = Reynolds 応力 = 乱流が平均流に与える影響
    ↓
パラメータ化が必要（Ch4）→ 境界層（PBL）パラメータ化スキーム
    ↓
PBL スキームが鉛直混合強度を決定する
    ↓
分岐A：強混合 → 水蒸気上昇 → 雲形成 → GHI 減少
分岐B：弱混合 → 安定成層 → 少雲 → GHI 増加
    ↓
Yang Ch7.1.2：PBL スキームは WRF-Solar で最も感度の高い選択の一つ
```

**核心的洞察**：Reynolds 平均という純粋に数学的な操作から、4 ステップの推論を経て、太陽光日射量予測の精度へと直接つながります。「知識は孤島ではない」ということの体現です——1895 年の乱流分解法が、2024 年の太陽光発電予測の品質を決定します。

### 推論連鎖 4：静力近似の選択は太陽光予測にどう影響するのか？

```
Warner Ch2.3.1：静力近似は音波をフィルタリングする → 大きな時間ステップが可能 → 計算が速い
    ↓
しかし：Δx < 10km では静力近似は成立しない
    ↓
GFS（全球モデル、~13km）→ 静力近似が使える → 粗解像度だが全球カバレッジ
HRRR（領域モデル、3km）→ 非静力方程式が必要 → 高解像度だが CONUS 限定
    ↓
Yang Ch6.6 の比較：HRRR の日射量予測は GFS より優れている
    ↓
理由の一つ：非静力方程式は対流セルを解析できる → より正確な雲シミュレーション
    ↓
しかし HRRR は米国のみをカバー → 他の地域は GFS か ECMWF しか使えない
    ↓
WRF-Solar の価値：ユーザーが自分で非静力領域モデルを実行できる
    ただし気象学の知識が必要（Yang Ch4.6.1 の学際的課題）
```

**実際の意思決定への意味**：米国で太陽光予測を行うなら HRRR（3km 非静力）を使う；中国では GFS/ECMWF（>10km 静力）か自前の WRF-Solar しか選択肢がない。この意思決定の理論的根拠が Warner Ch2.3 の静力近似です。

### 推論連鎖 5：解像度を2倍にするとコストが×8になるのはなぜか？（Ch3 追加）

```
CFL 条件：U·Δt/Δx < 1
    ↓
Δx を半分にする → Δt も半分にしなければならない（さもなければ CFL を違反）
    ↓
2D グリッド点数 ×4（面積は同じ、各次元に×2）
    ↓
総計算量 = グリッド点数 × 時間ステップ数 = 4 × 2 = 8
    ↓
実際にはさらに多い：鉛直層も細かくなる可能性がある → ×16 以上
```

**太陽光予測への意思決定の意味**：RTX 4050 で WRF-Solar を実行する場合、9km → 3km には約 27 倍の計算量が必要です。精度の向上がコストに見合うかどうかを慎重に判断しなければなりません。

### 推論連鎖 6：打ち切り誤差はどのように日射量まで伝播するのか？（Ch3 追加）

```
有限差分で微分を計算する → 打ち切り誤差（短波 6Δx 誤差 ~17%）
    ↓
不正確な気圧傾度力 → 地衡風の偏差
    ↓
不正確な発散 → 鉛直運動の偏差
    ↓
鉛直運動が凝結/蒸発を駆動する → 雲液水量の偏差
    ↓
放射スキームが雲水を用いて日射量を計算する → GHI 出力の偏差
```

**核心的洞察**：日射量予測誤差は放射パラメータ化からのみ生じるわけではありません——最も根本的な微分近似のレベルから積み重なります。Yang が「雲微物理がボトルネック」と言い、Warner が「数値手法も重要」と強調する理由はここにあります——両者は加算関係ではなく乗算関係にあります。

### 推論連鎖 7：「雲が日射量予測の最大のボトルネック」なのはなぜか？（Ch4 追加）

```
放射パラメータ化の物理はすでに成熟している（Beer の法則 + Planck 関数）
    ↓
放射スキームへの入力 = 雲の三次元分布と光学特性
    ↓
雲の光学特性 ← 雲微物理（雲粒サイズ分布）
    ↓
雲の存在 ← 対流パラメータ化（いつどこで凝結が引き起こされるか）
    ↓
対流への水蒸気供給 ← 境界層パラメータ化（鉛直混合強度）
    ↓
境界層の熱力学的駆動力 ← 地表エネルギーバランス（日射量の関数！）
    ↓
結論：日射量予測誤差の根本原因は放射スキーム自体にあるのではなく、
「放射→地表→PBL→対流→微物理→雲→放射」というサイクル全体にある
```

**核心的洞察**：Yang は「雲微物理がボトルネック」と言い、Warner は「すべてのパラメータ化スキームは結合したシステムである」と補足します——単一点での修正は存在しません。

### 推論連鎖 8：エアロゾルは太陽光予測にどう影響するのか？（Ch4 追加）

```
人為的/自然的エアロゾル → CCN 濃度の変化
    ↓
CCN が多い → 多いが小さい雲粒 → 光学的厚さ↑ → アルベド↑ → GHI↓
CCN が少ない → 少ないが大きい雲粒 → 光学的厚さ↓ → 降水効率↑ → 雲の寿命が短い → GHI↑
    ↓
業務 NWP は通常エアロゾルをシミュレートしない（コストが高すぎる）
しかし ECMWF は CAMS エアロゾルを同化する → より良い日射量予測
    ↓
Yang Ch5.4 の REST2 は AOD（τ₅₅₀）を入力として必要とする
Yang Ch4.5.1 の晴天モデルは 6 つの透過率の積を用い、そのうち Tₐ = エアロゾル減衰
    ↓
結論：エアロゾルの不確実性 → 晴天モデルの不確実性 + NWP 雲シミュレーションの不確実性
```

### 推論連鎖 9：地表エネルギー分配は太陽光予測にどう影響するのか？（Ch5 追加）

```
太陽放射が地表に到達：R = (Q+q)(1-α) - I↑ + I↓
    ↓
R は三部分に分配される：LE（潜熱）+ H（顕熱）+ G（地中熱流量）
    ↓
Bowen 比 β = H/LE がエネルギーの行き先を決定する：
  - 湿潤地（β 小）→ ほとんどのエネルギーが蒸発に使われる → 涼しく湿った境界層
  - 乾燥地（β 大）→ ほとんどのエネルギーが大気を加熱する → 熱くて深い境界層
    ↓
乾燥：PBL が深くなる → 対流が強化される → 午後の積雲が増える → GHI 急減
湿潤：PBL が浅い → 対流が弱い → 積雲が少ない → GHI がより安定
    ↓
結論：土壌水分 → Bowen 比 → PBL 深さ → 対流発生 → 雲 → 日射量
```

**太陽光実践への意味**：灌漑期と非灌漑期の太陽光発電出力曲線には系統的な差異があります——灌漑は Bowen 比を低下させる → 午後の急変イベントが減少します。

### 推論連鎖 10：PILPS はなぜ複雑な LSM が必ずしも優れていないと発見したのか？（Ch5 追加）

```
複雑な LSM にはより多くのパラメータがある（植生、根系、土壌水理学...）
    ↓
各パラメータを定義するには観測データが必要——しかしほとんどのパラメータは直接観測できない
    ↓
不確実なパラメータ × 多数 = 誤差空間の膨張
    ↓
シンプルな LSM はパラメータが少ない → 較正可能 → 実際の精度が高い可能性がある
    ↓
類比：Yang Ch3 のバイアス-分散トレードオフ
  - 複雑モデル = 低バイアス・高分散
  - シンプルモデル = 高バイアス・低分散
```

**教訓**：これは機械学習の過学習問題と完全に同型です——モデルの複雑さは利用可能なデータの情報量に合致させなければなりません。

## 五、解決済みの疑問

### ❓ Yang Ch7.1.1 は「7 つの原始方程式」と言っているが、方程式は非常に抽象的に見える——各項の物理的意味は何か？

✅ **Warner Ch2 がこの疑問を完全に解決しました。** すべての方程式のすべての項に明確な物理的解釈があります：気圧傾度力が風を駆動し、コリオリ力が風を偏向させ、加熱率 $H$ には放射 + 潜熱 + 顕熱が含まれ、水蒸気の源汚項 $Q_v$ には蒸発と凝結が含まれます。

### ❓ Yang Ch7.1.2 は「パラメータ化は NWP で最も議論の多いテーマ」と言っているが、なぜか？

✅ **Warner Ch2 の Reynolds 平均が答えを提供しています。** パラメータ化の数学的起源は Reynolds 応力項です——これは本質的に**閉合問題**（方程式の数が未知数より少ない）です。いかなるパラメータ化スキームも、この閉合問題への一つの近似であり、「唯一の正解」は存在しません。これが議論が永遠に続く理由です。

### ❓ CFL 条件の具体的な導出は何か？時間ステップ制限は NWP のリアルタイム性にどう影響するのか？

✅ **Warner Ch3 が詳細な導出を提供しています。** CFL 条件：$\Delta t \leq \Delta x / c$、ここで $c$ は最速の信号伝播速度（音波 ~340 m/s）。これは $\Delta x = 1$ km のとき $\Delta t \leq 3$ s を意味し、1 日をシミュレートするのに 28,800 ステップが必要となります——これが高解像度 NWP の計算量が爆発的に増大する根本的な理由です。半陰的手法は CFL 制限を緩和できますが、精度の損失をもたらします。

### ❓ 雲微物理パラメータ化は日射量予測の最大の誤差源なのか？

✅ **Warner Ch4 がこれを確認し深化させました。** 雲微物理自体がボトルネックであるだけでなく、パラメータ化サイクル全体（放射→地表→PBL→対流→微物理→雲→放射）が結合しています。放射スキームの物理はすでに成熟していますが、ボトルネックはその**入力**——雲の三次元分布にあります。雲の品質は上流のすべてのパラメータ化スキームの協調に依存します。エアロゾル（CCN 濃度）の不確実性が雲粒サイズ分布に影響することで、誤差をさらに増幅させます。

### ❓ 地表パラメータ化の誤差寄与はどれくらい大きいのか？植生/土壌パラメータの不確実性は過小評価されていないか？

✅ **Warner Ch5 が部分的に検証しました。** 23 個の LSM の PILPS 比較では、複雑な LSM がシンプルな LSM より優れているとは限らないことを示しています——パラメータの不確実性が膨張するためです。土壌水分は熱伝導率に 2 桁の影響を与え（乾燥土 vs 湿潤土）、水分誤差は増幅されます。LDAS が数ヶ月のスピンアップを必要とすることは、土壌状態の初期誤差の減衰が極めて遅いことを示しています。**結論**：地表は確かに過小評価された誤差源ですが、雲ほど致命的ではありません——地表過程の時間スケール（時間〜日）は雲（分〜時間）より遅く、誤差伝播も遅いです。
