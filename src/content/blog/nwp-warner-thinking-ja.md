## Ⅱ. 因果推論連鎖

### 推論連鎖 1：なぜ NWP は太陽光発電の前日予測における唯一の信頼できる情報源なのか？

```
前提1（Yang Ch4.5.4）：日射量の時空間変動は移動する雲によってのみ生じる
    ↓
前提2（Yang Ch4.5.4）：全天カメラは雲が「凍結」していると仮定 → 30分後に破綻
前提3（Yang Ch4.5.4）：衛星 CMV は雲が等速直線運動すると仮定 → 4時間後に破綻
    ↓
前提4（Warner Ch2）：雲の動力学的発展を解けるのは原始方程式のみ
    ↓
前提5（Yang Ch5.6.2）：電力市場 DAM には >36 時間のリードタイムが必要
    ↓
結論：NWP は前日予測における唯一の信頼できる情報源
```

この推論連鎖は Warner Ch2 の原始方程式から出発し、カメラと衛星が前日スケールでなぜ失敗するかを説明します — それらの物理的仮定（凍結場／等速平流）は長時間後に成立しなくなり、NWP の原始方程式だけが雲の生成・発達・消散をシミュレートできるからです。

### 推論連鎖 2：NWP の予測誤差はどこから来るのか？

```
Warner Ch2 原始方程式 → 7 変数を解く必要がある
    ↓
問題1：方程式は非線形結合 → 解析解なし → 離散化が必要（Ch3）
    → 離散化により打ち切り誤差が生じる
    ↓
問題2：サブグリッド過程（乱流、雲微物理）は陽解的に解けない
    → パラメータ化が必要（Ch4-5）
    → パラメータ化は実際の物理の近似 → パラメータ化誤差が生じる
    ↓
問題3：初期条件は有限の観測ネットワークから得られる
    → 観測には測定誤差＋空間カバレッジ不足がある
    → 不完全な初期条件 → カオス増幅（Ch8）
    ↓
総誤差 = 打ち切り誤差 + パラメータ化誤差 + 初期条件誤差
```

**Yang 教材との接続**：Yang Ch3.1 は「予測誤差には2つの発生源がある：不完全な初期条件＋不完全な力学則」と述べています。Warner Ch2 はさらに「不完全な力学則」の具体的な意味を明確にします — 方程式自体に問題があるのではなく（原始方程式は正確です）、**求解方法**が誤差を導入する（離散化＋パラメータ化）のです。

### 推論連鎖 3：Reynolds 平均は日射量予測にどう繋がるのか？

```
Warner Ch2：大気にはサブグリッドの乱流がある → 直接解析できない
    ↓
Reynolds 分解：u = ū + u' → 平均後に共分散項 -ρ(u'w') が現れる
    ↓
共分散項 = Reynolds 応力 = 乱流の平均流への影響
    ↓
パラメータ化が必要（Ch4）→ 大気境界層（PBL）スキーム
    ↓
PBL スキームが鉛直混合強度を決定する
    ↓
分岐A：強い混合 → 水蒸気上昇 → 雲の形成 → GHI 減少
分岐B：弱い混合 → 安定成層 → 雲が少ない → GHI 増加
    ↓
Yang Ch7.1.2：PBL スキームは WRF-Solar で最も感度が高い選択の一つ
```

**核心的洞察**：Reynolds 平均という純粋に数学的な操作から4ステップの推論を経て、太陽光日射量予測の精度に直結する。これが「知識は孤島ではない」の実例です — 1895年の乱流分解手法が、2024年の太陽光予測品質を決定している。

### 推論連鎖 4：静力近似の選択は太陽光発電予測にどう影響するか？

```
Warner Ch2.3.1：静力近似は音波をフィルタリング → より大きな時間刻みが可能 → 計算が高速化
    ↓
ただし：Δx < 10 km では静力近似が成立しない
    ↓
GFS（全球モデル、~13 km）→ 静力近似を使用可能 → 粗い分解能だが全球をカバー
HRRR（地域モデル、3 km）→ 非静力方程式が必要 → 高分解能だが CONUS のみ
    ↓
Yang Ch6.6 の比較：HRRR の日射量予測は GFS を上回る
    ↓
理由の一つ：非静力方程式は対流セルを解像できる → より正確な雲シミュレーション
    ↓
ただし HRRR は米国のみをカバー → 他の地域は GFS または ECMWF に頼るしかない
    ↓
WRF-Solar の価値：ユーザーが独自に非静力地域モデルを実行できる
    ただし気象学の知識が必要（Yang Ch4.6.1 の学際的課題）
```

**実務的な意思決定**：米国で太陽光発電予測をする場合は HRRR（3 km 非静力）を使用する。中国では GFS/ECMWF（>10 km 静力）を使うか、自分で WRF-Solar を実行するしかない。この判断の理論的根拠が Warner Ch2.3 の静力近似です。

### 推論連鎖 5：分解能を2倍にするとコストが×8になるのはなぜか？（Ch3 より新規）

```
CFL 条件：U·Δt/Δx < 1
    ↓
Δx を半分にする → Δt も半分にしなければならない（さもなければ CFL 違反）
    ↓
2D 格子点数 ×4（面積は一定、各次元で×2）
    ↓
総計算量 = 格子点数 × 時間ステップ数 = 4 × 2 = 8
    ↓
実際にはさらに多い：鉛直層も細かくする場合がある → ×16 以上
```

**太陽光発電予測における意思決定の意味**：RTX 4050 上で WRF-Solar を実行する場合、9 km → 3 km への変更には約 27 倍の計算量が必要。精度向上がコストに見合うかを慎重に検討する必要がある。

### 推論連鎖 6：打ち切り誤差はどのように日射量まで伝播するのか？（Ch3 より新規）

```
有限差分による導関数の計算 → 打ち切り誤差（短波 6Δx で 17% の誤差）
    ↓
不正確な気圧傾度力 → 地衡風バイアス
    ↓
不正確な発散 → 鉛直運動のバイアス
    ↓
鉛直運動が凝結・蒸発を駆動 → 雲液水量のバイアス
    ↓
放射スキームが雲水を使って日射量を計算 → GHI 出力のバイアス
```

**核心的洞察**：日射量予測誤差は放射パラメータ化だけから来るのではなく、最も底層の導関数近似から積み上がっていく。Yang が「雲微物理がボトルネック」と言い、Warner が「数値手法も重要」と強調するのはこのためです — 両者は加算関係ではなく乗算関係にあります。

### 推論連鎖 7：なぜ「雲は日射量予測の最大のボトルネック」なのか？（Ch4 より新規）

```
放射パラメータ化の物理はすでに成熟している（Beerの法則 + Planck関数）
    ↓
放射スキームの入力 = 雲の三次元分布と光学的性質
    ↓
雲の光学的性質 ← 雲微物理（雲粒径分布）
    ↓
雲の存在 ← 対流パラメータ化（凝結がいつどこで引き起こされるか）
    ↓
対流への水蒸気供給 ← PBL パラメータ化（鉛直混合強度）
    ↓
境界層の熱力学的駆動力 ← 地表エネルギー収支（日射量の関数！）
    ↓
結論：日射量予測誤差の根源は放射スキーム自体ではなく、
「放射→地表→PBL→対流→微物理→雲→放射」というサイクル全体にある
```

**核心的洞察**：Yang が「雲微物理がボトルネック」と言い、Warner は「すべてのパラメータ化スキームは結合したサイクルを形成する」と補足します — 単一箇所の修正では解決しません。

### 推論連鎖 8：エアロゾルは太陽光発電予測にどう影響するか？（Ch4 より新規）

```
人為的・自然のエアロゾル → CCN 濃度の変化
    ↓
CCN 多 → 雲粒が多く小さい → 光学的厚さ↑ → アルベド↑ → GHI↓
CCN 少 → 雲粒が少なく大きい → 光学的厚さ↓ → 降水効率↑ → 雲の寿命が短い → GHI↑
    ↓
業務 NWP は通常エアロゾルをシミュレートしない（コストが高い）
しかし ECMWF は CAMS エアロゾルデータを同化 → 日射量予測が向上
    ↓
Yang Ch5.4 の REST2 は AOD(τ₅₅₀) を入力として必要とする
Yang Ch4.5.1 の晴天モデルは6つの透過率を掛け合わせ、そのうち Tₐ = エアロゾル減衰
    ↓
結論：エアロゾルの不確実性 → 晴天モデルの不確実性 + NWP 雲シミュレーションの不確実性
```

## Ⅴ. 解決済みの疑問

### ❓ Yang Ch7.1.1 は「7 本の原始方程式」に言及しているが、方程式が抽象的で各項の物理的意味が不明だった

✅ **Warner Ch2 がこの疑問を完全に解決した。** すべての方程式のすべての項に明確な物理的解釈がある：気圧傾度力が風を駆動し、コリオリ力が風を偏向させ、加熱率 $H$ は放射＋潜熱＋顕熱を含み、水蒸気の源・汚く項 $Q_v$ は蒸発と凝結を含む。

### ❓ Yang Ch7.1.2 は「パラメータ化は NWP で最も議論の多いテーマ」と言っているが、理由が不明だった

✅ **Warner Ch2 の Reynolds 平均が答えを与えてくれた。** パラメータ化の数学的起源は Reynolds 応力項です — これは本質的に**閉合問題**（未知数より方程式の数が少ない）です。どのパラメータ化スキームもこの閉合問題への一つの近似であり、「唯一の正解」は存在しません。だからこそ議論が尽きないのです。
