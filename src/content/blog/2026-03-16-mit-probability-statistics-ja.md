# MIT 確率統計の核心：ベイズ推論からMLEまで

> **カリキュラム体系**：MIT 18.05（確率・統計入門、Spring 2022）＋ MIT 18.650（応用統計、Fall 2016）
> **参考教材**：Grinstead & Snell *Introduction to Probability*；Murphy *Probabilistic Machine Learning* Book 1

## なぜ確率統計が太陽光発電予測に不可欠なのか？

太陽光発電予測の本質は**不確実性の定量化**にある：

- 「明日の正午の太陽光出力は 500kW」→ 確定論的予測（不十分）
- 「明日の正午の出力は 400〜600kW の範囲に 90% の確率で収まる」→ **確率的予測**（Yang 第11章の核心）

確率統計が提供するもの：
1. **モデリングツール**：確率分布、尤度関数、ベイズ推論
2. **推定手法**：MLE、モーメント法、最小二乗法
3. **検証フレームワーク**：仮説検定、信頼区間、p 値
4. **次元削減ツール**：PCA（統計的視点）

## Part 1：確率論の基礎（18.05 前半）

### 確率空間の三つ組

$$(\Omega, \mathcal{F}, P)$$

- $\Omega$：標本空間（すべての起こり得る結果）
- $\mathcal{F}$：事象の集合体（$\sigma$-代数）
- $P$：確率測度（$P(\Omega) = 1$、可算加法性）

### 条件付き確率とベイズの定理

$$P(A|B) = \frac{P(B|A)P(A)}{P(B)}$$

**3 つの解釈**：
- **頻度主義**：$P(A)$ ＝ 長期的な相対頻度
- **ベイズ主義**：$P(A)$ ＝ 信念の強さ、更新可能
- **Box の立場**：第7章で両者を論じつつ尤度主義を好む――「データに語らせよ」

### 重要な分布族

**離散分布**：
- Bernoulli($p$)：二値の結果（晴れ／曇り）
- Binomial($n, p$)：$n$ 回の試行における成功回数
- Poisson($\lambda$)：まれな事象の個数（機器故障回数）

**連続分布**：
- Normal($\mu, \sigma^2$)：**中心極限定理の主役**――Box ARIMA は $a_t \sim N(0, \sigma_a^2)$ を仮定
- Exponential($\lambda$)：待ち時間（故障間隔）
- Beta($\alpha, \beta$)：比率・確率の事前分布（晴れ指数 $k_t \in [0,1]$）
- Gamma($\alpha, \beta$)：正値の連続量（日射量、出力）

### 大数の法則と中心極限定理

**大数の法則**：$\bar{X}_n \xrightarrow{P} \mu$（標本平均が母平均に収束）

**中心極限定理**：$\sqrt{n}(\bar{X}_n - \mu) \xrightarrow{d} N(0, \sigma^2)$

**CLT は統計的推論の礎石**――元のデータが正規分布でなくても、標本平均の分布は正規分布に近づく。これが Box の残差正規性仮定が一般に妥当である理由だ。

## Part 2：パラメータ推定（18.650 第1〜4講）

### 最尤推定（MLE）

$$\hat{\theta}_{MLE} = \arg\max_\theta \mathcal{L}(\theta) = \arg\max_\theta \prod_{i=1}^n f(x_i|\theta)$$

実用上は対数尤度を最大化：$\hat{\theta}_{MLE} = \arg\max_\theta \ell(\theta) = \arg\max_\theta \sum_{i=1}^n \ln f(x_i|\theta)$

**MLE の優れた性質**：
- **一致性**：$\hat{\theta}_n \xrightarrow{P} \theta_0$
- **漸近正規性**：$\sqrt{n}(\hat{\theta}_n - \theta_0) \xrightarrow{d} N(0, I(\theta_0)^{-1})$
- **漸近有効性**：Cramér-Rao 下限を達成

**Fisher 情報量**：$I(\theta) = -E\left[\frac{\partial^2 \ell}{\partial \theta^2}\right]$ → パラメータ推定精度の理論的限界

### Box との完璧な対応

Box 第7章のパラメータ推定は**まさに** MLE：
- 条件付き MLE ＝ 条件付き最小二乗（残差平方和 $S^*(\phi, \theta)$ の最小化）
- Fisher 情報行列 ＝ ヘッシアン → パラメータ標準誤差 $\hat{V}(\hat{\xi}) \approx 2\hat{\sigma}_a^2 H^{-1}$
- Marquardt アルゴリズム ＝ 正則化ニュートン法（行列微積分 第9講！）

### モーメント法（矩推定）

$$\hat{\mu}_k = \frac{1}{n}\sum_{i=1}^n X_i^k \quad \text{（標本モーメント = 母モーメント）}$$

方程式系を解いてパラメータの推定値を求める。

**Box 第6章の Yule-Walker 推定はモーメント法の特殊ケース**：標本 ACF（モーメント）を理論 ACF（モーメント）に合わせることで AR パラメータを推定する。

モーメント法は単純だが MLE より効率的ではない――Box は MA パラメータのモーメント推定は**非効率**であり、反復 MLE を使わなければならないと強調している。

### ベイズ推定

$$p(\theta|x) = \frac{p(x|\theta)p(\theta)}{p(x)} \propto \text{尤度} \times \text{事前分布}$$

- **事前分布** $p(\theta)$：推定前の信念
- **尤度** $p(x|\theta)$：データが与える証拠
- **事後分布** $p(\theta|x)$：更新後の信念

**MAP 推定**：$\hat{\theta}_{MAP} = \arg\max_\theta p(\theta|x)$

**無情報事前分布のとき MAP ＝ MLE**（Box 第7章で確認済み）

**太陽光発電予測への示唆**：
- **事前知識**：物理的制約（出力 $\geq 0$、$\leq$ 設置容量）を事前分布として符号化できる
- **オンライン更新**：毎日新しいデータが届く → 事後分布のベイズ更新 → モデルの自己適応
- **不確実性の定量化**：事後分布から直接予測区間を取得（Box 第5章の等幅区間より柔軟）

## Part 3：仮説検定（18.650 第5〜6講）

### 基本的な枠組み

$$H_0: \theta = \theta_0 \quad \text{vs} \quad H_1: \theta \neq \theta_0$$

- **第一種過誤**（$\alpha$）：$H_0$ が真のとき棄却（誤警報）
- **第二種過誤**（$\beta$）：$H_1$ が真のとき棄却しない（見逃し）
- **検出力**（power） ＝ $1 - \beta$

### 尤度比検定

$$\Lambda = 2(\ell(\hat{\theta}_{MLE}) - \ell(\theta_0)) \xrightarrow{d} \chi^2(k)$$

**Box 第8章の Ljung-Box Q 検定は尤度比検定の変形**：
- $H_0$：残差がホワイトノイズ（モデルが十分）
- $\tilde{Q}(K) = n(n+2)\sum_{k=1}^K \frac{r_k^2}{n-k} \sim \chi^2(K-p-q)$

### 適合度検定

**Kolmogorov-Smirnov 検定**：経験 CDF と理論 CDF の最大乖離を比較する

**Box 第8章の累積ピリオドグラム検定**は KS 検定の周波数領域版――残差スペクトルが一様か否かを検定する（ホワイトノイズの特徴）。

### 情報量規準

$$AIC = -2\ell(\hat{\theta}) + 2k, \quad BIC = -2\ell(\hat{\theta}) + k\ln n$$

**Box 第6章**では AIC/BIC を用いて $p, q$ を選択――適合度とモデルの複雑さのバランスを取る。

## Part 4：回帰分析（18.650 第7講）

### 線形回帰の確率論的視点

$$Y_i = \mathbf{x}_i^T\boldsymbol{\beta} + \epsilon_i, \quad \epsilon_i \sim N(0, \sigma^2)$$

**MLE ＝ 最小二乗法**（誤差が正規分布のとき）：

$$\hat{\boldsymbol{\beta}} = (X^TX)^{-1}X^T\mathbf{y}$$

これはまさに MIT 18.06 Part 2 の直交射影の公式だ！

### 統計的推論

- **$\hat{\beta}_j$ の分布**：$\hat{\beta}_j \sim N(\beta_j, \sigma^2(X^TX)^{-1}_{jj})$
- **t 検定**：$t = \hat{\beta}_j / \text{se}(\hat{\beta}_j) \sim t(n-p)$
- **F 検定**：モデル全体の有意性

### 多重共線性

$X^TX$ がほぼ特異（条件数が大きい）なとき、$\hat{\boldsymbol{\beta}}$ の分散が爆発する。

**解決策**：
- リッジ回帰：$\hat{\boldsymbol{\beta}}_{ridge} = (X^TX + \lambda I)^{-1}X^T\mathbf{y}$（ベイズ解釈：正規事前分布）
- Lasso：$L_1$ 正則化 → 疎な解（特徴選択）
- PCA 回帰：次元削減してから回帰

**18.06 との接続**：条件数 $\kappa(X^TX) = \sigma_1^2/\sigma_r^2$ → リッジは小さい特異値を打ち切ることに相当

## Part 5：ベイズ統計（18.650 第8講）

### 共役事前分布

事前分布と尤度の積が同じ分布族に属するとき：

- 正規尤度 ＋ 正規事前分布 → 正規事後分布
- ベルヌーイ尤度 ＋ ベータ事前分布 → ベータ事後分布
- ポアソン尤度 ＋ ガンマ事前分布 → ガンマ事後分布

### MCMC サンプリング

事後分布を解析的に計算できない場合（ほとんどの場合）：
- **Metropolis-Hastings**：提案-採択/棄却
- **ギブスサンプリング**：各成分の条件付き分布からの逐次サンプリング
- **ハミルトニアン MC**：勾配情報を利用した効率的な探索（PyMC/Stan）

### ベイズ予測

$$p(y_{new}|x_{new}, \text{data}) = \int p(y_{new}|x_{new}, \theta) p(\theta|\text{data}) d\theta$$

点予測だけでなく、**完全な予測分布**が得られる！

**これが Yang 第11章の確率的予測の数学的基盤だ。**

## Part 6：PCA と GLM（18.650 第9〜10講）

### PCA の統計的視点

- 18.06 の PCA ＝ SVD の幾何学的視点
- 18.650 の PCA ＝ **標本共分散行列の固有値分解**

$$\hat{\Sigma} = \frac{1}{n-1}X^TX, \quad \hat{\Sigma}\mathbf{v}_k = \lambda_k\mathbf{v}_k$$

上位 $k$ 個の固有ベクトルを選ぶと、全分散の $\frac{\sum_{i=1}^k \lambda_i}{\sum_{i=1}^p \lambda_i}$ が保持される。

### 一般化線形モデル（GLM）

$$g(E[Y]) = \mathbf{x}^T\boldsymbol{\beta}$$

- **線形回帰**：$g$ ＝ 恒等関数、$Y$ は正規分布
- **ロジスティック回帰**：$g$ ＝ logit、$Y$ は二値
- **ポアソン回帰**：$g$ ＝ log、$Y$ はカウント

**太陽光発電への示唆**：
- 天気タイプの分類（晴れ／曇り／雨）→ 多項ロジスティック回帰
- 機器故障回数 → ポアソン回帰
- 設置容量での出力打ち切り → Tobit モデル（打ち切り回帰）

## 太陽光発電予測プロジェクトへの行動指針

1. **MLE をデフォルトに**：モデルのパラメータ推定には MLE を優先――シンプルかつ有効
2. **不確実性の定量化にはベイズ推論**：事後分布から直接予測区間を導出
3. **AIC/BIC でモデル選択**：ARIMA の $p,d,q$ だけでなく ML モデルのハイパーパラメータにも適用
4. **Ljung-Box は必須**：すべてのモデルで残差のホワイトノイズ性を検定
5. **PCA で次元削減**：気象特徴量行列の条件数が大きい場合は先に PCA を適用
6. **GLM フレームワーク**：予測目的に応じて異なるリンク関数を選択
7. **交差検証**：時系列にはローリングウィンドウ CV を使用――ランダム分割は不適
