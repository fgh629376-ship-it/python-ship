# MIT 18.063 行列微積分：導関数の線形作用素としての本質から誤差逆伝播まで

> **コース**：MIT 18.063 Matrix Calculus for ML and Beyond, IAP January 2026
> **講師**：Alan Edelman & Steven G. Johnson
> **前提知識**：18.06 線形代数 + 18.02 多変数微積分

## 核心的な考え方：導関数は「数」ではなく線形作用素である

### 従来の微積分の限界

18.01 では $f'(x) = \lim_{h \to 0} \frac{f(x+h) - f(x)}{h}$ と教わり、導関数は「数」とされる。

18.02 ではこれをベクトルに拡張し、勾配 $\nabla f$ はベクトル、ヤコビアン $J$ は行列となる。

**しかし入力も出力も行列である場合はどうか？** $f(X) = X^{-1}$ の「導関数」とは何か？

### 統一的な視点：導関数＝線形作用素

$$f(x + dx) - f(x) = df = f'(x)[dx] + o(\|dx\|)$$

$f'(x)$ は**線形作用素**であり、$dx$（微小な摂動）を入力として受け取り、$df$（微小な変化）を出力する。

**例**：
- $f(X) = X^2 \Rightarrow f'(X)[dX] = X \cdot dX + dX \cdot X$
- $f(X) = X^{-1} \Rightarrow f'(X)[dX] = -X^{-1} dX \, X^{-1}$
- $f(X) = \det(X) \Rightarrow f'(X)[dX] = \det(X) \cdot \text{tr}(X^{-1}dX)$

**これらはすべて行列 $dX$ に対する線形演算である**——行列をベクトルに「平坦化」してヤコビアンを掛ける必要はない！

**深層学習への意義**：ニューラルネットワークの重みは行列である。行列微積分は行列から行列への導関数を直接扱い、非効率なベクトル化操作を必要としない。

## Lec 1-2：基本的な規則

### 積の法則（行列版）

$$d(AB) = (dA)B + A(dB)$$

**順序は交換できない**（行列の乗算は可換ではない）ことに注意！

### スカラー関数の勾配

スカラー関数 $f(X)$ に対して、**フロベニウス内積** $\langle A, B \rangle = \text{tr}(A^T B)$ を使うと：

$$df = f'(X)[dX] = \langle \nabla f, dX \rangle = \text{tr}((\nabla f)^T \, dX)$$

$\nabla f$ と $X$ は**同じ形状**を持つ——これが PyTorch の `.grad` がパラメータテンソルと同じ形状を持つ理由である。

**主要な勾配公式**：

- $f(A) = \text{tr}(A) \Rightarrow \nabla f = I$
- $f(A) = \|A\|_F \Rightarrow \nabla f = A / \|A\|_F$
- $f(A) = \mathbf{x}^T A \mathbf{y} \Rightarrow \nabla f = \mathbf{x}\mathbf{y}^T$
- $f(A) = \det(A) \Rightarrow \nabla f = \det(A)(A^{-1})^T = \text{adj}(A)^T$
- $f(\mathbf{x}) = \mathbf{x}^T A\mathbf{x} \Rightarrow \nabla f = (A + A^T)\mathbf{x}$（$A$ が対称なら $= 2A\mathbf{x}$）

### クロネッカー積とベクトル化

$m \times n$ 行列を $mn \times 1$ ベクトルに「平坦化」する：$\text{vec}(X)$

$$\text{vec}(AXB) = (B^T \otimes A)\text{vec}(X)$$

$\otimes$ はクロネッカー積である。これにより行列関数のヤコビアンが通常の行列になるが——次元が爆発するため、理論解析にのみ有用である。

## Lec 3：連鎖律——順伝播と逆伝播

### 多次元連鎖律

$$\frac{d}{dx}[g(f(x))] = g'(f(x)) \circ f'(x)$$

線形作用素の合成 ＝ 行列の乗算。

### 順伝播モード（Forward Mode）

入力から出力へ、層ごとに $f'(x)[dx]$ を計算する：

$$dx \to df_1 = f_1'[dx] \to df_2 = f_2'[df_1] \to \cdots \to df_n$$

**各パスは1つの入力方向についてのみ微分する。** 入力が $n$ 個あれば、$n$ 回の順伝播が必要。

### 逆伝播モード（Reverse Mode）＝ 誤差逆伝播

出力から入力へ、層ごとに「随伴」（adjoint）を計算する：

$$\bar{f}_n = 1 \gets \bar{f}_{n-1} = f_n'^T[\bar{f}_n] \gets \cdots \gets \bar{f}_0 = \nabla f$$

**1回のパスで全入力に対する勾配を同時に計算できる！** 1回の逆伝播で完全な勾配が得られる。

### 重要な洞察

- **入力が多く、出力が少ない**場合（例：損失関数 $\mathbb{R}^{10^6} \to \mathbb{R}$）→ **逆伝播モードが効率的**
- **入力が少なく、出力が多い**場合（例：順シミュレーション）→ **順伝播モードが効率的**

**深層学習が誤差逆伝播を使う理由がここにある**：パラメータは $10^6$ 以上、損失はスカラー。

**太陽光発電予測への意義**：
- ニューラルネットワーク予測モデルの訓練 → 誤差逆伝播（PyTorch/JAX が自動で実行）
- 感度解析（1つのパラメータの変化が予測にどう影響するか）→ 順伝播モード
- 物理制約付き最適化（PINN）→ 混合モードが必要

## Lec 5-6：自動微分

### 双対数（Dual Numbers）——順伝播 AD

$\epsilon^2 = 0$（ゼロではなく「無限小」）となる $a + b\epsilon$ を定義する：

$$f(a + b\epsilon) = f(a) + f'(a) \cdot b\epsilon$$

**プログラム中の浮動小数点数を双対数に置き換えるだけで、自動的に導関数が得られる！**

```python
# 概念的な例示（Python）
class Dual:
    def __init__(self, val, deriv=0):
        self.val = val
        self.deriv = deriv
    def __mul__(self, other):
        return Dual(self.val * other.val,
                     self.val * other.deriv + self.deriv * other.val)
```

### 計算グラフ上の逆伝播 AD

ニューラルネットワーク ＝ 有向非巡回グラフ（DAG）。各ノードが1つの演算。

**順伝播**：入力から出力へ、各ノードの値を計算する
**逆伝播**：出力から入力へ、各ノードの「随伴」（vJP = vector-Jacobian product）を計算する

**重要**：現代の AD は**コンパイラ技術**であり、記号微分でも有限差分でもない。JAX の `jax.grad()` と PyTorch の `loss.backward()` はいずれもこれに基づいている。

## Lec 4：一般化勾配と内積

### 勾配の定義には内積が必要

任意のベクトル空間 $V$ 上で、スカラー関数 $f: V \to \mathbb{R}$ の導関数 $f'(x)$ は線形汎関数である。

**リース表現定理**により：任意の線形汎関数はあるベクトルとの内積として表せる：

$$f'(x)[dx] = \langle \nabla f, dx \rangle$$

$\nabla f \in V$ ——勾配は入力と「同じ形状」を持つ。

### 勾配＝最急上昇方向

$$\nabla f = \arg\max_{\|v\|=1} f'(x)[v]$$

**しかし「最急上昇」はノルムの選択に依存する！** 異なる内積は異なる勾配方向をもたらす。

これが**前処理**（プリコンディショニング）と**自然勾配**が有効な理由である——それらはより良い内積を用いて勾配を定義することに相当する。

**太陽光発電での例**：特徴量のスケールが大きく異なる場合（気温 ~300 K vs 日射量 ~1000 W/m²）、標準的な勾配降下法は方向によって収束速度が異なる。特徴量の標準化 ＝ 全方向を等しく重み付けするように内積を変更すること。

## Lec 6：変分法

### 関数空間上の微積分

「変数」が関数 $u(x)$ のとき、導関数は**変分導関数**（変分微分）になる：

$$\delta f[u] = f(u + \delta u) - f(u) = f'(u)[\delta u]$$

停留条件 $\nabla f = 0$ → **オイラー・ラグランジュ方程式**

**古典的な例**：
- 最短経路 → 測地線方程式
- 最小作用原理 → ラグランジュ力学
- 最適制御 → ハミルトン・ヤコビ・ベルマン方程式

**太陽光発電予測への意義**：
- Box Ch15 の最適制御理論は変分法の離散版
- PINN（物理情報ニューラルネットワーク）の損失関数 ＝ PDE 残差の積分 → 変分問題
- 蓄電池の最適スケジューリング ＝ 離散時間の最適制御問題

## Lec 7：複素微積分（CR 微積分）

### 問題：$f(z) = |z|^2 = z\bar{z}$ は微分不可能？

従来の複素解析はコーシー・リーマン条件（正則関数）を要求するが、$|z|^2$ はこれを満たさない。

### ヴィルティンガー導関数

$\frac{\partial}{\partial z} = \frac{1}{2}\left(\frac{\partial}{\partial x} - i\frac{\partial}{\partial y}\right)$、$\frac{\partial}{\partial \bar{z}} = \frac{1}{2}\left(\frac{\partial}{\partial x} + i\frac{\partial}{\partial y}\right)$ と定義する。

これにより**非正則関数に対しても「勾配」を定義できる**。

**機械学習への意義**：複素数値ニューラルネットワーク（信号処理・量子 ML に使用）の勾配計算には CR 微積分が必要。JAX の複素数 AD はヴィルティンガー導関数に基づいている。

## Lec 7+9：制約付き導関数と固有値の微分

### 固有値の導関数

$A(t)$ をパラメータ化された行列族、$\lambda(t)$ を固有値、$\mathbf{v}(t)$ を固有ベクトルとする：

$$\frac{d\lambda}{dt} = \frac{\mathbf{v}^T \frac{dA}{dt} \mathbf{v}}{\mathbf{v}^T \mathbf{v}}$$

（対称行列の場合、すなわち**ヘルマン・ファインマン定理**）

**注意**：固有値が重複すると導関数が存在しない場合がある！これは最適化における難点である。

### ヘッセ行列と二次近似

$$f(x + \delta x) \approx f(x) + \nabla f^T \delta x + \frac{1}{2}\delta x^T H \delta x$$

$H$ 正定値 → 局所最小値；$H$ 不定値 → 鞍点

**ニュートン法**：$\delta x = -H^{-1}\nabla f$（二次収束、ただしヘッセ行列が必要）

**準ニュートン法（BFGS）**：$H^{-1}$ を逐次近似し、完全なヘッセ行列を必要としない

## コース全体の大統一：行列微積分の知識マップ

```
導関数 = 線形作用素 f'(x)[dx]
    ↓
スカラー関数 → 勾配 ∇f（内積が必要）
行列関数 → ヤコビアン（クロネッカー積／ベクトル化が必要）
    ↓
連鎖律 = 線形作用素の合成
    ↓
順伝播モード（入力が少ない）  ←→  逆伝播モード（出力が少ない）
    ↓                                    ↓
双対数 AD                        誤差逆伝播 / vJP
    ↓                                    ↓
順伝播 AD ライブラリ              PyTorch / JAX
    ↓
二次導関数 → ヘッセ行列 → ニュートン法 / BFGS
    ↓
変分法 → オイラー・ラグランジュ → 最適制御 / PINN
```
