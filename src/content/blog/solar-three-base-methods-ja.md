# NWP vs 衛星 vs 地上データ：3大基礎手法の物理的本質

> 📖 本記事は教科書 *Solar Irradiance and Photovoltaic Power Forecasting*（Yang & Kleissl, 2024）第7章に基づき、実務エンジニアの視点で解説します。

太陽光発電予測における「基礎手法」とは、**後処理（Post-processing）前の一次予測を生成する手法**のことです。予測パイプライン全体の土台です。

3大基礎手法を重要度順に：

| 手法 | 有効予測範囲 | 核心物理 | 不可欠性 |
|------|-------------|---------|---------|
| **NWP** | 前日～数日 | 大気力学方程式の求解 | ⭐⭐⭐ 唯一の選択肢 |
| **衛星** | 15分～4時間 | 雲運動ベクトル移流 | ⭐⭐ 日内の重要補完 |
| **地上データ** | 1分～1時間 | 時系列統計／空間相関 | ⭐ 超短期微調整 |

---

## 1. NWP：大気を「解く」唯一の方法

### 1.1 7つの方程式、5つの物理法則

NWPの核心は**原始方程式群（Primitive Equations）**——7つの方程式で7つの変数（u, v, w, T, q, ρ, p）を解きます：

```
① ニュートンの第2法則 → 3つの運動量方程式（3次元風速の予測）
② 熱力学第1法則 → 温度予測方程式（位温θで定式化）
③ 水蒸気保存則 → 水蒸気予測（蒸発 - 凝結）
④ 質量保存則 → 連続の式
⑤ 理想気体の状態方程式 → 診断関係（時間微分なし）
```

### 1.2 パラメタリゼーション：NWPのアキレス腱

原始方程式はグリッドスケール（>10km）のプロセスのみ記述可能です。より小さなスケールの**サブグリッドプロセス**はパラメタリゼーションで近似する必要があります。

> ⚠️ **直感に反する発見**：放射パラメタリゼーションが最も関連性が高く見えますが、実はボトルネックではありません。放射伝達物理は十分に成熟しており、雲水量を地表日射量に変換するだけです。真のボトルネックは**雲微物理パラメタリゼーション**です。雲はマイクロメートルスケールで形成されますが、NWPグリッドはキロメートルスケール——このスケールギャップが日射量予測誤差の主因です。

### 1.3 離散化とデータ同化

- **有限差分法**：連続偏微分方程式を代数方程式へ変換
- **スペクトル法**：球面調和関数展開（ECMWF IFS は Tco1279 ≈ 31.4km）
- **CFL条件**：Δt ≤ Δx/c（安定性の制約）
- **4D-Var データ同化**：NWP予測（物理的整合性のある事前分布）と観測（証拠）の最適妥協点を探索

## 2. 衛星予測：4時間の窓

### 2.1 衛星画像から日射量への変換

**物理的手法（FARMS/NSRDB）**：`Gh = η·Gho + (1-η)·Ghc`——雲分率ηで加重。GOES雲属性 + MERRA-2大気 + MODIS等が必要。LUTで1000倍高速化。

**半経験的手法（Heliosat-2/SolarGIS）**：`Gh = g(ν)·Ghc`——雲指数νを晴天指数κにマッピング。可視光チャンネルのみで0.5km解像度。商用太陽エネルギー評価の主力。

**純ML手法**：Random Forest/CNN でチャンネルからGHIを直接回帰。精度は劣るが、成熟した雲属性製品がない新型衛星に有用。

### 2.2 雲運動ベクトル（CMV）

**Lucas-Kanade光流法**：輝度一定性制約 → `Ix·U + Iy·V = -It` → 近傍ピクセルで最小二乗解。

**ブロックマッチング（3ステップ探索）**：連続フレーム間で最大相互相関ブロックを探索。

> ⚠️ **4時間の限界**：「凍結雲場」仮定は15-30分後に劣化し始め、4時間後に完全に破綻します。衛星手法は**新しい雲を生成できません**——NWPのみが雲のライフサイクル全体をシミュレートできます。

## 3. 地上データ予測：時間次元の限界

### 3.1 参照予測（ベースライン）

```
持続性:        κ̂(t+h) = κ(t)
スマート持続性: ĜHI(t+h) = κ(t) × Ghc(t+h)
気候値:        κ̂(t+h) = mean(κ_historical)
CLIPER:       κ̂(t+h) = γ_h·κ(t) + (1-γ_h)·μ_κ
```

CLIPERは最強のシンプルベースライン：自己相関γ_hに基づき、持続性と気候値を最適に混合します。

### 3.2 情報理論的限界

どんなに高度なモデル（LSTM、Transformer）でも、**予測ホライズンhが大きくなるとγ_hがゼロに近づく**という基本的事実から逃れられません。情報のない入力から情報を抽出することは不可能です。

### 3.3 時空間手法

STAR、クリギング、グラフニューラルネットワーク（GNN）は多地点の空間相関を活用しますが、空間相関も距離と時間で急速に減衰します。4時間を超えると、NWPのみが新情報を提供できます。

## 4. エンジニアリング戦略：3手法の融合

```
最適パイプライン:
  超短期 (0-30分)    → 地上データ（リアルタイム修正）
  日内   (30分-4時間) → 衛星CMV（雲運動の捕捉）
  前日   (4時間-3日)  → NWP（唯一の選択肢）
  数日   (3日+)      → NWPアンサンブル予報

全ホライズン共通:
  → 晴天モデル正規化（κ = GHI/Ghc）
  → 後処理（MOS/ML）
  → 確率的キャリブレーション
```

「どれを選ぶか」ではなく、**「どのホライズンでどれを使うか」**の問題です。
