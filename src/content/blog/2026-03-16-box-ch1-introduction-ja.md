# Box 時系列分析 Ch1：5つの応用分野と ARIMA フレームワークの起源

> **教科書**：George E.P. Box, Gwilym M. Jenkins, Gregory C. Reinsel — *Time Series Analysis: Forecasting and Control*, 4th Edition (Wiley, 2008)

## なぜこの本を読むのか？

Box-Jenkins 方法論は時系列分析の礎石である。金融予測、工業プロセス制御、太陽光発電予測のいずれにおいても——ARIMA、伝達関数、介入モデルといったツールはすべて Box の名前に結びついている。これは「ARIMA を紹介するチュートリアル」ではなく、**ARIMA を発明した本人がなぜそれを発明したのかを語る書**である。

第1章は18ページを使って一つの核心的な問いに答える：**時系列分析はどのような問題を解決するのか？**

---

## 1.1 5つの応用分野

Box は最初から数式を並べるのではなく、5つの実際的な問題から出発し、各問題に対応する数理モデルの類型を示す。この「問題駆動型」の記述スタイルが本書最大の特徴である。

### 1.1.1 時系列予測（Forecasting）

**問題**：$z_t, z_{t-1}, z_{t-2}, \ldots$ が既知のとき、将来の $z_{t+l}$ を予測する。ここで $l$ はリードタイム（先行期間）。

**重要な概念**：
- **予測関数** $\hat{z}_t(l)$：時刻 $t$、リードタイム $l$ での最適予測
- **目標**：平均二乗誤差 $E[(z_{t+l} - \hat{z}_t(l))^2]$ の最小化
- **確率的信頼区間**：予測は一点ではなく区間（50%、95% 確率帯）

**太陽光発電予測への示唆**：太陽光発電量予測はまさにこの問題である——過去の日射量/発電量系列から、1時間後、24時間後、72時間後の出力を予測する。Box が教えてくれるのは、最適予測の導出には**既知の形式を持つ確率モデルが必要不可欠**であり、そのためにモデル同定（Ch6）、パラメータ推定（Ch7）、診断検定（Ch8）が Box-Jenkins 方法論の3段階サイクルを構成しているということだ。

### 1.1.2 伝達関数推定（Transfer Function）

**問題**：入力 $X_t$ と出力 $Y_t$ の間に動的線形関係 $Y_t = \sum_{j=0}^{\infty} v_j X_{t-j}$ が存在する場合、インパルス応答関数 $\{v_j\}$ をどう推定するか？

Box はガス炉の例を使った：入力は空気供給速度 $X_t$、出力は CO₂ 濃度 $Y_t$、観測間隔は9秒。重要な難点は**ノイズ**——実システムでは入出力関係は常に制御不能な外乱に隠される。

**太陽光発電への示唆**：日射量→発電量は典型的な伝達関数問題である。NWP 予報→実際の日射量も同様。伝達関数モデルにより「入力の変化がどれだけの遅延で、どの程度出力に影響するか」を定量化できる——太陽光システムでは、日射量変化後の発電応答の遅延と減衰特性にあたる。

### 1.1.3 異常介入事象分析（Intervention Analysis）

**問題**：ある外部事象（政策変更、機器故障、異常気象）が特定時刻に発生したことが既知の場合、時系列への影響をどう定量化するか？

Box と Tiao (1975) は介入モデルを用いてロサンゼルスの大気汚染規制がオゾン濃度に与えた影響を研究した。入力は 0/1 のみをとる指示変数である。

**太陽光発電への示唆**：
- インバータ故障、パネル清掃、積雪 → 介入事象
- 系統政策変更（出力抑制、給電指令）→ 介入事象
- Ch13 の介入分析により、これらの異常値を**自動検出・補正**し、モデルパラメータ推定の汚染を防ぐことができる

### 1.1.4 多変量時系列分析（Multivariate）

**問題**：$k$ 個の関連する時系列 $\mathbf{Z}_t = (z_{1t}, z_{2t}, \ldots, z_{kt})'$ の同時モデリング。

2つの目的：
1. 複数系列間の**動的関係**を理解する
2. 関連系列の情報を利用して**予測精度を向上**させる

**太陽光発電への示唆**：太陽光システムは本質的に多変量——GHI、DNI、DHI、気温、風速、湿度、発電量——これらの間に複雑な動的結合が存在する。Ch14 の VAR モデルにより全変数を同時にモデリングし、変数間の Granger 因果関係を捉えることができる。

### 1.1.5 離散制御システム（Discrete Control）

**問題**：入力変数 $X_t$ を調整することで、出力 $Y_t$ を目標値 $T$ にできるだけ近づける。

Box は2つの制御哲学を区別した：
- **SPC（統計的プロセス管理）**：Shewhart 管理図、CUSUM 図——受動的監視で異常を発見
- **EPC（工学的プロセス制御）**：フィードバック/フィードフォワード制御——能動的調整で外乱を補償

重要な洞察：**予測問題と制御問題は密接に関連している**。最小平均二乗誤差予測は直接的に最小平均二乗誤差制御を導く。

**太陽光発電への示唆**：太陽光発電所の無効電力補償、蓄電池の充放電戦略、インバータの MPPT 制御——すべて EPC 問題である。Ch15 の制御理論と予測理論の統一フレームワークは、「予測駆動型ディスパッチ」を理解するための数学的基盤を提供する。

## 1.2 確率モデルの数理的枠組み

### 決定論的 vs 確率論的

Box は冒頭で重要な区別を行う：
- **決定論的モデル**：物理法則による正確な計算（ミサイルの軌道）
- **確率モデル（確率論的モデル）**：将来の値がある区間に入る確率を計算

**時系列 ≠ 確率過程**。時系列 $z_1, z_2, \ldots, z_N$ は無限確率過程の**一つの標本実現**（sample realization）である。統計的推測の目標は、この一回の実現から母集団の性質を推測することだ。

### 線形フィルタモデル

これが本書全体の数学的出発点である。観測可能な時系列 $z_t$ は、白色雑音 $a_t$（独立同分布、平均 0、分散 $\sigma_a^2$）が線形フィルタを通過した出力とみなせる：

$$z_t = \mu + a_t + \psi_1 a_{t-1} + \psi_2 a_{t-2} + \cdots = \mu + \psi(B) a_t$$

ここで $\psi(B) = 1 + \psi_1 B + \psi_2 B^2 + \cdots$ は伝達関数、$B$ は後退オペレータ（$Bz_t = z_{t-1}$）。

**物理的直観**：時系列における観測値間の依存性は、独立なランダムショックがシステムの「記憶」（重み $\psi_j$）を通じて蓄積されることで生じる。

**定常条件**：$\sum_{j=0}^{\infty} |\psi_j| < \infty$ ならフィルタは安定、過程は定常であり、$\mu$ は過程の平均となる。

### 3つの基本モデル

#### AR(p) — 自己回帰モデル

$$\tilde{z}_t = \phi_1 \tilde{z}_{t-1} + \phi_2 \tilde{z}_{t-2} + \cdots + \phi_p \tilde{z}_{t-p} + a_t$$

後退オペレータを使うと：$\phi(B)\tilde{z}_t = a_t$

- $p+2$ 個の未知パラメータ：$\mu, \phi_1, \ldots, \phi_p, \sigma_a^2$
- **定常条件**：$\phi(B) = 0$ のすべての根が単位円の外にある
- AR は線形フィルタの特殊ケース：$\tilde{z}_t = \phi^{-1}(B)a_t = \psi(B)a_t$

**AR(1) の展開**：$\tilde{z}_t = \phi \tilde{z}_{t-1} + a_t$ → 再帰的に代入 → $\tilde{z}_t = \sum_{j=0}^{\infty} \phi^j a_{t-j}$（$|\phi| < 1$ のとき収束）。これが「AR は無限 MA として表現できる」ことを理解する鍵である。

#### MA(q) — 移動平均モデル

$$\tilde{z}_t = a_t - \theta_1 a_{t-1} - \theta_2 a_{t-2} - \cdots - \theta_q a_{t-q} = \theta(B) a_t$$

- $q+2$ 個のパラメータ
- 「移動平均」という名前は誤解を招きやすい——重みは正である必要も、合計が1である必要もない

#### ARMA(p,q) — 混合モデル

$$\phi(B)\tilde{z}_t = \theta(B)a_t$$

- $p+q+2$ 個のパラメータ
- 等価な線形フィルタ形式：$\tilde{z}_t = \psi(B)a_t$、ここで $\psi(B) = \phi^{-1}(B)\theta(B)$
- 実用上、$p, q$ は通常 $\leq 2$

### 非定常への拡張：ARIMA(p,d,q)

多くの実データ系列（株価、売上）は固定された平均の周りで変動しない。Box の重要な洞察：**差分により非定常系列を定常系列に変換できる**。

一般化自己回帰オペレータ $\varphi(B)$ が $d$ 個の単位根を持つ場合：

$$\varphi(B) = \phi(B)(1-B)^d$$

$w_t = \nabla^d z_t = (1-B)^d z_t$ と定義すると：

$$\phi(B) w_t = \theta(B) a_t$$

これが ARIMA(p,d,q) モデルである。$d$ は通常 0、1、最大でも 2。

**「積分」の意味**：ARIMA の "I" は Integrated（積分/総和）を表す。$z_t = S^d w_t$ であり、ここで $S = \nabla^{-1} = (1-B)^{-1} = 1 + B + B^2 + \cdots$ は総和オペレータ。定常 ARMA 過程 $w_t$ を $d$ 回累積加算することで非定常 ARIMA 過程 $z_t$ が得られる。

### 伝達関数モデル（ノイズあり）

入出力関係にノイズを加えたもの：

$$Y_t = v(B)X_t + N_t = \delta^{-1}(B)\omega(B)X_t + \varphi^{-1}(B)\theta(B)a_t$$

同時に推定する必要がある：
1. 動的関係の伝達関数 $v(B)$
2. ノイズモデルの伝達関数 $\psi(B)$

### 制御システムモデル

誤差 $\varepsilon_t = Y_t - T = v(B)X_t + N_t - T$

制御方程式：$x_t = \zeta_1 x_{t-1} + \zeta_2 x_{t-2} + \cdots + \chi_0 \varepsilon_t + \chi_1 \varepsilon_{t-1} + \cdots$

**予測と制御の統一**：最小 MSE 予測 → 最小 MSE 制御。

## 1.3 モデル構築の基本思想

### 節約の原則（Parsimony）

> 「できるだけ少ないパラメータで十分な表現を得る」

Box は例を挙げる：本来 2 パラメータ $(1-\delta B)Y_t = \omega_0 X_t$ で表すべきシステムを、$s+1$ パラメータの多項式でフィッティングすると、データ情報を浪費するだけでなく推定精度も悪化する。

**太陽光発電への示唆**：これは過学習への警告である。深層学習の時代には節約の原則が特に見落とされやすい——100層 Transformer は限られたデータでは ARIMA(1,1,1) に勝てないかもしれない。

### 反復的3段階モデル構築法

これが有名な Box-Jenkins 方法論の核心である：

1. **同定（Identification）**：ACF/PACF などのツールでモデル構造（$p, d, q$）を暫定的に決定
2. **推定（Estimation）**：最尤法または最小二乗法でパラメータを推定
3. **診断検定（Diagnostic Checking）**：残差分析によりモデルの適切性を検証

不合格なら第1段階に戻り、満足のいくモデルが見つかるまで繰り返す。

Box は特に強調する：最低 **50個の観測値**、できれば **100個以上**が必要。

## 深い考察：Ch1 の太陽光発電予測への示唆

### 1. 太陽光発電における5つの分野の統合

太陽光発電予測は実際には Box の5つの応用分野すべてに同時に関わっている：

| Box の応用分野 | 太陽光発電の対応場面 |
|---------------|-------------------|
| 予測 | 1h/24h/72h 発電量予測 |
| 伝達関数 | 日射量→発電量の動的応答 |
| 介入分析 | 機器故障、清掃、積雪検出 |
| 多変量時系列 | GHI+気温+風速+湿度の同時モデリング |
| 制御システム | 蓄電池充放電ディスパッチ、MPPT 制御 |

つまり Box のフレームワークは単に「ARIMA を学ぶ」だけでなく、**データから意思決定までの完全なパイプライン**を提供する。

### 2. 確率モデル vs 物理モデル

Box は決定論的モデルと確率モデルを区別した。太陽光発電分野では：
- **物理モデル**（pvlib）：放射伝達方程式に基づく、決定論的
- **統計モデル**（ARIMA）：過去データに基づく、確率論的
- **ベストプラクティス**：物理モデルでベースラインを与え、統計モデルで残差を捕捉する——これが「物理＋統計」ハイブリッド予測フレームワークの理論的基盤

### 3. 節約の原則の現代的価値

2026年の深層学習全盛の時代に、1970年に Box が提唱した節約の原則を再読するのは格別に示唆に富む：
- 太陽光発電データは通常限られている（1発電所の数年分 ≈ 数千個の日次サンプル）
- パラメータが多すぎるモデルは分布シフト時により脆弱
- ARIMA の 3〜5 パラメータ vs Transformer の数百万パラメータ——少データの場面では前者がより信頼できるかもしれない

### 4. 反復的モデリングの工学的意義

Box-Jenkins の3段階法は学術的方法論にとどまらず、一つの**エンジニアリング思考**である：
- 一度で完璧なモデルを見つけようとしない
- 素早く反復する：同定 → 推定 → 検定 → 修正
- モデルは道具であって教条ではない——ARIMA(1,1,1) の残差が検定を通過すれば、ARIMA(3,2,3) は必要ない

## まとめ

Ch1 はロードマップである。Box は18ページで以下を伝える：
1. 時系列分析は5種類の実問題を解決することを目指す
2. これらすべての問題は**線形フィルタ＋白色雑音**の枠組みでモデル化できる
3. ARIMA は定常・非定常系列の両方を扱う統一的ツールである
4. モデル構築は反復的かつ節約的であるべきだ
