> 結論を暗記するのではなく、推論プロセスを鍛える。各章を読むたびに「なぜそうなのか？」「他の知識とどう繋がっているか？」を問い続ける。

## 一、章横断的な推論チェーン

### 推論チェーン 1：白色雑音から ARIMA への論理的連鎖
**Ch1 線形フィルタモデル → Ch3 ARMA → Ch4 ARIMA**

1. 観測可能なあらゆる時系列は、白色雑音 $a_t$ が線形フィルタを通った出力として見ることができる：$z_t = \psi(B)a_t$
2. $\psi(B)$ が有理関数 $\phi^{-1}(B)\theta(B)$ として表せるなら、ARMA が得られる：$\phi(B)\tilde{z}_t = \theta(B)a_t$
3. 系列が非定常でも差分後に定常であれば、$(1-B)^d$ を導入して ARIMA を得る：$\phi(B)\nabla^d z_t = \theta(B)a_t$

**核心的洞察**：なぜ有理関数近似で十分か？→ 簡約原理。Padé 近似理論によれば、低次有理関数（分子・分母ともに ≤ 2 次）は非常に複雑な無限級数を高精度で近似できる。

### 推論チェーン 2：予測と制御の統一
**Ch1.1.1 予測 + Ch1.1.5 制御 → Ch5 + Ch15**

1. 予測問題：$E[(z_{t+l} - \hat{z}_t(l))^2]$ を最小化する
2. 制御問題：$E[\varepsilon_t^2]$（$\varepsilon_t = Y_t - T$）を最小化する
3. 制御 = 「制御なし時の偏差」を予測 + 補正調整量を計算
4. **最小 MSE 予測 → 最小 MSE 制御**

**核心的洞察**：この統一は偶然ではない——両者とも条件付き期待値の最適性に帰着する。$\hat{z}_t(l) = E[z_{t+l} | z_t, z_{t-1}, \ldots]$ は条件付き期待値であり、条件付き期待値は $L^2$ 空間における正射影であり、自然に MSE を最小化する。

### 重要な教科書横断的洞察

**Box の線形フィルタリング ↔ Warner の数値フィルタリング**：
- Box の $\psi(B)$ 重み系列はデジタルフィルタのインパルス応答
- Warner Ch4 の Robert-Asselin 時間フィルタは特定の AR 型フィルタ
- NWP における陰的時間差分スキーム（例：半陰的法）は、周波数領域での高周波成分への特定の $\psi$ 重みの適用と等価

**Box の ARIMA 差分 ↔ Yang の去トレンド**：
- ARIMA は $\nabla^d = (1-B)^d$ でトレンドを除去——「盲目的」な去トレンド（トレンドの形を仮定しない）
- Yang Ch7 は快晴モデルで物理的に去トレンド：$\text{anomaly}_t = \text{actual}_t - \text{clearsky}_t$
- **ベストプラクティス**：物理的去トレンド（Yang）+ 統計的差分（Box）は相補的——物理モデルが予測可能なトレンドを除き、統計手法が残余の非定常性を処理する

## 三、解決済みの疑問

### 疑問 1：MA モデルでなぜ「マイナス符号」を使うのか？（Ch1）
**問い**：AR モデルは $\tilde{z}_t = \phi_1 \tilde{z}_{t-1} + a_t$（プラス符号）と書くのに、なぜ MA モデルは $\tilde{z}_t = a_t - \theta_1 a_{t-1}$（マイナス符号）と書くのか？

**解答**：これは AR と MA の双対関係をより簡潔にするための約定（convention）である。MA がプラス符号を使うと、AR↔MA の変換公式に余分なマイナス符号が生じる。マイナス符号の約定を使えば：
- AR(1)：$\phi(B) = 1 - \phi_1 B$
- MA(1)：$\theta(B) = 1 - \theta_1 B$

両者の形式が完全に対称となり、$\phi(B)\tilde{z}_t = \theta(B)a_t$ が美しくなる。

### 疑問 2：ARIMA の「I」はなぜ「積分」と呼ばれるのか？（Ch1）
**問い**：差分 $\nabla^d$ が非定常を定常にするなら、「積分」と差分にはどんな関係があるのか？

**解答**：名前は**逆演算**に由来する。$\nabla = (1-B)$ は差分演算子；その逆 $S = \nabla^{-1} = (1-B)^{-1} = 1 + B + B^2 + \cdots$ は和算（summation）、つまり離散的な「積分」である。ARIMA 過程 $z_t$ は定常 ARMA 過程 $w_t$ を $d$ 回和算／積分して得られる：$z_t = S^d w_t$。

### 疑問 3：なぜ簡約原理がこれほど重要なのか？（Ch1.3）
**問い**：現代の計算能力はこれほど高いのに、なぜもっと多くのパラメータを使えないのか？

**解答**：Box の洞察は計算能力の問題を超えている。核心的な理由は**データ量が有限**であることだ：
1. パラメータが多いほど、必要なデータ量は超線形的に増大する（次元の呪い）
2. 有限データではパラメータ間の関係を同定するのが難しい（Box の $\omega_0, \omega_1, \ldots, \omega_s$ の例）
3. 過学習したモデルは予測時の汎化能力が低い

2026年のディープラーニング時代でも、一つの発電所の3年間の15分データ（〜10万点）しかない場合、ARIMA(1,1,1) の5つのパラメータは10万パラメータのモデルより信頼性が高いかもしれない。

### 推論チェーン 3：正定値性の二つの顔——時間領域の制約と周波数領域の保証
**Ch2.1.3 自己共分散行列の正定値性 ↔ Ch2.2.3 スペクトルの非負性**

1. 自己共分散行列 $\boldsymbol{\Gamma}_n$ は正定値でなければならない（任意の線形結合 $L_t$ の分散が > 0 だから）
2. 正定値性は $\rho_k$ の取り得る値の範囲を制約する（任意に決められない！）
3. 等価な条件：スペクトル密度 $p(f) \geq 0$ がすべての $f$ で成立
4. これが Herglotz の定理の内容

**核心的洞察**：正定値性は「合法的な ACF」の必要十分条件である。任意に $\{\rho_k\}$ 系列を構築しても、実際の確率過程に対応するとは限らない——正定値性の確認が必要。プログラミングでは、データから推定した ACF を打ち切った後に正定値化処理が必要になることを意味する。

### 推論チェーン 4：Bartlett の公式はなぜ真の ρ を必要とするのか？
**Ch2.1.6 標準誤差 → Ch6 モデル同定の困難**

1. $\text{var}[r_k]$ の公式には真の $\rho_v$（未知！）が含まれている
2. 実際には推定値 $r_v$ で代用するしかない → 追加の不確実性が生じる
3. 白色雑音（$q=0$）の場合は $1/\sqrt{N}$ という簡潔な公式があるが、非白色雑音の場合はブートストラップが必要
4. **これがモデル同定が「純粋な科学」ではなく「アート」である理由**——ACF の不確実性が $p, q$ の正確な判断を困難にする

### 推論チェーン 5：ペリオドグラム → 分散分析 → スペクトル推定の発展
**Ch2.2.1–2.2.3 離散から連続へ**

1. ペリオドグラム：分散を $N/2$ 個の離散周波数に分解（フーリエ級数）
2. 分散分析：各周波数が自由度 2 を持ち、$\chi^2(2)$ 分布（白色雑音の場合）
3. 標本スペクトル：連続周波数を許容するが、**激しく変動する**（本質的に周波数窓が狭すぎる）
4. 平滑スペクトル推定：窓関数 $\lambda_k$ または $W(f_j)$ を導入し、分解能を犠牲にして安定性を得る

**NWP との対比**：これは Warner Ch4 のスペクトル解析と完全に対応している——NWP は FFT で数値解の周波数特性を解析し計算ノイズを検出；Box は同じツールで時系列の周波数特性を解析し周期成分を識別する。

## 三B、解決済みの疑問（Ch2）

### 疑問 4：ACF 推定でなぜ $1/(N-k)$ ではなく $1/N$ を使うのか？（Ch2.1.5）
**問い**：$c_k = \frac{1}{N}\sum_{t=1}^{N-k}(z_t-\bar{z})(z_{t+k}-\bar{z})$ は $N$ を分母として使うが、実際の和は $N-k$ 項しかない。

**解答**：$1/N$ を使うには二つの利点がある：
1. 推定 ACF 行列が半正定値であることを保証する（$1/(N-k)$ では保証されない）
2. 大標本では両者は漸近的に等価
3. Box は Jenkins & Watts [170] の系統的比較を引用しており、結論は $1/N$ が優れているというもの

### 疑問 5：なぜ標本スペクトルは「激しく変動する」のか？（Ch2.2.3）
**問い**：$I(f)$ がスペクトルの推定として不良なのはなぜか？

**解答**：$I(f)$ の分散は **$N$ が増大しても減少しない**（直感に反する！）。理由は各 $I(f_i) \sim \sigma^2\chi^2(2)$ であり、$\chi^2(2)$ の変動係数が 100% で $N$ の大きさに無関係だから。解決策は平滑化——隣接する周波数の $I(f)$ を平均化し、自由度を増やす代わりに周波数分解能を失う。

### 推論チェーン 6：Wold 分解 → ARMA の正当性 → Padé 近似
**Ch3.1 Wold の定理 → Ch3.1.4 有理関数によるパラメータ化**

1. Wold の定理：任意の定常過程 = 白色雑音の線形フィルタ（$\tilde{z}_t = \psi(B)a_t$）
2. $\psi(B)$ は無限多項を持つ可能性がある → 操作不能
3. 有理関数 $\psi(B) = \theta(B)/\phi(B)$ で近似 → ARMA(p,q)
4. これが Padé 近似——低次有理関数は同次数の多項式より優れた近似を与える
5. **簡約原理の数学的根拠**：$p+q \leq 4$ の ARMA は実践ではほぼ常に十分

### 推論チェーン 7：ACF/PACF の打ち切り = モデルの「指紋」
**Ch3.2–3.4 → Ch6 モデル同定**

1. AR(p)：PACF がラグ $p$ で打ち切り（$\phi_{kk}$ は $k$ 次回帰の最後の係数であり、$k > p$ でゼロ）
2. MA(q)：ACF がラグ $q$ で打ち切り（ラグ $> q$ の自己共分散は重複しないショックを含むため共分散がゼロ）
3. ARMA(p,q)：両方が裾を引く（ただしパターンは識別可能：ACF はある lag 以降で φ の差分方程式に支配される）
4. **実際の操作**：ACF/PACF を描く → 打ち切りパターン → 初期次数同定 → 推定 → 診断

**これが Ch6 モデル同定の全論理的基盤である。**

## 三C、解決済みの疑問（Ch3）

### 疑問 6：なぜ可逆性が重要なのか？（Ch3.1.3）
**問い**：MA(1) は $\theta_1$ がどんな値でも定常なのに、なぜ $|\theta_1| < 1$ に制限するのか？

**解答**：可逆性は**現在値を過去値で合理的に表現できること**を保証する。$|\theta_1| \geq 1$ の場合、$\pi_j = -\theta_1^j$ が発散し——現在の $z_t$ が遠い過去の $z_{t-k}$ に対してますます大きな依存重みを持つことになり、物理的に不合理。

より深い理由：ある ACF が与えられると、それを生成できる MA(1) モデルは二つある（$\theta_1$ と $\theta_1^{-1}$）；可逆性条件が唯一の解を選ぶ。

### 疑問 7：AR(2) の「擬似周期」とはどういう意味か？（Ch3.2.4）
**問い**：AR(2) は周期的な挙動を生成できるか？本物の周期性との違いは何か？

**解答**：AR(2) の複素根が生成する ACF は**減衰正弦波** $\rho_k = D^k \sin(2\pi f_0 k + F)/\sin F$ であり、固定振幅の正弦波ではない。物理的には「準周期」——各「サイクル」は前のものより弱い。

真の周期性（日周期、年周期など）は**決定論的成分**であり、フーリエ項や季節差分（Ch9）で処理する必要があり、AR の複素根だけでは捉えられない。AR(2) の擬似周期は「天気システムの約3〜7日準周期」のような統計的規則性を記述するのに適している。
