# Box 時系列分析 — 思考トレーニング

> 結論を暗記するのではなく、推論プロセスを鍛える。各章を読むたびに「なぜこうなるのか？」「他の知識とどう関係するのか？」と問い続ける。

## 一、章横断的な推論チェーン

### 推論チェーン 1：白色雑音から ARIMA への論理的道筋
**Ch1 線形フィルタモデル → Ch3 ARMA → Ch4 ARIMA**

1. すべての観測可能な時系列は白色雑音 $a_t$ が線形フィルタを通過した出力とみなせる：$z_t = \psi(B)a_t$
2. $\psi(B)$ が $\phi^{-1}(B)\theta(B)$（有理関数）として表現できれば ARMA が得られる：$\phi(B)\tilde{z}_t = \theta(B)a_t$
3. 系列が非定常だが差分後に定常になるなら、$(1-B)^d$ を導入して ARIMA が得られる：$\phi(B)\nabla^d z_t = \theta(B)a_t$

**重要な推論**：なぜ有理関数近似で十分なのか？→ 節約の原則。Padé 近似理論によれば、低次の有理関数（分子＋分母がそれぞれ2次以下）で非常に複雑な無限級数を高精度に近似できる。

### 推論チェーン 2：予測と制御の統一
**Ch1.1.1 予測 + Ch1.1.5 制御 → Ch5 + Ch15**

1. 予測問題：$E[(z_{t+l} - \hat{z}_t(l))^2]$ を最小化する
2. 制御問題：$E[\varepsilon_t^2]$ を最小化する。ここで $\varepsilon_t = Y_t - T$
3. 制御 = 「制御なしの場合の偏差」の予測 + 補償調整量の計算
4. **最小 MSE 予測 → 最小 MSE 制御**

**重要な推論**：この統一は偶然ではない——両者とも条件付き期待値の最適性に帰着する。$\hat{z}_t(l) = E[z_{t+l} | z_t, z_{t-1}, \ldots]$ は条件付き期待値であり、条件付き期待値は $L^2$ 空間における直交射影で、本質的に MSE を最小化する。

### 重要な教科書横断的洞察

**Box の線形フィルタ ↔ Warner の数値フィルタ**：
- Box の $\psi(B)$ 重み系列はデジタルフィルタのインパルス応答そのもの
- Warner Ch4 の Robert-Asselin 時間フィルタは特定の AR 型フィルタ
- NWP における陰的時間差分スキーム（半陰的手法など）は周波数領域で高周波成分に特定の $\psi$ 重みを適用することと等価

**Box の ARIMA 差分 ↔ Yang のトレンド除去**：
- ARIMA は $\nabla^d = (1-B)^d$ でトレンドを除去する——「ブラインド」なトレンド除去（トレンドの形を仮定しない）
- Yang Ch7 は晴天モデルの物理によるトレンド除去：$\text{anomaly}_t = \text{actual}_t - \text{clearsky}_t$
- **ベストプラクティス**：物理的トレンド除去（Yang）＋ 統計的差分（Box）は相補的——物理モデルで予測可能なトレンドを除去し、統計的手法で残余の非定常性を処理する

## 三、解決済みの疑問

### 疑問 1：なぜ MA モデルでは「マイナス符号」を使うのか？（Ch1）
**質問**：AR モデルは $\tilde{z}_t = \phi_1 \tilde{z}_{t-1} + a_t$（プラス記号）と書くのに、なぜ MA モデルは $\tilde{z}_t = a_t - \theta_1 a_{t-1}$（マイナス記号）と書くのか？

**回答**：これは慣例（convention）であり、AR と MA の双対関係をより簡潔にするためのものだ。MA にプラス記号を使うと、AR↔MA の変換公式に余計なマイナス記号が生じる。マイナス記号の慣例を採用すると：
- AR(1)：$\phi(B) = 1 - \phi_1 B$
- MA(1)：$\theta(B) = 1 - \theta_1 B$

両者は完全に対称的な形式となり、$\phi(B)\tilde{z}_t = \theta(B)a_t$ がエレガントになる。

### 疑問 2：なぜ ARIMA の "I" は「積分」と呼ばれるのか？（Ch1）
**質問**：差分 $\nabla^d$ は非定常を定常にするものだが、「積分」と差分にはどんな関係があるのか？

**回答**：名前は**逆演算**に由来する。$\nabla = (1-B)$ は差分であり、その逆 $S = \nabla^{-1} = (1-B)^{-1} = 1 + B + B^2 + \cdots$ は総和（summation）——離散版の「積分」である。ARIMA 過程 $z_t$ は定常 ARMA 過程 $w_t$ に $d$ 回の総和/積分を施して得られる：$z_t = S^d w_t$。

### 疑問 3：なぜ節約の原則はそれほど重要なのか？（Ch1.3）
**質問**：現代の計算能力はこれほど強力なのに、なぜパラメータを増やせないのか？

**回答**：Box の洞察は計算能力の問題を超越している。核心的な理由は**データ量の有限性**である：
1. パラメータが増えるほど、必要なデータ量は超線形的に増大する（次元の呪い）
2. 有限データではパラメータ間の関係を同定しにくい（Box の $\omega_0, \omega_1, \ldots, \omega_s$ の例）
3. 過学習されたモデルは予測時の汎化能力が低い

2026年の深層学習時代においても、1つの発電所の3年分の15分データ（約10万ポイント）しかない場合、ARIMA(1,1,1) の5つのパラメータは、10万パラメータのモデルより信頼できるかもしれない。

### 推論チェーン 3：正定値性の二つの側面——時間領域の制約と周波数領域の保証
**Ch2.1.3 自己共分散行列の正定値性 ↔ Ch2.2.3 スペクトルの非負性**

1. 自己共分散行列 $\boldsymbol{\Gamma}_n$ は正定値でなければならない（任意の線形結合 $L_t$ の分散が > 0 だから）
2. 正定値性は $\rho_k$ の取り得る範囲を制約する（任意の値は取れない！）
3. 等価条件：スペクトル密度 $p(f) \geq 0$（すべての $f$ に対して）
4. これが Herglotz の定理の内容である

**重要な推論**：正定値性は「合法的な ACF」の必要十分条件である。任意に構成した $\{\rho_k\}$ 列は、実際の確率過程に必ずしも対応しない——正定値性を検証しなければならない。プログラミングでは、データから推定した ACF を打ち切った後に正定値化処理が必要ということを意味する。

### 推論チェーン 4：なぜ Bartlett の公式は真の ρ を知る必要があるのか？
**Ch2.1.6 標準誤差 → Ch6 モデル同定の困難さ**

1. $\text{var}[r_k]$ の公式には真の $\rho_v$（未知！）が含まれる
2. 実際の応用では推定値 $r_v$ で代替するしかない → 追加の不確実性が生じる
3. 白色雑音（$q=0$）には簡潔な公式 $1/\sqrt{N}$ があるが、非白色雑音にはブートストラップが必要
4. **これがモデル同定が純粋な科学ではなく「芸術」である理由**——ACF の不確実性が $p, q$ の正確な決定を非常に困難にする

### 推論チェーン 5：ペリオドグラム → 分散分析 → スペクトル推定の進化
**Ch2.2.1–2.2.3 離散から連続へ**

1. ペリオドグラム：分散を $N/2$ 個の離散周波数に分解（フーリエ級数）
2. 分散分析：各周波数が2つの自由度を占め、$\chi^2(2)$ 分布（白色雑音の場合）
3. 標本スペクトル：連続的な周波数を許容するが、**激しく変動する**（本質的に狭すぎる周波数窓を使用）
4. 平滑化スペクトル推定：窓関数 $\lambda_k$ または $W(f_j)$ を導入し、分解能を犠牲にして安定性を得る

**NWP との平行性**：これは Warner Ch4 のスペクトル分析と完全に対応する——NWP は FFT を使って数値解の周波数特性を分析し、計算ノイズを検出する。Box は同じツールを使って時系列の周波数特性を分析し、周期的成分を同定する。

## 三B、解決済みの疑問（Ch2）

### 疑問 4：なぜ ACF の推定で $1/(N-k)$ ではなく $1/N$ を使うのか？（Ch2.1.5）
**質問**：$c_k = \frac{1}{N}\sum_{t=1}^{N-k}(z_t-\bar{z})(z_{t+k}-\bar{z})$ は分母に $N$ を使うが、実際の加算は $N-k$ 項しかない。

**回答**：$1/N$ を使う利点は二つある：
1. 推定された ACF 行列が半正定値であることを保証する（$1/(N-k)$ はこれを保証しない）
2. 大標本では両者は漸近的に等価
3. Box は Jenkins & Watts [170] の体系的な比較を引用し、$1/N$ が優れていると結論づけた

### 疑問 5：なぜ標本スペクトルは「激しく変動する」のか？（Ch2.2.3）
**質問**：$I(f)$ はなぜスペクトルの良い推定量ではないのか？

**回答**：$I(f)$ の分散は **$N$ が増大しても減少しない**（これは直感に反する！）。理由は各 $I(f_i) \sim \sigma^2\chi^2(2)$ であり、$\chi^2(2)$ の変動係数は $N$ の大きさに関係なく100%だからだ。解決策は平滑化——隣接する周波数の $I(f)$ を平均することで、自由度を増やすのと等価だが、周波数分解能は失われる。
