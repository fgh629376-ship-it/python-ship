> 結論を暗記するのではなく、推論プロセスを鍛える。各章を読んだ後、「なぜそうなのか？」「他の知識とどう繋がるのか？」と問い続ける。

## 一、跨章推論チェーン

### 推論チェーン1：白色雑音からARIMAへの論理的連鎖
**Ch1 線形フィルターモデル → Ch3 ARMA → Ch4 ARIMA**

1. すべての観測可能な時系列は、白色雑音 $a_t$ が線形フィルターを通った出力として見ることができる：$z_t = \psi(B)a_t$
2. もし $\psi(B)$ が $\phi^{-1}(B)\theta(B)$（有理関数）として表現できれば、ARMAが得られる：$\phi(B)\tilde{z}_t = \theta(B)a_t$
3. 系列が非定常だが差分後に定常になる場合、$(1-B)^d$ を導入してARIMAを得る：$\phi(B)\nabla^d z_t = \theta(B)a_t$

**重要な推論**：なぜ有理関数近似で十分なのか？→ 簡約原則。Padé近似理論は、低次の有理関数（分子＋分母それぞれ≤2次）が非常に複雑な無限級数を高精度で近似できることを示している。

### 推論チェーン2：予測と制御の統一
**Ch1.1.1 予測 + Ch1.1.5 制御 → Ch5 + Ch15**

1. 予測問題：$E[(z_{t+l} - \hat{z}_t(l))^2]$ を最小化
2. 制御問題：$E[\varepsilon_t^2]$ を最小化（$\varepsilon_t = Y_t - T$）
3. 制御 = 「制御なしの偏差」を予測 + 補償調整量の計算
4. **最小MSE予測 → 最小MSE制御**

**重要な推論**：この統一は偶然ではない——両者は条件付き期待値の最適性に帰着する。$\hat{z}_t(l) = E[z_{t+l} | z_t, z_{t-1}, \ldots]$ は条件付き期待値であり、$L^2$ 空間での直交射影として自然にMSEを最小化する。

### 重要な跨教材の洞察

**Boxの線形フィルタリング ↔ Warnerの数値フィルタリング**：
- Boxの $\psi(B)$ 重み系列はデジタルフィルターのインパルス応答
- Warner Ch4のRobert-Asselin時間フィルターは特定のAR型フィルター
- NWPの暗示的時間差分格式（半陰的法など）は周波数領域で高周波成分に特定の $\psi$ 重みを適用することと等価

**BoxのARIMA差分 ↔ Yangのトレンド除去**：
- ARIMAは $\nabla^d = (1-B)^d$ でトレンドを除去する——「盲目的な」トレンド除去（トレンドの形を仮定しない）
- Yang Ch7は晴天時モデルによる物理的トレンド除去：$\text{anomaly}_t = \text{actual}_t - \text{clearsky}_t$
- **ベストプラクティス**：物理的トレンド除去（Yang）+ 統計的差分（Box）は補完的——物理モデルが予測可能なトレンドを除去し、統計的方法が残余の非定常性を処理する

## 三、解決した疑問

### 疑問1：なぜMAモデルで「マイナス符号」を使うのか？（Ch1）
**問い**：ARモデルは $\tilde{z}_t = \phi_1 \tilde{z}_{t-1} + a_t$（プラス符号）と書くのに、なぜMAモデルは $\tilde{z}_t = a_t - \theta_1 a_{t-1}$（マイナス符号）と書くのか？

**答え**：これはARとMAの双対関係をより簡潔にするための慣例（convention）である。MAがプラス符号を使えば、ARとMAの変換公式に余分なマイナス符号が入る。マイナス符号の慣例を使うと：
- AR(1)：$\phi(B) = 1 - \phi_1 B$
- MA(1)：$\theta(B) = 1 - \theta_1 B$

両者は完全に対称な形になり、$\phi(B)\tilde{z}_t = \theta(B)a_t$ が優雅になる。

### 疑問2：ARIMAの「I」はなぜ「積分」と呼ばれるのか？（Ch1）
**問い**：差分 $\nabla^d$ が非定常を定常にするなら、「積分」と差分はどんな関係があるのか？

**答え**：名前は**逆演算**から来ている。$\nabla = (1-B)$ は差分で、その逆 $S = \nabla^{-1} = (1-B)^{-1} = 1 + B + B^2 + \cdots$ は総和（summation）——離散版の「積分」。ARIMA過程 $z_t$ は定常ARMA過程 $w_t$ を $d$ 回総和/積分することで得られる：$z_t = S^d w_t$。

### 疑問3：なぜ簡約原則がそれほど重要なのか？（Ch1.3）
**問い**：現代の計算能力があれば、なぜもっと多くのパラメータを使えないのか？

**答え**：Boxの洞察は計算能力の問題を超えている。核心的な理由は**データ量の限界**：
1. パラメータが増えるほど、必要なデータ量が超線形的に増加する（次元の呪い）
2. 限られたデータではパラメータ間の関係を識別することが困難（Boxの $\omega_0, \omega_1, \ldots, \omega_s$ の例）
3. 過学習モデルは予測時の汎化能力が低い

2026年の深層学習時代でも、1つの太陽光発電所3年分の15分データ（~10万点）しかない場合、5つのパラメータを持つARIMA(1,1,1)は10万パラメータのモデルより信頼できるかもしれない。

### 推論チェーン3：正定値性の二面——時間領域の制約と周波数領域の保証
**Ch2.1.3 自己共分散行列の正定値性 ↔ Ch2.2.3 スペクトルの非負性**

1. 自己共分散行列 $\boldsymbol{\Gamma}_n$ は正定値でなければならない（任意の線形結合 $L_t$ の分散は > 0 でなければならないため）
2. 正定値性は $\rho_k$ の取りうる範囲を制約する（恣意的に設定できない！）
3. 等価条件：すべての $f$ に対してスペクトル密度 $p(f) \geq 0$
4. これがHerglotzの定理の内容

**重要な推論**：正定値性は「合法なACF」の必要十分条件。恣意的に $\{\rho_k\}$ 系列を構成しても、実際の確率過程に対応しないかもしれない——正定値性の確認が必要。プログラミングでは、データから推定されて切り詰められたACFには正定値化処理が必要。

### 推論チェーン4：なぜBartlett公式は真のρを知る必要があるのか？
**Ch2.1.6 標準誤差 → Ch6 モデル識別の困難さ**

1. $\text{var}[r_k]$ の公式には真の $\rho_v$（未知！）が含まれる
2. 実際の適用では推定値 $r_v$ で代替するしかない → 追加の不確実性が入る
3. 白色雑音（$q=0$）には簡潔な公式 $1/\sqrt{N}$ があるが、非白色雑音にはブートストラッピングが必要
4. **これがモデル識別が「芸術」であって純科学でない理由**——ACFの不確実性が $p, q$ の精確な判断を非常に困難にする

### 推論チェーン5：ピリオドグラム → 分散分析 → スペクトル推定の進化
**Ch2.2.1-2.2.3 離散から連続へ**

1. ピリオドグラム：分散を $N/2$ 個の離散周波数に分解（フーリエ級数）
2. 分散分析：各周波数は自由度2、$\chi^2(2)$ 分布（白色雑音の場合）
3. サンプルスペクトル：連続周波数を許可するが、**激しく変動する**（本質的に周波数窓が狭すぎる）
4. 平滑化スペクトル推定：窓関数 $\lambda_k$ または $W(f_j)$ を導入し、分解能を犠牲にして安定性を得る

**NWPとの並行性**：これはWarner Ch4のスペクトル分析と完全に対応する——NWPはFFTで数値解の周波数特性を分析し、計算ノイズを検出する；Boxは同じツールで時系列の周波数特性を分析し、周期成分を識別する。

## 三B、解決した疑問（Ch2）

### 疑問4：なぜACF推定に $1/(N-k)$ ではなく $1/N$ を使うのか？（Ch2.1.5）
**問い**：$c_k = \frac{1}{N}\sum_{t=1}^{N-k}(z_t-\bar{z})(z_{t+k}-\bar{z})$ は分母に $N$ を使っているが、実際の和の項数は $N-k$ しかない。

**答え**：$1/N$ を使うことには2つの利点がある：
1. 推定されたACF行列が半正定値であることを保証する（$1/(N-k)$ は保証しない）
2. 大サンプルでは両者は漸近的に等価
3. BoxはJenkins & Watts [170]の系統的比較を引用し、$1/N$ がより優れていると結論づけている

### 疑問5：なぜサンプルスペクトルは「激しく変動する」のか？（Ch2.2.3）
**問い**：なぜ $I(f)$ はスペクトルの良い推定量でないのか？

**答え**：$I(f)$ の分散は **$N$ が増えても減少しない**（これは直感に反する！）。理由は各 $I(f_i) \sim \sigma^2\chi^2(2)$ であり、$\chi^2(2)$ の変動係数 = 100%で、$N$ がどんなに大きくても変わらない。解決策は平滑化——隣接する $I(f)$ 値を平均することで自由度を増やすが、周波数分解能を失う。

### 推論チェーン6：Wold分解 → ARMAの正当性 → Padé近似
**Ch3.1 Woldの定理 → Ch3.1.4 有理関数パラメータ化**

1. Woldの定理：任意の定常過程 = 白色雑音の線形フィルター（$\tilde{z}_t = \psi(B)a_t$）
2. $\psi(B)$ には無限に多くの項がある可能性 → 実用的でない
3. 有理関数 $\psi(B) = \theta(B)/\phi(B)$ で近似 → ARMA(p,q)
4. これがPadé近似——低次の有理関数は同次の多項式より良い近似を与える
5. **簡約原則の数学的基盤**：$p+q \leq 4$ のARMAは実際にはほぼ常に十分

### 推論チェーン7：ACF/PACFの打ち切り＝モデルの「指紋」
**Ch3.2-3.4 → Ch6 モデル識別**

1. AR(p)：PACFがラグ $p$ で打ち切り（$\phi_{kk}$ は $k$ 次回帰の最後の係数で、$k > p$ のときゼロ）
2. MA(q)：ACFがラグ $q$ で打ち切り（ラグ $> q$ の自己共分散は重複しないショックを含み、共分散がゼロ）
3. ARMA(p,q)：両方とも拖尾（ただしパターンは判別可能：ACFはあるラグから $\phi$ の差分方程式によって制御される）
4. **実際の操作**：まずACF/PACFを描く → 打ち切りパターン → 初期の次数決定 → 推定 → 診断

**これがCh6モデル識別の全論理的基盤。**

## 三C、解決した疑問（Ch3）

### 疑問6：なぜ可逆性が重要なのか？（Ch3.1.3）
**問い**：MA(1)は $\theta_1$ がどんな値でも定常であるのに、なぜ $|\theta_1| < 1$ に制限するのか？

**答え**：可逆性は**現在の値が過去の値で合理的に表現できること**を保証する。$|\theta_1| \geq 1$ の場合、$\pi_j = -\theta_1^j$ は発散する——現在の $z_t$ は遠い過去の $z_{t-k}$ に対してますます大きな依存重みを持つことになり、物理的に不合理。

より深い理由：与えられたACFに対して、それを生成できるMA(1)モデルは2つある（$\theta_1$ と $\theta_1^{-1}$）；可逆性条件が唯一の有効なものを選択する。

### 疑問7：AR(2)の「擬似周期」とは何を意味するのか？（Ch3.2.4）
**問い**：AR(2)は周期性を生成できるのか？本当の周期性とどう違うのか？

**答え**：AR(2)の複素数根が生成するACFは**減衰正弦波** $\rho_k = D^k \sin(2\pi f_0 k + F)/\sin F$ であり、固定振幅の正弦波ではない。物理的には「準周期性」——各「周期」は前の周期より弱い。

本当の周期性（日周期、年周期など）は**確定的な成分**であり、フーリエ項や季節差分（Ch9）で処理する必要があり、ARの複素数根では処理できない。AR(2)の擬似周期は「天気系の約3-7日の準周期」のような統計的規則性を記述するのに適している。

### 推論チェーン8：予測更新公式 → リアルタイムシステム設計
**Ch5.2.3 → 太陽光発電リアルタイム予測システム**

1. 更新公式：$\hat{z}_{t+1}(l) = \hat{z}_t(l+1) + \psi_l a_{t+1}$
2. 新データ到着 → 1ステップ誤差 $a_{t+1}$ を計算 → すべてのリードタイム予測が同時更新
3. $\psi_l$ の減衰が速い → 長期予測はほぼ変わらない（AR型過程）
4. $\psi_l$ の減衰が遅い → 近期のショックが長期予測にも影響（IMA型過程）
5. **設計の選択**：明確な日周期を持つ日射量系列 → 周期除去後の残差はARMAに近い → $\psi_l$ が速く減衰 → 更新公式が素早く収束

### 推論チェーン9：指数平滑 ⊂ ARIMAの統計的意味
**Ch5.4 → Ch4.3 → 予測方法論の統一**

1. Holt/Brown/Winters（1950-60年代）は純粋に経験的な方法で、統計モデルの基盤がなかった
2. Box（1970）が証明：単純指数平滑 = IMA(0,1,1)の最適予測
3. Holtの線形法 = IMA(0,2,2)の最適予測
4. **含意**：すべての指数平滑法には暗黙の時系列モデル仮定がある
5. データがIMA構造に合致しない場合、指数平滑はもはや最適ではない
6. **実践ガイド**：まずデータのARIMA構造を判断し、それから方法を選ぶ——盲目的に指数平滑を使わない

## 三D、解決した疑問（Ch4-5）

### 疑問8：なぜ回帰によるトレンド除去ではなく差分なのか？（Ch4.1.2）
**問い**：線形回帰 $z_t = a + bt + \epsilon_t$ でトレンドを除去できるのに、なぜBoxは差分を主張するのか？

**答え**：回帰によるトレンド除去はトレンドが**確定的**（$a + bt$）であることを仮定し、差分はトレンドが**確率的**であることを仮定する。Boxは多くの実際の系列のトレンドは時間とともに変化する（株価のように）と考えており、確定的トレンドを使った予測はモデルを過去のトレンドに「縛りつける」が、確率的トレンドモデルは自動的に適応する。

より重要なのは：本当のトレンドが確率的なのに確定的な回帰でトレンド除去をすると、残差はまだ非定常——トレンドを除去したつもりが実際には除去されていない。

### 疑問9：なぜ予測区間は時間とともに広がるのか？（Ch5.1.1）
**問い**：$V(l)$ が $l$ とともに単調増加するのは必然なのか？

**答え**：因果的で可逆なARIMAでは $\psi_j \neq 0$ なので、$V(l)$ は厳密に増加する——**不確実性は予測リードタイムとともに不可避的に増加する**。

定常過程の場合：$V(l) \to \gamma_0$（無条件分散に収束）、予測は平均に回帰する。
非定常過程（$d \geq 1$）の場合：$V(l) \to \infty$、予測区間は無限に広がる——**非定常過程の長期予測は本質的に信頼できない**。

### 推論チェーン10：Box-Jenkins反復ループの論理構造
**Ch6 → Ch7 → Ch8 → Ch6に戻る**

1. Ch6識別：ACF/PACFパターン → 候補モデル + 初期パラメータ
2. Ch7推定：MLE/条件付き最小二乗 → 精密なパラメータ + 標準誤差
3. Ch8診断：Ljung-Box Q + 残差ACF + 過剰適合 → モデルは十分良いか？
4. 十分でなければ → 残差ACFが修正の方向を示す → Ch6に戻る
5. **これは科学的方法論の時系列実装**：仮説→検定→修正→再検定

### 推論チェーン11：過パラメータ化と簡約原則の究極の理由
**Ch7.3.5 → Ch8.1 → Ch3.4**

1. ARMA(p,q)で $\phi_i \approx \theta_j$ のとき、ARとMAがほぼ相殺される
2. 尤度面に長い尾根が生じる → パラメータが識別不能 → 標準誤差が爆発
3. 過剰適合検定の過用も偽のパラメータを導入しうる
4. **簡約原則は審美的な好みではなく、統計的に必要**
5. 太陽光発電予測にも同様に適用される：ARMA(2,2)は必ずしもAR(2)より良いわけではない

## 三E、解決した疑問（Ch6-8）

### 疑問10：モデル識別は本当に「グラフを見る」だけに依存するのか？（Ch6.1）
**問い**：それはあまりにも主観的ではないか？

**答え**：Boxは識別が「inexact」で「judgment」を必要とすることを明確に認めている——しかしこれは欠点ではない。理由：
- 推定されたACF/PACFにはノイズがあり、打ち切り点を数学的に正確に判断することは不可能
- 異なるモデルが類似したACFを生成する可能性 → 複数の候補モデルがすべて合理的
- **識別は入口に過ぎない**——真の品質保証はCh7の推定とCh8の診断にある
- 現代の自動化（AIC/BIC + `auto.arima()`）が人間の判断を置き換えられるが、背後のロジックを理解することは依然として必要

### 疑問11：「すべてのモデルは間違っている」はモデリングが無意味であることを意味するのか？（Ch8.1.1）
**問い**：正しいモデルがないなら、なぜこれほど苦労するのか？

**答え**：Boxの完全な格言は「All models are wrong, but some are useful.」——鍵は「useful」（有用）——
- モデルは完璧である必要はなく、**予測タスク**に対して十分良ければよい
- 診断検定はモデルが「正しい」ことを証明しようとするのではなく、**深刻な欠陥**がないことを確認する
- Ljung-Boxが通過し、残差ACFがクリーンなら、モデルは「使用可能」
- **実用主義 > 完全主義**——太陽光発電予測にも同様

### 推論チェーン12：乗積季節モデル＝2スケール分離仮定の数学的実装
**Ch9.1.3 → Ch3.4 ARMA → Warner Ch8 物理パラメータ化**

1. 乗積SARIMA：$\phi(B)\Phi(B^s)\nabla^d\nabla_s^D z_t = \theta(B)\Theta(B^s)a_t$
2. 「乗積」は：月内ダイナミクス $\phi(B), \theta(B)$ と年際ダイナミクス $\Phi(B^s), \Theta(B^s)$ が**分離可能**であることを意味する
3. これは年際関係の「イノベーション」$\alpha_t$ 自体がARMA過程（月内相関を持つ）であることを仮定するのと等価
4. 数学的に：$(1-\theta B)(1-\Theta B^s) = 1 - \theta B - \Theta B^s + \theta\Theta B^{s+1}$——交差項 $\theta\Theta$ が本来必要だった独立した第三パラメータを**2つのパラメータの積**で置き換える
5. **NWPとの並行性**：Warner Ch8の物理パラメータ化も「スケール分離」を行う——乱流（小スケール）と対流（中スケール）を別々にパラメータ化し、結合項で相互作用させる。乗積構造 = パラメータ化結合の最も簡約な形

**深い洞察**：スケール分離は科学的モデリングの普遍的原則——時系列分析のテクニックだけではない。NWPは大気運動をresolved（明示的に計算）とsubgrid（パラメータ化）に分け、Boxは時系列を非季節（$B$）と季節（$B^s$）に分ける。哲学は完全に一致している。

### 推論チェーン13：GARCH＝2次モーメントのARMA → ボラティリティ自体が予測可能
**Ch10.1 → Ch3 ARMA → Yang Ch11 確率的予測**

1. ARMAは**平均**をモデル化：$z_t = \phi z_{t-1} + a_t$
2. GARCHは**分散**をモデル化：$h_t = \omega_0 + \omega_1 a_{t-1}^2 + \beta_1 h_{t-1}$
3. $\eta_t = a_t^2 - h_t$ と定義すると、$a_t^2 = (\omega_0 + (\omega_1 + \beta_1)a_{t-1}^2 - \beta_1 \eta_{t-1}) + \eta_t$
4. **これは $a_t^2$ のARMAモデルだ！** GARCH(1,1) = 二乗ショックのARMA(1,1)
5. 含意：ボラティリティ（分散）自体に記憶構造があり、平均を予測するように予測できる

**太陽光発電予測への重要な連結**：
- Yang Ch11の確率的予測には**時変予測区間**が必要
- Box Ch5の固定 $\sigma_a^2$ は等幅区間を与える——**晴天時には広すぎ、曇天時には狭すぎる**
- GARCHは $h_t$ を適応的に調整：昨日変動大 → 今日の区間は広い；昨日安定 → 今日の区間は狭い
- **これが現代の確率的予測でGARCHや類似手法を使う理由**

### 推論チェーン14：伝達関数＝pvlib Model Chainの統計版
**Ch11 → Yang Ch3-6 → pvlib**

1. Box伝達関数：$Y_t = v(B)X_t + N_t$（入力→システム→出力+ノイズ）
2. pvlib Model Chain：GHI → POA → DC → AC（日射量→発電量）
3. 両者のマッピング：
   - 入力 $X_t$ = GHI（全天日射量）
   - 伝達関数 $v(B)$ = 分解モデル + 傾斜面変換モデル + 温度モデル + IV曲線
   - 出力 $Y_t$ = AC電力
   - ノイズ $N_t$ = モデリング誤差（ほこり、遮蔽、劣化等、未モデル化因子）
4. 重要な違い：pvlibは**確定的**な伝達関数（物理公式）；Boxのは**確率的**な伝達関数（統計推定）
5. **ベストプラクティス**：pvlib物理モデルを第一層として使う → 残差 $N_t$ をARIMAでモデル化 → 予測を結合

**これは物理-統計ハイブリッド予測の理論的基盤であり、YangのCh7「後処理」の正確な数学的表現でもある。**

### 推論チェーン15：介入分析＝太陽光発電運維の数学的言語
**Ch13 → Yang Ch8 衛星 → 実際の運維**

2. ステップ介入（$\xi_t = S_t^{(T)}$）に対応：パネルの永続的遮蔽、インバータの定格制限、新規ストリング追加
3. パルス介入（$\xi_t = P_t^{(T)}$）に対応：一時的な積雪、鳥の糞、短期的な遮蔽
4. 漸変的介入（$\delta(B)$ が根を持つ）に対応：パネル劣化、ほこり蓄積

6. **IO外れ値** = 系統的変化の起点 → 再モデル化が必要

**実際の運維シナリオ**：監視システムが発電量の急落を検出 → 介入分析が自動的に「一時的遮蔽」か「永続的故障」かを判断 → 異なる保守対応を起動

### 推論チェーン16：VAR＝多地点空間-時間予測のフレームワーク

## 三F、解決した疑問（Ch9-15）

### 疑問12：季節差分と通常差分の順序は重要か？（Ch9.1.3）
**問い**：$\nabla\nabla_{12} z_t$ と $\nabla_{12}\nabla z_t$ は同じか？

**答え**：同じ！差分演算子は可換：$(1-B)(1-B^{12}) = (1-B^{12})(1-B)$。しかし**実践では**、まず季節差分 $\nabla_{12}$ を行い、次に通常差分 $\nabla$ を行う方がより直感的——まず年際変化を除去し、次に月際変化を除去する。ACFの解釈もより明確になる。

### 疑問13：GARCHの $a_t$ は白色雑音なのか？（Ch10.1）
**問い**：GARCHの $a_t$ はゼロ平均、無相関だが独立でない——これは白色雑音なのか？

**答え**：**弱白色雑音**（weak white noise）だが**強白色雑音**（strict white noise）ではない。
- 弱白色雑音：ゼロ平均 + 無相関 + 等分散 ✓
- 強白色雑音：独立同分布 ✗

GARCHの $a_t$ は**条件付き分散 $h_t$ を通じて相互に関連している**——$a_t^2$ は自己相関している！だからBox Ch8のLjung-Box検定は $\hat{a}_t$ に対しては有意でないが、$\hat{a}_t^2$ に対しては有意になる → ARCH効果の存在を示唆。

**検出方法**：二乗残差 $\hat{a}_t^2$ にLjung-Box検定を実施；有意なら → GARCHを使用。

### 疑問14：伝達関数の「事前白色化」はなぜ必要なのか？（Ch12.2）
**問い**：$X_t$ と $Y_t$ の交差相関を直接計算することはできないのか？

**答え**：できない。$X_t$ 自体に自己相関がある場合（ほぼ常にそうだが）、直接の交差相関 $r_{XY}(k)$ は $X_t$ の**自己相関構造によって「汚染」される**——見えているのは純粋な因果応答ではなく、因果応答と入力自己相関の混合。

事前白色化の巧妙さ：まず $X_t$ を白色雑音 $\alpha_t$ に変換し（ARIMAを適合させることで）、次に $\alpha_t$ と $\beta_t$（$Y_t$ に同じ変換を適用）の交差相関を見る → これが純粋なインパルス応答推定。

**NWPとの対応**：Warner Ch12のデータ同化における「観測演算子の増分」も一種の事前白色化——複雑な観測空間をモデル空間にマッピングしてから相関分析を行う。

### 疑問15：Granger因果性＝真の因果性なのか？（Ch14.3）
**問い**：XがYをGranger因果するなら、XがYを真に「引き起こす」ことを意味するのか？

**答え**：**違う。** Granger因果性は**予測的因果性**に過ぎない——「Xの過去がYの予測に役立つ」。考えられる非因果的説明：
- 共通の隠れた因子 $Z$ が $X$ と $Y$ の両方を駆動している（交絡）
- $X$ が同じ信号を $Y$ より早く反映している（先行-遅行関係）

BoxもGrangerもこの限界を明確に認識していた。真の因果推論には実験計画または因果グラフ（Pearlのdo計算）が必要。

**太陽光発電の例**：発電所AがBをGranger因果する → 雲系がAからBへ移動しているだけかもしれない（物理的には雲の因果性であり、発電量の因果性ではない）。しかし予測目的にはGranger因果性で十分——予測精度だけを気にする。
