# MIT ML 理論大統一：すべての監督学習を一本の論理線で串刺しにする

> **講義出典**：MIT 6.867 Machine Learning (Tommi Jaakkola) + Boyd *Convex Optimization* + MIT 18.065 Matrix Methods (Strang)
> **核心的主張**：すべての監督学習手法は同一フレームワークの特殊ケースである——**経験リスク最小化 + 正則化**

## 〇. 全体像：ML の統一フレームワーク

$$\hat{f} = \arg\min_{f \in \mathcal{F}} \underbrace{\frac{1}{n}\sum_{i=1}^n L(y_i, f(x_i))}_{\text{経験リスク}} + \underbrace{\lambda \Omega(f)}_{\text{正則化}}$$

**3 つの選択がすべてを決める**：
1. **損失関数** $L$：予測値と真値のギャップを測る
2. **仮説空間** $\mathcal{F}$：モデルが何を表現できるか
3. **正則化** $\Omega(f)$：モデルの複雑さを制御する

異なる組み合わせが異なるアルゴリズムを生み出すが、**数学的本質はまったく同じ**である。

## 一. 最小二乗法から正則化回帰へ

### 1.1 線形回帰 = 最もシンプルな ERM

$$\hat{\boldsymbol{\beta}} = \arg\min_{\boldsymbol{\beta}} \frac{1}{n}\|\mathbf{y} - X\boldsymbol{\beta}\|^2$$

解：$\hat{\boldsymbol{\beta}} = (X^TX)^{-1}X^T\mathbf{y}$（18.06 Part 2 の射影公式）

**問題点**：$p > n$（特徴量がサンプル数より多い）または特徴量の共線性が高い場合、$X^TX$ は ill-conditioned になり → $\hat{\boldsymbol{\beta}}$ の分散が爆発する。

### 1.2 Ridge 回帰 = $L_2$ 正則化

$$\hat{\boldsymbol{\beta}}_{ridge} = \arg\min_{\boldsymbol{\beta}} \frac{1}{n}\|\mathbf{y} - X\boldsymbol{\beta}\|^2 + \lambda\|\boldsymbol{\beta}\|^2$$

解：$\hat{\boldsymbol{\beta}}_{ridge} = (X^TX + \lambda I)^{-1}X^T\mathbf{y}$

**SVD の視点**（18.06 Part 3）：$X = U\Sigma V^T$ とおくと、

$$\hat{\boldsymbol{\beta}}_{ridge} = \sum_{j=1}^r \frac{\sigma_j^2}{\sigma_j^2 + \lambda} \frac{\mathbf{u}_j^T\mathbf{y}}{\sigma_j}\mathbf{v}_j$$

**$\lambda$ の役割**：小さな特異値方向の係数を縮小する。$\lambda \to 0$ で OLS に退化し、$\lambda \to \infty$ ですべての係数が 0 に近づく。

**ベイズ的解釈**（確率・統計）：Ridge = ガウス事前分布 $\boldsymbol{\beta} \sim N(0, \frac{1}{\lambda}I)$ の下での MAP 推定。

### 1.3 Lasso = $L_1$ 正則化

$$\hat{\boldsymbol{\beta}}_{lasso} = \arg\min_{\boldsymbol{\beta}} \frac{1}{n}\|\mathbf{y} - X\boldsymbol{\beta}\|^2 + \lambda\|\boldsymbol{\beta}\|_1$$

**$L_1$ の魔法**：**スパース解**を生む——多くの $\beta_j$ がちょうどゼロになる → **自動特徴選択**。

**幾何学的解釈**：$L_1$ の制約集合は「ひし形」で、等高線はコーナーで接しやすい → コーナーでの座標はゼロ。

**ベイズ的解釈**：Lasso = ラプラス事前分布の下での MAP。

### 1.4 Elastic Net = $L_1 + L_2$

$$\lambda_1\|\boldsymbol{\beta}\|_1 + \lambda_2\|\boldsymbol{\beta}\|^2$$

スパース性（$L_1$）と安定性（$L_2$）を両立する。

### 🔗 太陽光発電予測との接続

- **気象特徴量は高度に相関している**（気温 ↔ 日射量 ↔ 気圧）→ Ridge で安定した推定
- **特徴量は多いが本当に重要なのは少ない**（数十の NWP 変数のうち実際に有用なのは 5〜6 個程度）→ Lasso による自動選択
- **Box Ch7 の尾根状尤度面**（$\phi \approx \theta$ のとき）→ 正則化でパラメータ非識別性を解消

## 二. バイアス–バリアンストレードオフ——ML の核心的矛盾

### 2.1 分解定理

任意の推定量 $\hat{f}$ に対して：

$$E\left[(y - \hat{f}(x))^2\right] = \underbrace{\text{Bias}^2[\hat{f}(x)]}_{\text{過少適合}} + \underbrace{\text{Var}[\hat{f}(x)]}_{\text{過剰適合}} + \underbrace{\sigma^2}_{\text{不可約ノイズ}}$$

**単純なモデル**：高バイアス・低バリアンス（過少適合）
**複雑なモデル**：低バイアス・高バリアンス（過剰適合）

### 2.2 モデル選択

**交差検証**：データを訓練セット／検証セットに分割し、検証誤差が最小のモデルを選ぶ

**情報量規準**：
- AIC = $-2\ell + 2k$（Box Ch6 で使用）
- BIC = $-2\ell + k\ln n$（より強いペナルティ）

**⚠️ 時系列の交差検証にはローリングウィンドウを使うこと**——時間的依存性があるためランダム分割は無効！

### 2.3 VC 次元と汎化誤差界

**VC 次元**：仮説空間 $\mathcal{F}$ が「粉砕」できる最大サンプル数。

$$R(f) \leq \hat{R}_n(f) + O\left(\sqrt{\frac{d_{VC}}{n}}\right)$$

真のリスク $\leq$ 経験リスク + 複雑度ペナルティ。

**含意**：モデルが複雑になるほど（$d_{VC}$ が大きいほど）、汎化に必要なデータが増える。

**直感的対応**：
- 線形モデル：$d_{VC} = p + 1$
- Box ARIMA(p,d,q)：有効パラメータ数 = $p + q$（Box は $p, q \leq 2$ を推奨——簡潔さ！）
- 深層学習：$d_{VC}$ は非常に大きいが、実際には**暗黙的正則化**がある（SGD 自体がよりシンプルな解を好む）

## 三. カーネル法——高次元空間での線形回帰

### 3.1 特徴写像の限界

非線形問題 → 手動で特徴 $\phi(x) = [x, x^2, x^3, \ldots]$ を構築 → 次元の爆発。

### 3.2 カーネルトリック

**核心的洞察**：多くのアルゴリズムはサンプル間の**内積** $\langle \phi(x_i), \phi(x_j) \rangle$ のみを必要とし、$\phi(x)$ を陽に計算する必要はない。

**カーネル関数**：$K(x_i, x_j) = \langle \phi(x_i), \phi(x_j) \rangle$

**よく使われるカーネル**：
- **線形カーネル**：$K(x, z) = x^Tz$（通常の線形回帰）
- **多項式カーネル**：$K(x, z) = (x^Tz + c)^d$（$d$ 次多項式特徴と等価）
- **ガウス / RBF カーネル**：$K(x, z) = \exp(-\|x - z\|^2 / 2\sigma^2)$（**無限次元**特徴空間と等価！）

### 3.3 カーネル Ridge 回帰

$$\hat{f}(x) = \sum_{i=1}^n \alpha_i K(x_i, x)$$

ここで $\boldsymbol{\alpha} = (K + \lambda I)^{-1}\mathbf{y}$、$K_{ij} = K(x_i, x_j)$ はカーネル行列。

**注意**：$K$ は $p \times p$ ではなく $n \times n$ → $n < p$（特徴数よりサンプル数が少ない）場合により効率的。

### 3.4 表現定理（Representer Theorem）

> **カーネル法では、最適解は必ず訓練サンプルの線形結合として書ける。**

$$f^*(x) = \sum_{i=1}^n \alpha_i K(x_i, x)$$

これは極めて深い結果である——**無限次元の最適化問題が有限次元の解を持つ**。

### 🔗 太陽光発電予測との接続

- 日射量→発電量の関係は非線形（温度効果、インバータクリッピング等）
- RBF カーネルの $\sigma$ = 非線形性の「スケール」
- **ガウス過程回帰（GP）**= カーネル法のベイズ版 → 予測の不確実性を自動で提供 → Yang Ch11

## 四. SVM——最大マージン分類器

### 4.1 ハードマージン SVM

$$\max_{\mathbf{w}, b} \frac{2}{\|\mathbf{w}\|} \quad \text{s.t.} \quad y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1$$

等価な凸最適化問題：$\min \frac{1}{2}\|\mathbf{w}\|^2$ s.t. $y_i(\mathbf{w}^T\mathbf{x}_i + b) \geq 1$

### 4.2 ソフトマージン SVM + ヒンジ損失

$$\min_{\mathbf{w}, b} \frac{1}{2}\|\mathbf{w}\|^2 + C\sum_{i=1}^n \max(0, 1 - y_i(\mathbf{w}^T\mathbf{x}_i + b))$$

これはまさに ERM + 正則化！$L$ = ヒンジ損失、$\Omega = \|\mathbf{w}\|^2$。

### 4.3 双対問題（Boyd 凸最適化）

ラグランジュ双対を通じて：

$$\max_{\boldsymbol{\alpha}} \sum_i \alpha_i - \frac{1}{2}\sum_{i,j}\alpha_i\alpha_j y_i y_j K(x_i, x_j)$$

**カーネル関数のみを含む** → 任意の高次元空間で SVM を実行できる！

### 4.4 KKT 条件の物理的意味

- $\alpha_i = 0$：サンプルが決定境界から遠い（モデルに影響しない）
- $0 < \alpha_i < C$：**サポートベクター**——ちょうどマージン境界上にある
- $\alpha_i = C$：マージン制約に違反するサンプル

**スパース性**：ほとんどの $\alpha_i = 0$ → SVM の予測は少数のサポートベクターにのみ依存 → **効率的**

### 🔗 凸最適化は ML のあらゆる場所に存在する

Boyd 凸最適化のコア概念と ML における役割：
- **凸関数**：MSE・交差エントロピー・ヒンジはすべて凸 → 局所最適 = 大域最適
- **双対性**：SVM の双対問題；カーネル法
- **KKT 条件**：サポートベクターの数学的定義；Lasso の劣微分条件
- **近接作用素**：$L_1$ 正則化の効率的な求解（ISTA/FISTA）

## 五. アンサンブル学習——弱学習器の力

### 5.1 バギング（Bootstrap Aggregating）

$$\hat{f}_{bag}(x) = \frac{1}{B}\sum_{b=1}^B \hat{f}_b(x)$$

各 $\hat{f}_b$ はブートストラップサンプル上で訓練される。**分散を低減**し、バイアスは変化しない。

**ランダムフォレスト** = バギング + ランダム特徴選択 → さらなる非相関化

### 5.2 ブースティング——最適化の視点

$$\hat{f}_m(x) = \hat{f}_{m-1}(x) + \gamma_m h_m(x)$$

各ステップで**前のステップの残差**を適合させる。

**勾配ブースティング** = 関数空間での勾配降下：
- $h_m = \arg\min_h \sum_i L'(y_i, \hat{f}_{m-1}(x_i)) \cdot h(x_i)$
- 新しい弱学習器は損失関数の**負の勾配**を適合させる

**XGBoost/LightGBM** = 勾配ブースティング + 2 次近似（Newton 法）+ 正則化 + 高効率な実装

### 5.3 バイアス–バリアンスのアンサンブル的視点

- **バギング**：$n$ 個の相関した予測器の平均 → 分散が $\frac{1+(n-1)\rho}{n} \cdot \sigma^2$ に低下
- **ブースティング**：段階的にバイアスを低下させる（各ステップで残差を修正）→ ただし分散が増加する可能性あり

### 🔗 時系列予測との接続

- **Box の反復的方法論**（同定 → 推定 → 診断 → 修正）は精神的にブースティングと同じ——各ステップが前のステップの不足を修正する
- **NWP アンサンブル予報**（Warner Ch14）= バギングの物理版——複数の初期条件 → 複数の予測 → 平均
- **Yang Ch12 階層的予測** = 複数モデルの融合 → スタッキング（別の形のアンサンブル）

## 六. 深層学習——非凸の新世界

### 6.1 万能近似定理

単一隠れ層ネットワークは任意の連続関数を任意の精度で近似できる。しかしこれは以下を**意味しない**：
- 学習が容易（最適化問題は非凸）
- サンプル効率が良い（指数的な幅が必要な場合がある）
- 汎化がよい（VC 次元は非常に大きい）

### 6.2 深さの力

深い > 広い：ある関数は $O(n)$ の深さのネットワークで表現できるが、単一隠れ層では $O(2^n)$ の幅が必要。

**直感**：深いネットワークは**階層的特徴抽出**を行う——下位層はエッジを、中位層はテクスチャを、上位層は意味論を学習する。

### 6.3 最適化：なぜ SGD は非凸問題でも機能するのか？

古典的最適化理論では、非凸問題には指数的に多くの局所最適点と鞍点があるとされる。しかし実際には SGD はうまく機能する。その理由：

1. **過パラメータ化**：パラメータ数 >> データ数 → ほぼすべての局所最適が大域最適に近い
2. **SGD の暗黙的正則化**：ノイズが鞍点とシャープな極小値から脱出するのを助ける
3. **損失面の幾何学**：高次元空間では鞍点が局所最適よりはるかに多い

### 6.4 誤差逆伝播 = 行列微積分の逆モード

MIT 18.063 のコア内容：

$$\frac{\partial L}{\partial W^{(l)}} = \frac{\partial L}{\partial \mathbf{a}^{(L)}} \cdot \prod_{k=L}^{l+1} \frac{\partial \mathbf{a}^{(k)}}{\partial \mathbf{a}^{(k-1)}} \cdot \frac{\partial \mathbf{a}^{(l)}}{\partial W^{(l)}}$$

誤差逆伝播は出力から入力に向かって流れ、**連鎖律の逆モード**を利用してすべてのパラメータの勾配を効率的に計算する。

### 6.5 正則化技術

- **ドロップアウト**：ニューロンをランダムに除去 → バギングの近似
- **重み減衰**：$L_2$ 正則化 = Ridge
- **早期停止**：検証誤差が上昇し始めたら停止 → 暗黙的な $L_2$ 正則化
- **バッチ正規化**：訓練を安定化 + 軽微な正則化

### 🔗 太陽光発電予測における深層学習の応用

- **CNN**：空の画像 → 短期日射量予測（雲追跡）
- **LSTM/GRU**：時系列発電量予測（長期依存性の捕捉）
- **Transformer**：多ステップ発電量予測（アテンション機構 → 履歴情報の適応的重み付け）
- **PINN**：物理的制約 + ニューラルネットワーク → pvlib 物理モデルを NN の損失関数に組み込む

## 九. 太陽光発電予測プロジェクトへの行動指針

### 技術選択ロードマップ

1. **ベースラインモデル**：物理モデル（pvlib モデルチェーン）→ 残差分析
2. **統計的修正**：ARIMA/SARIMA で残差の時系列構造を修正（Box）
3. **ML 強化**：
   - 勾配ブースティング（XGBoost/LightGBM）で特徴量→発電量の非線形マッピング
   - 入力特徴量 = NWP 変数 + pvlib 物理出力 + 時間的特徴
   - Lasso で特徴選択 → 最も重要な NWP 変数を特定
4. **深層学習**：
   - Transformer で多ステップ予測（翌日予測）
   - CNN で空の画像処理（ナウキャスティング）
   - PINN で pvlib の物理的制約を組み込む
5. **アンサンブル**：
   - スタッキング：物理モデル + ARIMA + XGBoost + Transformer の加重結合
   - 重みの適応化（天気タイプに応じて変化）
6. **確率化**：
   - 分位点回帰または GP で予測区間を提供
   - GARCH で分散を適応化（Box Ch10 → ボラティリティクラスタリング → 区間幅が天気に応じて変化）

### 各手法の数学的基盤

- **線形回帰**：18.06 射影 + 確率・統計 MLE
- **Ridge/Lasso**：凸最適化 + SVD 正則化
- **SVM/GP**：カーネル法 + 双対理論
- **XGBoost**：勾配ブースティング + Newton 2 次近似
- **Transformer**：行列微積分 + アテンション = $\text{softmax}(QK^T/\sqrt{d})V$
- **PINN**：変分法 + 自動微分

**フェーズ 1 の数学的基礎（線形代数 / 行列微積分 / 確率・統計）は、上記すべての手法を理解するための前提条件である。フェーズ 2 はそれらを一つのフレームワークに統合する。**
