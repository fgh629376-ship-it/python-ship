# MIT 18.06 线性代数 Part 4：PCA、傅里叶变换与循环矩阵

> **课程**：MIT 18.06 Linear Algebra, Spring 2025
> **范围**：Lecture 28-36（PCA、复数空间、DFT、循环矩阵）

## Lec 28：PCA——数据科学的第一工具

### 什么是 PCA？

**主成分分析（PCA）= 对数据矩阵做 SVD，取前 $k$ 个右奇异向量作为新坐标轴。**

设 $X$ 是 $n \times p$ 数据矩阵（$n$ 个样本，$p$ 个特征，已中心化+标准化）：

$$X = U\Sigma V^T$$

- **主成分方向**：$V$ 的列（$\mathbb{R}^p$ 中的方向）
- **主成分得分**：$U\Sigma$ 的列（每个样本在主成分上的投影值）
- **方差解释率**：$\sigma_i^2 / \sum \sigma_j^2$

### 2D PCA

选择数据变异性最大的 2D 投影：取 $V$ 的前两列 $\mathbf{u}, \mathbf{v}$，画散点图 $(\mathbf{x}_i \cdot \mathbf{u}, \mathbf{x}_i \cdot \mathbf{v})$。

### PCA ≠ 最小二乘

- **最小二乘**：最小化 $y$ 方向的残差平方和（垂直于 $x$ 轴）
- **PCA**：最小化到子空间的**正交距离**平方和（"perpendicular least squares"）

**光伏中的 PCA**：
- **气象特征降维**：温度/湿度/气压/风速等高度相关 → PCA 降到 3-5 维保留 95% 方差
- **多站点功率模式**：$n$ 个电站的功率矩阵 PCA → 提取"晴天模式"、"过云模式"、"阴天模式"
- **NWP 集合分析**：Warner Ch14 的 EOF 分析 = 对 NWP 集合预报矩阵做 PCA

## Lec 29-30：复数向量空间

### 从实数到复数

- **共轭转置** $\bar{A}^T$（Hermitian 转置）代替 $A^T$
- **内积** $\langle \mathbf{v}, \mathbf{w} \rangle = \bar{\mathbf{v}}^T \mathbf{w}$ → 保证 $\|\mathbf{v}\|^2 = \langle \mathbf{v}, \mathbf{v} \rangle \geq 0$

### 三类特殊复矩阵

- **酉矩阵（Unitary）**：$U\bar{U}^T = I$ → 保长变换（正交矩阵的推广）
- **Hermitian 矩阵**：$A = \bar{A}^T$ → 特征值全实（对称矩阵的推广）
- **正规矩阵**：$A\bar{A}^T = \bar{A}^T A$ → 有正交特征基

### 谱定理推广

| 矩阵类型 | 谱定理 |
|----------|--------|
| 实对称 | $A = EDE^T$，$E$ 正交，$D$ 实对角 |
| Hermitian | $A = UDU^*$，$U$ 酉，$D$ 实对角 |
| 正规 | $A = UDU^*$，$U$ 酉，$D$ 复对角 |

## Lec 31-33：循环矩阵与傅里叶基

### 循环矩阵

循环排列矩阵 $P$（$1 \to 2 \to 3 \to \cdots \to n \to 1$）的特征值 = $n$ 次单位根：

$$\omega_k = e^{2\pi i k/n}, \quad k = 0, 1, \ldots, n-1$$

**循环矩阵** = $P$ 的幂的线性组合 = $c_0 I + c_1 P + c_2 P^2 + \cdots$

### 傅里叶基

$P$ 的特征向量组成**傅里叶矩阵** $F$——这就是**离散傅里叶变换（DFT）的矩阵**！

$$F_{jk} = \frac{1}{\sqrt{n}} \omega_n^{jk}$$

**核心性质**：傅里叶基是**所有循环矩阵的共同特征基**。

### 循环卷积

两个向量的循环卷积 = 对应循环矩阵的乘积。

**关键定理**：DFT 将卷积变为逐点乘法。

$$\text{DFT}(\mathbf{a} * \mathbf{b}) = \text{DFT}(\mathbf{a}) \odot \text{DFT}(\mathbf{b})$$

## Lec 34-36：DFT 应用

### DFT = 基变换

从标准基到傅里叶基的坐标变换：

$$\hat{\mathbf{f}} = F^* \mathbf{f}$$

逆变换：$\mathbf{f} = F\hat{\mathbf{f}}$

### 应用

- **信号压缩**：DFT 后只保留大系数 → JPEG/MP3 的数学基础
- **去噪**：DFT → 滤除高频分量 → 逆 DFT
- **卷积加速**：直接卷积 $O(n^2)$，DFT 后逐点乘法 $O(n \log n)$（FFT）

### 与 Box 时间序列的深度联系

Box Ch2 的功率谱 = 自协方差函数的 DFT：

$$p(f) = 2\left(\gamma_0 + 2\sum_{k=1}^{\infty} \gamma_k \cos 2\pi f k\right)$$

这**就是**傅里叶变换！ACF $\gamma_k$ 和谱 $p(f)$ 是傅里叶对偶关系。

**线性滤波器的频域视角**：

ARMA 模型 $\phi(B)\tilde{z}_t = \theta(B)a_t$ 的谱密度：

$$g(f) = \sigma_a^2 \frac{|\theta(e^{-2\pi if})|^2}{|\phi(e^{-2\pi if})|^2}$$

MA → 分子（零点）→ 在特定频率抑制能量
AR → 分母（极点）→ 在特定频率放大能量

**AR(2) 的谱峰**（太阳黑子 ~11 年周期）= 复特征值的辐角对应的频率。这和循环矩阵的特征值结构完全一致！

### 光伏中的 DFT

- **日周期提取**：功率序列 DFT → 频率 $1/24h$ 的峰 = 日变化；$1/8760h$ 的峰 = 年变化
- **去噪**：高频噪声（传感器抖动）通过低通滤波去除
- **特征工程**：DFT 系数可以作为 ML 模型的频域特征
- **SARIMA 的频域解释**：季节差分 $\nabla_{24}$ 相当于在频率 $k/24$ 处放置零点

## 全课程大统一：线性代数的知识网络

```
向量空间 → 基 → 维度 → 秩零度定理
    ↓                        ↓
线性变换 → 矩阵 → 四大子空间 → 正交互补
    ↓         ↓              ↓
特征值 → 对角化 → 谱定理 → 正定性
    ↓                        ↓
SVD ← 奇异值 = √(M^TM 特征值)
 ↓         ↓           ↓
PCA    低秩近似    伪逆/最小二乘
 ↓                      ↓
DFT  =  循环矩阵的特征分解  =  Box 的功率谱
```

**MIT 18.06 全课程的核心消息**：

> 线性代数不是"解方程组的工具"——它是**理解数据结构的语言**。
> 四大子空间告诉你"什么能被解释"；
> SVD 告诉你"按什么顺序重要"；
> 傅里叶变换告诉你"哪些频率在主导"。

## 对光伏预测项目的行动指南

基于 MIT 18.06 全课程的学习，对未来光伏预测框架的具体建议：

1. **数据探索第一步**：对气象特征矩阵做 SVD，看秩和条件数
2. **特征选择**：PCA 降维 or 看 $V$ 矩阵哪些原始特征贡献大
3. **多站点建模**：功率矩阵低秩分解 → 空间模式提取
4. **频域分析**：DFT 看周期性结构 → 确定 SARIMA 的 $s$
5. **数值稳定性**：用 QR 分解做回归，不要直接算正规方程
6. **正则化决策**：条件数 $\kappa > 100$ → 需要 Ridge/Lasso
7. **残差诊断**：残差协方差矩阵的特征值分布 → 检验模型充分性
