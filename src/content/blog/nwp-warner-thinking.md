## 二、因果推理链

### 推理链 1：为什么 NWP 是光伏日前预测的唯一可靠来源？

```
前提1（Yang Ch4.5.4）：辐照度的时空依赖完全来自移动云
    ↓
前提2（Yang Ch4.5.4）：天空相机假设云"冻结" → <30min 后失效
前提3（Yang Ch4.5.4）：卫星 CMV 假设云匀速直线运动 → <4h 后失效
    ↓
前提4（Warner Ch2）：只有原始方程能求解云的动力学演化
    ↓
前提5（Yang Ch5.6.2）：电网 DAM 需要 >36h 提前量
    ↓
结论：NWP 是日前预测的唯一可靠来源
```

这条推理链从 Warner Ch2 的原始方程出发，解释了为什么相机和卫星在日前尺度上失效——因为它们的物理假设（冻结场/匀速平流）在长时间后不成立，而只有 NWP 的原始方程才能模拟云的生成、发展和消散。

### 推理链 2：NWP 预测误差从哪里来？

```
Warner Ch2 原始方程 → 7 个变量需要求解
    ↓
问题1：方程是非线性耦合的 → 无解析解 → 必须离散化（Ch3）
    → 离散化引入截断误差
    ↓
问题2：次网格过程（湍流、云微物理）无法显式求解
    → 必须参数化（Ch4-5）
    → 参数化是对真实物理的近似 → 引入参数化误差
    ↓
问题3：初始条件来自有限的观测
    → 观测有测量误差 + 空间覆盖不足
    → 初始条件不完美 → 混沌放大（Ch8）
    ↓
总误差 = 截断误差 + 参数化误差 + 初始条件误差
```

**与 Yang 教材的对接**：Yang Ch3.1 说"预测误差有两个来源：初始条件不完美 + 动力学定律不完美"。Warner Ch2 进一步解释了"动力学定律不完美"的具体含义——不是方程本身有问题（原始方程是精确的），而是**求解方法**引入了误差（离散化 + 参数化）。

### 推理链 3：Reynolds 平均如何连接到辐照度预测？

```
Warner Ch2：大气有次网格湍流 → 无法直接解析
    ↓
Reynolds 分解：u = ū + u' → 平均后出现协方差项 -ρ(u'w')
    ↓
协方差项 = Reynolds 应力 = 湍流对平均流的影响
    ↓
必须参数化（Ch4）→ 边界层(PBL)参数化方案
    ↓
PBL 方案决定垂直混合强度
    ↓
分支A：强混合 → 水汽上升 → 成云 → GHI 减少
分支B：弱混合 → 稳定层结 → 少云 → GHI 增加
    ↓
Yang Ch7.1.2：PBL 方案是 WRF-Solar 最敏感的选择之一
```

**核心洞察**：从 Reynolds 平均这个纯数学操作，经过四步推理，直接连接到了光伏辐照度预测的精度。这就是"知识不是孤岛"的体现——一个 1895 年的湍流分解方法，决定了 2024 年的光伏预测质量。

### 推理链 4：静力近似的选择如何影响光伏预测？

```
Warner Ch2.3.1：静力近似过滤声波 → 允许更大时间步长 → 计算更快
    ↓
但：Δx < 10km 时静力近似失效
    ↓
GFS（全球模型，~13km）→ 可以用静力近似 → 粗分辨率但全球覆盖
HRRR（区域模型，3km）→ 必须用非静力方程 → 高分辨率但仅限 CONUS
    ↓
Yang Ch6.6 的对比：HRRR 辐照度预测优于 GFS
    ↓
原因之一：非静力方程能解析对流单体 → 更准确的云模拟
    ↓
但 HRRR 只覆盖美国 → 其他地区只能用 GFS 或 ECMWF
    ↓
WRF-Solar 的价值：用户可以自己运行非静力区域模型
    但需要气象学知识（Yang Ch4.6.1 的跨学科挑战）
```

**实际决策意义**：如果你在美国做光伏预测，用 HRRR（3km 非静力）；在中国，只能用 GFS/ECMWF（>10km 静力）或自己跑 WRF-Solar。这个决策的理论根基就是 Warner Ch2.3 的静力近似。

### 推理链 5：分辨率加倍为什么成本×8？（Ch3 新增）

```
CFL 条件：U·Δt/Δx < 1
    ↓
Δx 减半 → Δt 也必须减半（否则违反 CFL）
    ↓
2D 网格点数 ×4（面积不变，每维×2）
    ↓
总计算量 = 网格点数 × 时间步数 = 4 × 2 = 8
    ↓
实际还要更多：垂直层也可能加密 → ×16 甚至更多
```

**对光伏预测的决策意义**：在 BOSS 的 RTX 4050 上跑 WRF-Solar，9km → 3km 需要 ~27 倍计算量。必须权衡精度提升是否值得。

### 推理链 6：截断误差如何一路传播到辐照度？（Ch3 新增）

```
有限差分计算导数 → 截断误差（短波 6Δx 误差 17%）
    ↓
气压梯度力不准 → 地转风偏差
    ↓
散度不准 → 垂直运动偏差
    ↓
垂直运动驱动凝结/蒸发 → 云液态水含量偏差
    ↓
辐射方案用云水计算辐照度 → GHI 输出偏差
```

**核心洞察**：辐照度预测误差不只来自辐射参数化——它从最底层的导数近似就开始积累。这就是为什么 Yang 说"云微物理是瓶颈"但 Warner 强调"数值方法也很重要"——两者是乘法关系，不是加法关系。

### 推理链 7：为什么"云是辐照度预测的最大瓶颈"？（Ch4 新增）

```
辐射参数化物理已成熟（Beer定律+Planck函数）
    ↓
辐射方案的输入 = 云的三维分布和光学性质
    ↓
云的光学性质 ← 云微物理（云滴大小分布）
    ↓
云的存在 ← 对流参数化（何时何地触发凝结）
    ↓
对流的水汽供给 ← 边界层参数化（垂直混合强度）
    ↓
边界层的热力驱动 ← 地表能量平衡（辐照度的函数！）
    ↓
结论：辐照度预测误差的根源不在辐射方案本身，
而在"辐射→地表→PBL→对流→微物理→云→辐射"的整个循环
```

**核心洞察**：Yang 说"云微物理是瓶颈"，Warner 补充说"所有参数化方案都是一个循环系统"——不存在单点修复。

### 推理链 8：气溶胶如何影响光伏预测？（Ch4 新增）

```
人为/自然气溶胶 → CCN 浓度变化
    ↓
CCN 多 → 云滴多而小 → 光学厚度↑ → 反照率↑ → GHI↓
CCN 少 → 云滴少而大 → 光学厚度↓ → 降水效率↑ → 云更短命 → GHI↑
    ↓
运营 NWP 通常不模拟气溶胶（太贵）
但 ECMWF 同化 CAMS 气溶胶 → 辐照度预测更好
    ↓
Yang Ch5.4 REST2 需要 AOD(τ₅₅₀) 作为输入
Yang Ch4.5.1 晴空模型用 6 个透射率相乘，其中 Tₐ=气溶胶衰减
    ↓
结论：气溶胶不确定性 → 晴空模型不确定性 + NWP 云模拟不确定性
```

### 推理链 9：地表能量分配如何影响光伏预测？（Ch5 新增）

```
太阳辐射到达地面 R = (Q+q)(1-α) - I↑ + I↓
    ↓
R 分配为三部分：LE（潜热）+ H（感热）+ G（地热）
    ↓
Bowen ratio β = H/LE 决定能量去向：
  - 湿润地（β小）→ 大部分能量蒸发 → 边界层凉爽湿润
  - 干旱地（β大）→ 大部分能量加热大气 → 边界层热而深
    ↓
干旱：边界层更深→更强对流→更多午后积云→GHI ramp down
湿润：边界层较浅→对流较弱→积云较少→GHI更稳定
    ↓
结论：土壤湿度→Bowen ratio→PBL深度→对流触发→云→辐照度
```

**光伏实践意义**：灌溉季 vs 非灌溉季的光伏出力曲线会有系统性差异——灌溉降低 Bowen ratio → 午后 ramp 事件减少。

### 推理链 10：为什么 PILPS 发现复杂 LSM 不一定更好？（Ch5 新增）

```
复杂 LSM 有更多参数（植被、根系、土壤水力学...）
    ↓
每个参数都需要观测数据来定义——但大多数参数无法直接观测
    ↓
不确定的参数 × 多个 = 误差空间膨胀
    ↓
简单 LSM 参数少 → 可标定 → 实际精度可能更高
    ↓
类比：Yang 教材 Ch3 的 bias-variance tradeoff
  - 复杂模型 = 低偏差高方差
  - 简单模型 = 高偏差低方差
```

**启示**：这和机器学习的过拟合问题完全同构——模型复杂度要匹配数据的信息量。

### 推理链 11：数据同化的本质是什么？（Ch6 新增）

```
真实大气状态 x_true（未知）
    ↓
两个不完美的估计：
  背景场 x_b（上一次预报，系统误差 B）
  观测 y（稀疏、有仪器误差 R）
    ↓
最优组合：x_a = x_b + K(y - H(x_b))
  K = BH^T(HBH^T + R)^{-1}（卡尔曼增益矩阵）
    ↓
关键：B矩阵决定一切
  - 静态B（3DVAR）→ 不随天气型变化 → 限制分析质量
  - 动态B（EnKF）→ 用集合估计 → regime-dependent
  - 隐式B（4DVAR）→ 通过时间窗口传播 → 最优但需伴随模型
```

**对光伏**：NWP 辐照度预报质量 = 初始条件质量 × 模型质量。同化质量差 → 云场初始化错误 → 短期预报直接废掉。

### 推理链 12：为什么集成平均优于单一预报？（Ch7 新增）

```
非线性方程 → f(x̄) ≠ f̄(x)
    ↓
单一"最优"初始条件的预报 ≠ 最优预报
    ↓
多个成员的误差在不可预测分量上相互抵消
    ↓
集成平均 → 非线性滤波效应 → 更接近真实
    ↓
但！分岔时集成平均可能落在两支之间 → 不代表任何物理状态
    ↓
结论：集成提供的不是"最好的预报"，而是"不确定性的估计"
```

**对光伏**：概率预报（"明天 GHI 有 70% 概率 > 500 W/m²"）比确定性预报（"明天 GHI = 520 W/m²"）对电网调度更有价值。

### 推理链 13：可预测性的上限在哪里？（Ch8 新增）

```
Lorenz 蝴蝶效应 → 微小初始差异 → 几周后完全发散
    ↓
误差增长三阶段：诱导(10-15天) → 线性增长(~20天) → 饱和
    ↓
但！地表强迫提供"免费"可预测性：
  - 日变化：海陆风、山谷风、低空急流
  - 季节变化：太阳角度、植被状态
    ↓
光伏预测的好消息：辐照度的日变化主要由确定性太阳强迫驱动
光伏预测的坏消息：云的出现/消散是混沌的，难以超过几小时预测
```

### 推理链 14：代表性误差为什么不可消除？（Ch9 新增）

```
观测 = 某点某时刻的瞬时值
模型 = 网格框体积的时空平均值
    ↓
即使完美模型 + 完美仪器，两者也不相等
    ↓
代表性误差 ∝ 网格框内的次网格变率
    ↓
分辨率越高 → 代表性误差越小，但永远不为零
    ↓
有效分辨率 > 2Δx（数值耗散进一步平滑）
```

**对光伏**：地面辐照度站（点测量）验证 NWP 输出（网格平均）时，必须考虑代表性误差——特别是在云破碎的天气下。

### 推理链 15：MOS 后处理的困境（Ch12-13 新增）

```
MOS = 用历史预报-观测对建立统计回归
    ↓
优点：消除系统偏差，技巧等效于多年模型改进
    ↓
困境：每次改模型 → 统计关系失效 → 必须重新训练
    ↓
解决方案1：PP方法（用分析场而非预报建回归）→ 不依赖模型版本但不修正模型误差
解决方案2：动态MOS（短周期滚动训练）→ 快速适应但样本小
解决方案3：Reforecasts（用当前模型跑历史案例）→ 大样本但计算贵
```

**Yang Ch8 的对应**：Yang 的 EMOS/BMA 方法 = Warner Ch13 的集成后处理，完全同一框架。

### 推理链 16：再分析数据的双重性质（Ch16 新增）

```
再分析 = 用固定版本同化系统处理历史观测
    ↓
优点：长期一致的网格化数据，比原始观测更完整
    ↓
陷阱：降水和地表通量通常不被同化 → 纯模型产物
    ↓
ERA5 辐照度数据 = 模型辐射方案的输出，不是观测！
    ↓
用再分析验证NWP → 一个模型验证另一个模型 → 有偏
```

**对光伏**：ERA5 GHI 数据虽然方便但有系统偏差，必须和地面站或卫星反演数据交叉验证。

## 五、已解决的困惑

### ❓ Yang Ch7.1.1 说"7个原始方程"，但方程看起来很抽象，不知道每项的物理含义

✅ **Warner Ch2 完全解决了这个困惑。** 每个方程的每一项都有清晰的物理解释：气压梯度力驱动风、科里奥利力使风偏转、加热率 $H$ 包含辐射+潜热+感热、水汽源汇项 $Q_v$ 包含蒸发和凝结。

### ❓ Yang Ch7.1.2 说"参数化是 NWP 最具争议的主题"，但不知道为什么

✅ **Warner Ch2 的 Reynolds 平均给出了答案。** 参数化的数学起源是 Reynolds 应力项——它本质上是一个**闭合问题**（方程数少于未知数）。任何参数化方案都是对这个闭合问题的一种近似，没有"唯一正确答案"。这就是为什么争议永远存在。

### ❓ CFL 条件的具体推导是什么？时间步长限制如何影响 NWP 的实时性？

✅ **Warner Ch3 详细推导。** CFL 条件：$\Delta t \leq \Delta x / c$，其中 $c$ 是最快信号传播速度（声波~340m/s）。这意味着 $\Delta x = 1$ km 时 $\Delta t \leq 3$ s，每模拟 1 天需要 28800 步——这就是高分辨率 NWP 计算量爆炸的根本原因。半隐式方法可以放宽 CFL 限制但引入精度损失。

### ❓ 云微物理参数化是不是辐照度预测的最大误差来源？

✅ **Warner Ch4 确认并深化。** 不仅云微物理本身是瓶颈，整个参数化循环系统（辐射→地表→PBL→对流→微物理→云→辐射）都是耦合的。辐射方案的物理已经成熟，瓶颈在于它的**输入**——云的三维分布。而云的质量取决于上游所有参数化方案的协同。气溶胶（CCN 浓度）的不确定性还会通过影响云滴大小分布进一步放大误差。

### ❓ 地表参数化的误差贡献有多大？植被/土壤参数不确定性是否被低估？

✅ **Warner Ch5 部分验证。** PILPS 23 个 LSM 对比显示复杂 LSM 不一定优于简单 LSM——因为参数不确定性膨胀。土壤含水量对热传导率的影响跨两个数量级（干土 vs 湿土），含水量误差会被放大。LDAS 需数月 spin-up 说明土壤状态初始误差衰减极慢。**结论**：地表确实是被低估的误差源，但不如云致命——地表过程时间尺度（小时-天）慢于云（分钟-小时），误差传播较慢。

### ❓ ECMWF 的 4D-Var 比 3D-Var 好多少？

✅ **Warner Ch6 解答。** 4DVAR 通过时间窗口内的所有观测约束初始条件，隐式地让 B 矩阵随时间演化（regime-dependent）。3DVAR 的 B 是静态的（NMC 法估计，不随天气型变化）。但 4DVAR 需要伴随模型（构建+维护是巨大工程）且假设模型完美（Q=0）。EnKF 不需要伴随但需要足够多的成员。混合方法 B_hybrid 优于两者单独使用。

### ❓ Warner 的气象集成 vs Yang 的统计集成是否对应？随机参数化和模型不确定性扰动是同一回事吗？

✅ **Warner Ch7 解答。** 气象集成的三类不确定性来源——初始条件（Bred vectors/奇异向量/EnKF）、模型物理（多方案/随机参数化）、边界条件——和 Yang Ch3.3 的框架部分对应但不完全同构。随机参数化（Ch4 Grell & Devenyi 2002）确实是集成方法的一种特殊形式：给参数化趋势乘以 [0.5,1.5] 随机因子 → 等价于"给模型加噪声"。

### ❓ Warner 的 NWP 验证和 Yang 的预测验证框架有多大重叠？

✅ **Warner Ch9 解答。** 高度重叠。MAE/RMSE/Bias/AC/POD/FAR/CSI/HSS 完全一致。Yang 额外强调的：①辐照度验证需考虑日变化周期（不能用持续性作参考，要用日变化持续性）②GHI 的空间变率大（云破碎时 1km 内可变化 50%），代表性误差特别严重。

### ❓ 辐照度预测的验证有什么特殊之处？

✅ **Warner Ch9 的代表性误差概念直接适用。** 地面辐照度站（点测量）vs NWP 网格平均——在积云对流天气下，1km² 内 GHI 可能从 200 到 900 W/m²，代表性误差可能比模型误差还大。对象验证方法（MODE/SAL）可能比逐点验证更适合评估辐照度预报的空间模式。

### ❓ 实验设计中如何隔离单个因素的贡献？

✅ **Warner Ch10 给出了完整方法论。** 因子分离法（Stein & Alpert 1993）：N 个因素需要 2^N 次模拟，每个因素的纯贡献 = 包含该因素的模拟 - 不包含的模拟 - 交互项。OSSE（观测系统模拟实验）在"虚拟大气"中测试新观测系统的价值——不用花钱发卫星就能评估效果。伴随敏感性方法用代价函数对初始条件的梯度定义"对预报影响最大的区域"——这和 Ch6 的自适应观测（定向投放探空仪）是同一个工具。

### ❓ 模型输出有哪些高级分析手段？

✅ **Warner Ch11 系统梳理。** EOF/PCA 提取主要变率模态——辐照度时空数据的第一模态通常对应大尺度云覆盖的日变化。SOM（自组织映射）自动分类天气型——可以将辐照度日型分为"晴天/薄云/破碎积云/全阴"等类别，每类用不同预测策略。CCA/SVD 找耦合模态——大尺度环流模式和局地辐照度之间的统计关系，这正是统计降尺度的数学基础。

### ❓ 业务 NWP 系统面临什么工程约束？

✅ **Warner Ch12 给出了运营视角。** CFL 违例是运营中断的最常见原因——极端风速导致不稳定。解决方案包括自适应时间步、半隐式方法、以及最朴素的"重启并减小时间步"。预报必须比天气跑得快——ECMWF 的 IFS 10 天全球预报必须在约 1 小时内完成。模型升级是渐进的——不能频繁改动，因为下游的 MOS 统计关系会失效。预报员的角色从"修正模型"转向"解读不确定性"和"聚焦高影响天气"。

### ❓ 统计后处理到底能有多大改善？MOS 的局限性在哪？

✅ **Warner Ch13 明确回答。** MOS 的技巧改进等效于"多年的模型物理改进"——这说明统计方法的 ROI 极高。但 MOS 的困境：每次改模型 → 统计关系失效 → 必须积累新的预报-观测对重新训练。三种应对：①Perfect-Prog（用分析场而非预报建回归，不依赖模型版本但不修正模型误差）②动态 MOS（短周期滚动训练，快速适应但样本小）③Reforecasts（用当前模型跑历史案例，大样本但计算贵）。BMA/EMOS 将集成输出转化为校准的概率密度函数——Yang 教材 Ch8 的方法与此完全对应。

### ❓ NWP 在光伏之外还有哪些耦合应用？

✅ **Warner Ch14 展示了 NWP 的广泛下游应用。** 风功率 ramp 事件的四个物理原因：锋面过境、低空急流高度振荡、地形重力波、对流出流——辐照度也有 ramp 事件（云团突然遮蔽/消散），物理机制不同但预测挑战相似。空气质量预测需要 NWP 提供风场+边界层高度+降水（湿沉降）。传染病预测（疟疾/登革热）用 NWP 的温度+湿度驱动蚊虫种群模型——滞后相关性使提前预测成为可能。电力需求模型（Taylor & Buizza 2003）用集成预报提供 10 天概率预测。

### ❓ DNS/LES/RANS 的适用范围是什么？terra incognita 如何处理？

✅ **Warner Ch15 给出完整 CFD 分类。** DNS 解析所有湍流尺度（Re ~O(10³) 以下可行，大气 Re ~O(10⁸) 完全不可行）。LES 显式解析大涡、参数化小涡（网格 ~O(10m)，需要 SGS 模型）。RANS Reynolds 平均后参数化所有湍流（经典 NWP 方法）。**terra incognita**（Wyngaard 2004）：Δx = 100m-1km，湍流既不能完全参数化也不能完全解析——和 Ch4 的灰色地带（Δx = 1-10km，对流半参数化半解析）是同构问题，不同尺度但相同的哲学困境。未来超高分辨率 NWP 必然要面对这个问题。

### ❓ 再分析数据可以直接当观测用吗？气候降尺度的两种方法各有什么优劣？

✅ **Warner Ch16 详细解答。** 再分析 ≠ 观测。降水和地表通量通常不被同化（纯模型产物）。ERA5 辐照度 = 辐射参数化方案输出 → 有系统偏差。即使被同化的变量（温度/风/湿度），分析结果也是模型动力学和观测的混合产物——再分析之间的差异可以量化不确定性（如 ERA5 vs JRA-55 vs MERRA-2）。统计降尺度计算便宜但假设关系平稳（气候变化下可能失效，危险）、低估极端值。动力降尺度（区域气候模型 RCM）物理一致、能捕捉极端值，但计算贵且同样受 GCM 偏差传递影响。两者都对 GCM 的大尺度偏差敏感。对光伏：区域气候模型可提供高分辨率长期辐照度数据，用于电站选址和长期发电量评估——但必须和地面站/卫星数据交叉验证。
