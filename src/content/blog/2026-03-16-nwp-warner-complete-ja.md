## 1. 全書の知識連鎖

Warner の全書は、**物理からエンジニアリングへの一本の連鎖**に凝縮できる：

```
物理法則（原始方程式 7 本）
    ↓ 離散化
数値手法（有限差分・スペクトル法 + CFL 制約）
    ↓ クロージャ
パラメータ化（雲微物理 + 対流 + PBL + 放射 + 地表）
    ↓ 初期化
データ同化（OI → 3DVAR → 4DVAR/EnKF）
    ↓ 不確実性定量化
アンサンブル手法（初期条件擾乱 + モデル不確実性 + マルチモデル）
    ↓ 検証
技巧評価（バイアス/RMSE/AC + 確率的検証 + オブジェクト検証）
    ↓ 応用
業務予報 + 後処理(MOS) + 結合応用（大気質・エネルギー・感染症）
```

**重要な認識**：NWP は「方程式を解く」のではない——それは**エンジニアリングシステム**であり、すべての環に誤差があり、すべての環が次の環に影響する。

## 2. 六大モジュールの核心的な収穫

### モジュール 1：数学的基礎（Ch2–3）

**Ch2 支配方程式**：7 本の原始方程式（3 運動量方程式 + 連続方程式 + 熱力学方程式 + 水蒸気方程式 + 状態方程式）。Reynolds 平均がクロージャ問題を導入する——これがすべてのパラメータ化の数学的起源である。静力学近似により鉛直運動量方程式が診断方程式に単純化され、音波は消去されるがモデルの能力は制限される。

**Ch3 数値手法**：CFL 条件 $\Delta t \leq \Delta x / c$ は、高解像度 NWP の計算量が爆発的に増大する根本的な理由である。1 km グリッド → 3 秒タイムステップ → 1 日 28,800 ステップ。半陰的手法は CFL を緩和するが精度を犠牲にする。数値分散（短波が遅すぎる）と数値散逸（短波が平滑化される）はすべての有限差分スキームに固有の欠陥である。

### モジュール 2：物理的表現（Ch4–5）

**Ch4 パラメータ化** ——全書で最も核心的な章。パラメータ化は独立したモジュールではなく、**循環システム**である：放射→地表→PBL→対流→微物理→雲→放射。放射スキームの物理は成熟しており、ボトルネックは入力（雲の三次元分布）にある。「グレーゾーン」（3–9 km）の対流パラメータ化は「二重計算」の恐れがある。

**Ch5 地表過程**：$R = LE + H + G$ が循環システムの出発点。ボーエン比がエネルギーの行方を決定する——乾燥地域では顕熱↑→境界層が深くなる→対流が強まる→積雲が増加→GHI が低下する。PILPS の教訓：複雑な LSM が単純な LSM より必ずしも優れているわけではない（パラメータ不確実性 > モデル複雑度）。

### モジュール 3：初期化と不確実性（Ch6–8）

**Ch6 データ同化** ——全書で数学的密度が最も高い章。核心公式：$x_a = x_b + K(y - H(x_b))$。

| 手法 | B 行列 | 時間次元 | 随伴モデルが必要 | 業務利用 |
|---|---|---|---|---|
| OI | 事前計算、静的 | なし | 否 | 初期時代 |
| 3DVAR | NMC 法、静的 | なし | 否 | 広く普及 |
| 4DVAR | 暗黙的に発展 | あり | 是 | ECMWF |
| EnKF | アンサンブル推定、動的 | あり | 否 | 実験→業務 |

**Ch7 アンサンブル手法**：アンサンブル平均は単一予報より優れている（非線形フィルタリング効果）。スプレッド ↔ 不確実性（キャリブレーションが必要）。初期条件擾乱の 3 手法：EnKF、Bred vectors、特異ベクトル。確率的パラメータ化（Ch4 の拡張）がスプレッドを増大させる。

**Ch8 予測可能性**：Lorenz のバタフライ効果。誤差成長の 3 段階。地表強制が「無料の」予測可能性を提供する（日変化・季節変化の決定論的成分）。ブロッキングイベント → 予測可能性が急激に低下。

### モジュール 4：評価（Ch9–11）

**Ch9 検証手法**：RMSE/バイアス/AC が基本。確率的検証：信頼性図、ランクヒストグラム、Brier スコア。オブジェクト検証（MODE/SAL）が「二重ペナルティ」問題を解決する。有効解像度 > 2Δx。

**Ch10 実験設計**：OSSE が新しい観測システムの価値を評価する。随伴感度分析が最も重要な初期条件領域を特定する。再予報が大サンプル検証を提供する。

**Ch11 出力解析**：EOF/PCA による次元削減が主要な変動モードを抽出する。SOM が天気型を自動分類する。これらの手法は日射量の時空間解析に直接適用できる。

### モジュール 5：業務運用（Ch12–13）

**Ch12 業務 NWP**：ECMWF IFS がグローバルの金標準（~9 km）。HRRR は 1 時間ごとに更新（3 km）——太陽光発電の日内予測に最適。MOS/PPM が後処理の標準。

**Ch13 後処理**：BMA/EMOS がアンサンブル出力をキャリブレーション済み確率分布に変換する。太陽光発電については：Yang 教材 Ch8 の手法がこれに完全に対応する。

### モジュール 6：拡張（Ch14–16）

**Ch14 結合応用**：NWP と大気質・風力/太陽エネルギー・感染症予測の結合。WRF-Solar = NWP + 放射モデル → GHI/DNI 予報。

**Ch15 CFD**：DNS/LES の計算上の限界。「テラ・インコグニタ」（100 m – 1 km）が Ch4 のグレーゾーンと呼応する。

**Ch16 気候モデリング**：AOGCM + CMIP マルチモデルアンサンブル。ERA5 再解析は長期日射量データの重要な供給源。気候ダウンスケーリングが地域太陽光発電計画のための高解像度データを提供する。

## 3. 太陽光発電予測への核心的な示唆

全書を読み終えて、太陽光発電出力予測フレームワークへの理解は次のようにまとめられる：

### 1. NWP 日射量予報の誤差源

```
初期条件誤差（Ch6）
    + 雲微物理誤差（Ch4）← 最大の寄与
    + エアロゾル不確実性（Ch4）
    + 地表アルベド誤差（Ch5）
    + 数値分散・散逸（Ch3）
    + 解像度制限（Ch3 CFL + Ch4 グレーゾーン）
    = NWP GHI 予報誤差
```

### 2. 後処理は必須

NWP の出力は太陽光発電予測にそのままでは使えない——系統的バイアス、解像度不足、確率情報の欠如。MOS/BMA/EMOS（Ch12–13）+ Yang 教材 Ch8 の統計手法 → キャリブレーション済み確率予報。

### 3. アンサンブルが単一決定論的予報を上回る

単一の決定論的予報は信頼性が低い。アンサンブル手法（Ch7）+ 確率的キャリブレーション（Ch13）→ 信頼性の高い不確実性推定 → 系統運用の意思決定支援。

### 4. 時間スケールが手法を決める

| 時間スケール | 最適手法 | Warner の対応 |
|---|---|---|
| 0–6 時間 | 統計・リモートセンシング | Ch8 亜日次予測可能性 |
| 6–72 時間 | NWP + MOS | Ch12 + Ch13 |
| 1–2 週間 | アンサンブル NWP | Ch7 |
| 季節 | 気候モデル | Ch16 |
