# MIT 18.06 線形代数 Part 3：固有値・SVD・データサイエンス

> **コース**：MIT 18.06 Linear Algebra, Spring 2025
> **範囲**：講義15-27（行列式・固有値・SVD・低ランク近似）

## 講義15-19：行列式

### 行列式 = 体積の符号付きスケーリング因子

$$\det A = \text{線形変換 } A \text{ が } n \text{ 次元体積に与える符号付きスケーリング倍率}$$

**重要な性質**：
- 積の公式：$\det(AB) = (\det A)(\det B)$
- $\det A \neq 0 \Leftrightarrow A$ は可逆
- 上三角行列：$\det =$ 対角要素の積

### 大公式（講義16）

$$\det A = \sum_{\sigma} (\text{sgn}\,\sigma) \prod_{i=1}^n a_{i,\sigma(i)}$$

$n!$ 個の置換にわたる和——理論的には美しいが、計算にはガウス消去法の方がはるかに効率的である。

### Laplace展開と余因子行列（講義17, 19）

余因子 $C_{ij} = (-1)^{i+j}\det M_{ij}$（$M_{ij}$ は第 $i$ 行と第 $j$ 列を取り除いた部分行列）

$$A^{-1} = \frac{1}{\det A}C^T$$

**Cramerの公式**：$x_j = \frac{\det A_j}{\det A}$（$A_j$ は $A$ の第 $j$ 列を $\mathbf{b}$ で置き換えたもの）

**実際的な価値**：行列式は理論的な導出（特性方程式 $\det(A - \lambda I) = 0$）では優れた道具だが、大規模な数値計算ではほとんど直接使われない。

## 講義20-22：固有値と対角化

### 固有値問題

$$A\mathbf{v} = \lambda\mathbf{v}, \quad \mathbf{v} \neq \mathbf{0}$$

行列 $A$ は固有方向において方向を変えず、ただスケーリング（倍率 $\lambda$）するのみ。

### 特性多項式

$$p_A(\lambda) = \det(A - \lambda I) = (-1)^n\lambda^n + (-1)^{n-1}\alpha\lambda^{n-1} + \cdots + \det A$$

ここで $\alpha = \text{tr}(A)$。$n$ 個の根（複素数や重根を含む）= $n$ 個の固有値。

### 対角化

$$A = EDE^{-1}$$

$E$ の列 = 固有ベクトル、$D$ の対角要素 = 固有値。

**対角化可能の条件**：すべての固有値について幾何的重複度 = 代数的重複度の場合に限る。

- すべての固有値が異なる → 常に対角化可能
- 重根がある → 必ずしも対角化できない（例：Jordan ブロック）

### Box ARIMA との深い関係

Box 第3章の AR(p) 特性方程式 $\phi(B) = 0$ は本質的に $\phi$ 演算子の固有値を求めている：

- 根が単位円の外側 → $|\lambda| < 1$ → 定常（減衰）
- 根が単位円上 → $|\lambda| = 1$ → 非定常（ARIMAの $d$ 個の単位根）
- 根が単位円の内側 → $|\lambda| > 1$ → 爆発

AR(2) の複素固有値は**擬似周期的挙動**（減衰正弦波）を生む——太陽黒点データはAR(2)の古典的な例である。

Warner NWP 第5章の音波/重力波解析も固有値問題である：大気方程式系を線形化したときの固有値が波の伝播速度と安定性を決定する。

## 講義23：複素固有値とスペクトル定理

### 実行列の複素固有値は共役対で現れる

$\lambda = a + bi$ が固有値ならば、$\bar{\lambda} = a - bi$ も固有値である。

### スペクトル定理（対称行列）

> **$A$ が実対称行列（$A = A^T$）ならば：**
> 1. すべての固有値が**実数**
> 2. 固有ベクトルが**正規直交基底**を形成する
> 3. $A = E D E^T$（$E$ は直交行列）

**これは線形代数で最も美しい定理の一つである。** 対称行列の固有値分解 = 回転 + スケーリング + 逆回転。

**太陽光予測への応用**：共分散行列 $\Sigma = \frac{1}{n}X^TX$ は必ず対称半正定値 → 必ず直交固有値分解を持つ → これがPCAの数学的基盤。

## 講義24：正定値行列とSVDの導入

### 正定値性

対称行列 $A$ が**正定値（PD）** $\Leftrightarrow$ すべての固有値 $> 0$ $\Leftrightarrow$ $\mathbf{x}^T A\mathbf{x} > 0, \forall \mathbf{x} \neq 0$

**半正定値（PSD）**：すべての固有値 $\geq 0$

重要な事実：**任意の行列 $M$ に対して、$M^TM$ と $MM^T$ はどちらも半正定値である。**

### SVD：特異値分解

任意の $n \times p$ 行列 $M$（正方行列である必要はない！）に対して：

$$M = U\Sigma V^T$$

- $U$：$n \times r$、左特異ベクトル（$M$ の列空間の正規直交基底）
- $\Sigma$：$r \times r$、特異値の対角行列（$\sigma_1 \geq \sigma_2 \geq \cdots \geq \sigma_r > 0$）
- $V$：$p \times r$、右特異ベクトル（$M$ の行空間の正規直交基底）

**SVDは四つの基本部分空間すべての正規直交基底を同時に明らかにする！**

## 講義25-26：SVDの計算と幾何的解釈

### 計算手順

1. $A = M^TM$ の固有値分解：$A = EDE^T$
2. $V$ = $E$ の前 $r$ 列（$d_i > 0$ に対応する固有ベクトル）
3. $\sigma_i = \sqrt{d_i}$
4. $U = MV\Sigma^{-1}$

### 幾何的意味

- **作用素ノルム** $\|M\|_{op} = \sigma_1$：$M$ が単位球を引き伸ばせる最大倍率
- **SVDによる擬似逆行列**：$M^+ = V\Sigma^{-1}U^T$
- **条件数** $\kappa = \sigma_1/\sigma_r$：大きいほど悪条件

## 講義27：低ランク近似と画像圧縮

### Eckart-Young 定理

$$M_k = U_k\Sigma_k V_k^T = \text{最良の階数-}k\text{ 近似}$$

スペクトルノルム・Frobeniusノルム・核ノルムのいずれの下でも最適。

### 画像圧縮

元の画像 $M$：$n \times p$ ピクセル（$np$ 個の値を保存）
圧縮後 $M_k$：$(n + p)k + k$ 個の値を保存

$k \ll \min(n,p)$ のとき、**圧縮率は非常に高い**。

### 太陽光予測への示唆

**多地点発電量データ行列**（$m$ 個の時間ステップ × $n$ 個の発電所）：

階数 $r \ll n$（隣接する発電所が高い相関を持つため）ならば：
- 低ランク近似 $M_k$ でデノイズ
- $U_k$ の列 = 「典型的な発電パターン」（晴天・部分曇り・曇天など）
- $V_k$ の列 = 「空間パターン」（どの発電所が似た挙動を示すか）
- $\Sigma_k$ = 各パターンの「エネルギー」

Yang 第12章の階層的予測（hierarchical forecasting）は本質的に多地点発電量行列の低ランク構造を利用している。

**気象特徴行列の次元削減**：
- 元の特徴：温度・湿度・気圧・風速・風向・雲量・日射量……（$p$ 次元）
- SVD/PCAで $k$ 次元に削減 → 冗長性の排除 → 過学習の低減

## 深い洞察：SVDがすべてを統一する

SVDは線形代数の「究極の武器」——ほぼすべての重要な概念をSVDを通じて理解できる：

- **四つの基本部分空間**：$U$ の列 = $C(M)$ の基底、$V$ の列 = $C(M^T)$ の基底
- **階数**：$\sigma_i > 0$ の個数
- **最小二乗法**：$\hat{\mathbf{x}} = M^+\mathbf{b} = V\Sigma^{-1}U^T\mathbf{b}$
- **PCA**：データ行列の右特異ベクトル = 主成分方向
- **低ランク近似**：Eckart-Young → データ圧縮/デノイズ
- **条件数**：$\sigma_1/\sigma_r$ → 数値安定性の診断
- **正則化**：Tikhonov正則化 = 小さな特異値を切り捨てること

### 三つの教科書におけるSVD

| 教科書 | SVDの役割 |
|--------|-----------|
| **Box** | 残差共分散行列の固有値分解 → 主成分残差分析 |
| **Warner** | EOF（経験的直交関数）= 気象場のPCA = 気候データ行列のSVD |
| **Yang** | 多地点予測のための空間パターン抽出 |

## モジュールまとめ

講義15-27は行列式 → 固有値 → SVD へと段階的に行列の内部構造を明らかにする。

**コアな思想のつながり**：

$$\underbrace{\det(A-\lambda I) = 0}_{\text{特性方程式}} \to \underbrace{A = EDE^{-1}}_{\text{対角化}} \to \underbrace{A = EDE^T}_{\text{スペクトル定理（対称）}} \to \underbrace{M = U\Sigma V^T}_{\text{SVD（任意行列）}}$$

各ステップは前のステップの一般化である。SVDが最終形態——任意の行列に適用でき、同時に最良近似を与える。
