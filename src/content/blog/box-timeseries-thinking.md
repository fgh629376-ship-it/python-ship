# Box 时间序列分析 — 思维训练

> 不是记住结论，而是训练推理过程。每读一章，追问"为什么是这样？"、"和其他知识有什么联系？"

---

## 一、跨章推理链

### 推理链 1：从白噪声到 ARIMA 的逻辑链
**Ch1 线性滤波模型 → Ch3 ARMA → Ch4 ARIMA**

1. 所有可观测时序都可以看作白噪声 $a_t$ 经过线性滤波器的输出：$z_t = \psi(B)a_t$
2. 如果 $\psi(B)$ 可以表示为 $\phi^{-1}(B)\theta(B)$（有理函数），就得到 ARMA：$\phi(B)\tilde{z}_t = \theta(B)a_t$
3. 如果序列非平稳但差分后平稳，引入 $(1-B)^d$，得到 ARIMA：$\phi(B)\nabla^d z_t = \theta(B)a_t$

**关键推理**：为什么有理函数近似够用？→ 简约原则。Padé 逼近理论告诉我们，低阶有理函数（分子+分母各 ≤2 阶）可以高精度逼近很复杂的无穷级数。

### 推理链 2：预测与控制的统一
**Ch1.1.1 预测 + Ch1.1.5 控制 → Ch5 + Ch15**

1. 预测问题：最小化 $E[(z_{t+l} - \hat{z}_t(l))^2]$
2. 控制问题：最小化 $E[\varepsilon_t^2]$，其中 $\varepsilon_t = Y_t - T$
3. 控制 = 预测"无控制时的偏差" + 计算补偿调整量
4. **最小 MSE 预测 → 最小 MSE 控制**

**关键推理**：这个统一不是偶然的——两者都归结为条件期望的最优性。$\hat{z}_t(l) = E[z_{t+l} | z_t, z_{t-1}, \ldots]$ 是条件期望，而条件期望就是 $L^2$ 空间中的正交投影，天然最小化 MSE。

### 关键跨教材洞察

**Box 的线性滤波 ↔ Warner 的数值滤波**：
- Box 的 $\psi(B)$ 权重序列就是数字滤波器的冲击响应
- Warner Ch4 的 Robert-Asselin 时间滤波器是一种特定的 AR 型滤波器
- NWP 中的隐式时间差分格式（如半隐式法）等价于在频域上对高频分量施加特定的 $\psi$ 权重

**Box 的 ARIMA 差分 ↔ Yang 的去趋势**：
- ARIMA 用 $\nabla^d = (1-B)^d$ 消除趋势，这是一种"盲"去趋势（不假设趋势形式）
- Yang Ch7 用晴空模型物理去趋势：$\text{anomaly}_t = \text{actual}_t - \text{clearsky}_t$
- **最佳实践**：物理去趋势（Yang）+ 统计差分（Box）互补——物理模型去掉可预测的趋势，统计方法处理残余非平稳性

## 三、已解决的困惑

### 困惑 1：为什么 MA 模型里用"减号"？（Ch1）
**问题**：AR 模型写 $\tilde{z}_t = \phi_1 \tilde{z}_{t-1} + a_t$（加号），为什么 MA 模型写 $\tilde{z}_t = a_t - \theta_1 a_{t-1}$（减号）？

**解答**：这是一个约定（convention），目的是让 AR 和 MA 的对偶关系更简洁。如果 MA 用加号，那么 AR↔MA 的转换公式会多出负号。用减号约定后：
- AR(1)：$\phi(B) = 1 - \phi_1 B$
- MA(1)：$\theta(B) = 1 - \theta_1 B$

两者形式完全对称，$\phi(B)\tilde{z}_t = \theta(B)a_t$ 变得优雅。

### 困惑 2：ARIMA 中的 "I" 为什么叫"积分"？（Ch1）
**问题**：差分 $\nabla^d$ 使非平稳变平稳，那"积分"和差分有什么关系？

**解答**：名字来自**逆运算**。$\nabla = (1-B)$ 是差分，它的逆 $S = \nabla^{-1} = (1-B)^{-1} = 1 + B + B^2 + \cdots$ 是求和（summation），离散版的"积分"。ARIMA 过程 $z_t$ 是平稳 ARMA 过程 $w_t$ 经过 $d$ 次求和/积分得到的：$z_t = S^d w_t$。

### 困惑 3：为什么简约原则这么重要？（Ch1.3）
**问题**：现代计算能力这么强，为什么不能用更多参数？

**解答**：Box 的洞察超越了计算能力问题。核心原因是**数据量有限**：
1. 参数越多，需要的数据量以超线性速度增长（维度诅咒）
2. 参数之间的关系在有限数据下难以被识别（Box 举的 $\omega_0, \omega_1, \ldots, \omega_s$ 例子）
3. 过拟合的模型在预测时泛化能力差

即使在 2026 年的深度学习时代，当你只有一个电站 3 年的 15 分钟数据（~10 万个点），ARIMA(1,1,1) 的 5 个参数可能比 10 万参数的模型更可靠。

### 推理链 3：正定性的两面——时域约束与频域保证
**Ch2.1.3 自协方差矩阵正定 ↔ Ch2.2.3 谱非负**

1. 自协方差矩阵 $\boldsymbol{\Gamma}_n$ 必须正定（因为任何线性组合 $L_t$ 的方差 > 0）
2. 正定性约束了 $\rho_k$ 的取值范围（不是随便取的！）
3. 等价条件：谱密度 $p(f) \geq 0$ 对所有 $f$
4. 这就是 Herglotz 定理的内容

**关键推理**：正定性是"合法 ACF"的充要条件。如果你随意构造一个 $\{\rho_k\}$ 序列，它不一定对应任何实际的随机过程——必须检查正定性。在编程中，这意味着从数据估计的 ACF 截断后需要正定化处理。

### 推理链 4：Bartlett 公式为什么需要知道真实的 ρ？
**Ch2.1.6 标准误差 → Ch6 模型识别的困难**

1. $\text{var}[r_k]$ 的公式里包含真实的 $\rho_v$（未知！）
2. 实际应用中只能用估计值 $r_v$ 代替 → 引入额外不确定性
3. 对白噪声（$q=0$）有简洁公式 $1/\sqrt{N}$，但对非白噪声需要 bootstrapping
4. **这就是为什么模型识别是"艺术"而非纯科学**——ACF 的不确定性使得精确判断 $p, q$ 很困难

### 推理链 5：周期图 → 方差分析 → 谱估计的演化
**Ch2.2.1-2.2.3 从离散到连续**

1. 周期图：将方差分解到 $N/2$ 个离散频率（傅里叶级数）
2. 方差分析：每个频率占 2 个自由度，$\chi^2(2)$ 分布（如果白噪声）
3. 样本谱：允许连续频率，但**剧烈波动**（本质是用太窄的频率窗口）
4. 平滑谱估计：引入窗函数 $\lambda_k$ 或 $W(f_j)$，牺牲分辨率换取稳定性

**与 NWP 的平行**：这和 Warner Ch4 的谱分析完全对应——NWP 用 FFT 分析数值解的频率特性，检测计算噪声；Box 用同样的工具分析时间序列的频率特性，识别周期性成分。

## 三B、已解决的困惑（Ch2）

### 困惑 4：为什么 ACF 估计用 $1/N$ 而不是 $1/(N-k)$？（Ch2.1.5）
**问题**：$c_k = \frac{1}{N}\sum_{t=1}^{N-k}(z_t-\bar{z})(z_{t+k}-\bar{z})$ 用 $N$ 作分母，但实际求和只有 $N-k$ 项。

**解答**：用 $1/N$ 有两个好处：
1. 保证估计的 ACF 矩阵是半正定的（$1/(N-k)$ 不保证）
2. 大样本下两者渐近等价
3. Box 引用了 Jenkins & Watts [170] 的系统比较，结论是 $1/N$ 更优

### 困惑 5：样本谱为什么"剧烈波动"？（Ch2.2.3）
**问题**：$I(f)$ 作为谱的估计为什么不好？

**解答**：$I(f)$ 的方差**不随 $N$ 增大而减小**（这与直觉相反！）。原因是每个 $I(f_i) \sim \sigma^2\chi^2(2)$，$\chi^2(2)$ 的变异系数 = 100%，不管 $N$ 多大。解决方案是平滑——把相邻频率的 $I(f)$ 平均，相当于增加自由度，但损失频率分辨率。

### 推理链 6：Wold 分解 → ARMA 的合法性 → Padé 近似
**Ch3.1 Wold 定理 → Ch3.1.4 有理函数参数化**

1. Wold 定理：任何平稳过程 = 白噪声的线性滤波（$\tilde{z}_t = \psi(B)a_t$）
2. $\psi(B)$ 可能有无穷多项 → 不可操作
3. 用有理函数 $\psi(B) = \theta(B)/\phi(B)$ 近似 → ARMA(p,q)
4. 这就是 Padé 近似——低阶有理函数比同阶多项式逼近更好
5. **简约原则的数学根基**：$p+q \leq 4$ 的 ARMA 在实践中几乎总是够用

### 推理链 7：ACF/PACF 截断 = 模型的"指纹"
**Ch3.2-3.4 → Ch6 模型识别**

1. AR(p)：PACF 在 lag $p$ 截断（因为 $\phi_{kk}$ 是 $k$ 阶回归的最后一个系数，$k > p$ 时为零）
2. MA(q)：ACF 在 lag $q$ 截断（因为 lag $> q$ 的自协方差涉及不重叠的冲击，协方差为零）
3. ARMA(p,q)：两者都拖尾（但模式可辨：ACF 从某 lag 开始由 $\phi$ 的差分方程控制）
4. **实际操作**：先画 ACF/PACF → 截断模式 → 初步判阶 → 估计 → 诊断

**这就是 Ch6 模型识别的全部逻辑基础。**

## 三C、已解决的困惑（Ch3）

### 困惑 6：为什么可逆性很重要？（Ch3.1.3）
**问题**：MA(1) 无论 $\theta_1$ 取什么值都平稳，为什么还要限制 $|\theta_1| < 1$？

**解答**：可逆性保证**当前值可以用过去值合理表达**。如果 $|\theta_1| \geq 1$，则 $\pi_j = -\theta_1^j$ 发散——当前 $z_t$ 对很久以前的 $z_{t-k}$ 有越来越大的依赖权重，这物理上不合理。

更深层原因：给定一个 ACF，有两个 MA(1) 模型可以产生它（$\theta_1$ 和 $\theta_1^{-1}$），可逆性条件选择了唯一的那个。

### 困惑 7：AR(2) 的"伪周期"是什么意思？（Ch3.2.4）
**问题**：AR(2) 能产生周期性吗？和真正的周期性有什么区别？

**解答**：AR(2) 的复数根产生的 ACF 是**阻尼正弦波** $\rho_k = D^k \sin(2\pi f_0 k + F)/\sin F$，不是固定振幅的正弦波。物理上，这是一种"准周期"——每一个"周期"都比上一个弱。

真正的周期性（如日周期、年周期）是**确定性成分**，需要用傅里叶项或季节性差分处理（Ch9），不能靠 AR 的复数根。AR(2) 的伪周期适合描述"天气系统的约3-7天准周期"这类统计规律性。

### 推理链 8：预测更新公式 → 实时系统设计
**Ch5.2.3 → 光伏实时预测系统**

1. 更新公式：$\hat{z}_{t+1}(l) = \hat{z}_t(l+1) + \psi_l a_{t+1}$
2. 新数据到达 → 计算一步误差 $a_{t+1}$ → 所有 lead time 预测同时更新
3. $\psi_l$ 衰减快 → 远期预测几乎不变（AR 型过程）
4. $\psi_l$ 衰减慢 → 远期预测也受近期冲击影响（IMA 型过程）
5. **设计选择**：辐照度序列有明确日周期 → 去周期后的残差更接近 ARMA → $\psi_l$ 衰减快 → 更新公式收敛快

### 推理链 9：指数平滑 ⊂ ARIMA 的统计含义
**Ch5.4 → Ch4.3 → 预测方法论统一**

1. Holt/Brown/Winters（1950s-60s）纯经验方法，无统计模型基础
2. Box (1970) 证明：简单指数平滑 = IMA(0,1,1) 最优预测
3. Holt 线性 = IMA(0,2,2) 最优预测
4. **含义**：所有指数平滑方法都有隐含的时序模型假设
5. 如果数据不符合 IMA 结构，指数平滑就不再是最优的
6. **实践指导**：先判断数据的 ARIMA 结构，再选方法——不要盲目用指数平滑

## 三D、已解决的困惑（Ch4-5）

### 困惑 8：为什么差分而不是回归去趋势？（Ch4.1.2）
**问题**：去掉趋势可以用线性回归 $z_t = a + bt + \epsilon_t$，为什么 Box 坚持用差分？

**解答**：回归去趋势假设趋势是**确定性**的（$a + bt$），差分假设趋势是**随机性**的。Box 认为大部分实际序列的趋势会随时间改变（如股价），用确定性趋势预测会让模型被过去的趋势"锁住"，而随机趋势模型会自动适应。

更重要的是：如果真实趋势是随机的，但你用确定性回归去趋势，残差仍然非平稳——你以为去掉了趋势实际上没有。

### 困惑 9：预测区间为什么越来越宽？（Ch5.1.1）
**问题**：$V(l)$ 随 $l$ 单调递增是必然的吗？

**解答**：对于因果（causal）且可逆的 ARIMA，$\psi_j \neq 0$，所以 $V(l)$ 严格递增——**不确定性随预测 lead time 不可避免地增加**。

对平稳过程：$V(l) \to \gamma_0$（收敛到无条件方差），预测回归到均值。
对非平稳过程（$d \geq 1$）：$V(l) \to \infty$，预测区间无限展宽——**非平稳过程的长期预测本质上不可靠**。

### 推理链 10：Box-Jenkins 迭代闭环的逻辑结构
**Ch6 → Ch7 → Ch8 → 回到 Ch6**

1. Ch6 识别：ACF/PACF 模式 → 候选模型 + 初步参数
2. Ch7 估计：MLE/条件最小二乘 → 精确参数 + 标准误差
3. Ch8 诊断：Ljung-Box Q + 残差 ACF + 过度拟合 → 模型够好吗？
4. 如果不够好 → 残差 ACF 指示修改方向 → 回到 Ch6
5. **这是科学方法论的时序实现**：假设→检验→修正→再检验

### 推理链 11：过参数化与简约原则的终极理由
**Ch7.3.5 → Ch8.1 → Ch3.4**

1. ARMA(p,q) 当 $\phi_i \approx \theta_j$ 时，AR 和 MA 几乎对消
2. 似然面出现长脊 → 参数不可识别 → 标准误差爆炸
3. 过度拟合检验的过度使用也会引入伪参数
4. **简约原则不只是审美偏好，而是统计上的必要**
5. 光伏预测中同样适用：ARMA(2,2) 不一定比 AR(2) 好

## 三E、已解决的困惑（Ch6-8）

### 困惑 10：模型识别真的靠"看图"吗？（Ch6.1）
**问题**：这也太主观了吧？

**解答**：Box 明确承认识别是"inexact"和"judgment"——但这不是缺点。理由：
- 估计的 ACF/PACF 有噪声，数学上不可能精确判断截断点
- 不同模型可能产生相似的 ACF → 多个候选模型都合理
- **识别只是入口**——真正的质量保证在 Ch7 估计和 Ch8 诊断
- 现代自动化（AIC/BIC + `auto.arima()`）可以替代人工判断，但理解底层逻辑仍然必要

### 困惑 11："All models are wrong" 是否意味着建模无意义？（Ch8.1.1）
**问题**：如果没有正确模型，为什么还要这么费劲？

**解答**：Box 的完整格言是"All models are wrong, but some are useful。关键在"useful"——
- 模型不需要完美，只需要对**预测任务**足够好
- 诊断检验不是要证明模型"正确"，而是确认没有**严重缺陷**
- 如果 Ljung-Box 通过、残差 ACF 干净，模型就"可用"
- **实用主义 > 完美主义**——光伏预测也是如此

### 推理链 12：乘积季节模型 = 两尺度分离假设的数学实现
**Ch9.1.3 → Ch3.4 ARMA → Warner Ch8 物理参数化**

1. 乘积 SARIMA：$\phi(B)\Phi(B^s)\nabla^d\nabla_s^D z_t = \theta(B)\Theta(B^s)a_t$
2. "乘积"意味着：月内动态 $\phi(B), \theta(B)$ 和年际动态 $\Phi(B^s), \Theta(B^s)$ **可分离**
3. 这等价于假设：年际关系的"创新"$\alpha_t$ 本身是一个 ARMA 过程（月内相关）
4. 数学上：$(1-\theta B)(1-\Theta B^s) = 1 - \theta B - \Theta B^s + \theta\Theta B^{s+1}$——交叉项 $\theta\Theta$ 用**两个参数的乘积**代替了本来需要的独立第三参数
5. **与 NWP 的平行**：Warner Ch8 的物理参数化也做"尺度分离"——湍流（小尺度）和对流（中尺度）分别参数化，然后通过耦合项交互。乘积结构 = 参数化耦合的最简约形式

**深层洞察**：尺度分离是科学建模的通用原则——不只是时序分析的技巧。NWP 把大气运动分为 resolved（显式计算）和 subgrid（参数化），Box 把时序分为非季节（$B$）和季节（$B^s$）。两者的哲学完全一致。

### 推理链 13：GARCH = 二阶矩的 ARMA → 波动率本身是可预测的
**Ch10.1 → Ch3 ARMA → Yang Ch11 概率预测**

1. ARMA 对**均值**建模：$z_t = \phi z_{t-1} + a_t$
2. GARCH 对**方差**建模：$h_t = \omega_0 + \omega_1 a_{t-1}^2 + \beta_1 h_{t-1}$
3. 定义 $\eta_t = a_t^2 - h_t$，则 $a_t^2 = (\omega_0 + (\omega_1 + \beta_1)a_{t-1}^2 - \beta_1 \eta_{t-1}) + \eta_t$
4. **这就是 $a_t^2$ 的 ARMA 模型！** GARCH(1,1) = 冲击平方的 ARMA(1,1)
5. 含义：波动率（方差）本身有记忆结构，可以像预测均值一样预测

**与光伏预测的关键连接**：
- Yang Ch11 的概率预测需要**时变预测区间**
- Box Ch5 的固定 $\sigma_a^2$ 给出等宽区间——**对晴天太宽、对多云天太窄**
- GARCH 自适应调整 $h_t$：昨天波动大 → 今天区间宽；昨天稳定 → 今天区间窄
- **这就是为什么现代概率预测用 GARCH 或类似方法**

### 推理链 14：传递函数 = pvlib Model Chain 的统计版
**Ch11 → Yang Ch3-6 → pvlib**

1. Box 传递函数：$Y_t = v(B)X_t + N_t$（输入→系统→输出+噪声）
2. pvlib Model Chain：GHI → POA → DC → AC（辐照度→功率）
3. 两者的映射：
   - 输入 $X_t$ = GHI（全球水平辐照度）
   - 传递函数 $v(B)$ = 分解模型 + 转置模型 + 温度模型 + IV曲线
   - 输出 $Y_t$ = AC 功率
   - 噪声 $N_t$ = 建模误差（灰尘、遮蔽、衰减等未建模因素）
4. 关键区别：pvlib 是**确定性**传递函数（物理公式），Box 的是**随机**传递函数（统计估计）
5. **最佳实践**：用 pvlib 物理模型作为第一层 → 残差 $N_t$ 用 ARIMA 建模 → 合并预测

**这是物理-统计混合预测的理论基础，也是 Yang 教材 Ch7 "Post-processing" 的精确数学表达。**

### 推理链 15：干预分析 = 光伏运维的数学语言
**Ch13 → Yang Ch8 卫星 → 实际运维**

1. Box 干预模型：$Y_t = \frac{\omega(B)}{\delta(B)}B^b \xi_t + N_t$
2. 阶跃干预（$\xi_t = S_t^{(T)}$）对应：面板永久性遮蔽、逆变器降额、新增组串
3. 脉冲干预（$\xi_t = P_t^{(T)}$）对应：暂时性积雪、鸟粪、短期遮蔽
4. 渐变干预（$\delta(B)$ 有根）对应：面板衰减、灰尘累积
5. **AO 异常值** = 传感器故障导致的单点错误 → 需要检测并修正
6. **IO 异常值** = 系统性变化的起点 → 需要重新建模

**实际运维场景**：监控系统检测到功率突降 → 干预分析自动判断是"暂时遮蔽"还是"永久故障" → 触发不同的运维响应

### 推理链 16：VAR = 多站点空间-时间预测的框架
**Ch14 → Yang Ch12 层次预测 → Warner Ch8 集合预测**

1. 单变量 ARIMA：一个电站的功率预测
2. VAR(p)：$\mathbf{z}_t = \sum \Phi_i \mathbf{z}_{t-i} + \mathbf{a}_t$，$m$ 个电站同时预测
3. Granger 因果：电站 A 的过去功率是否帮助预测电站 B？ → **空间信息流方向**
4. 协整：多个电站的功率虽然各自非平稳，但有稳定的线性关系 → VECM
5. **与 NWP 的对应**：Warner Ch14 的集合预测也是多变量——多个模式的预测向量，用误差协方差矩阵 $\Sigma$ 最优组合

**三教材的大统一**：
- Box Ch14 给出数学框架（VAR/VECM）
- Warner Ch14 给出物理实现（集合 NWP 后处理）
- Yang Ch12 给出光伏应用（层次预测/空间平滑）

## 三F、已解决的困惑（Ch9-15）

### 困惑 12：季节差分和普通差分的顺序重要吗？（Ch9.1.3）
**问题**：$\nabla\nabla_{12} z_t$ 和 $\nabla_{12}\nabla z_t$ 一样吗？

**解答**：一样！因为差分算子可交换：$(1-B)(1-B^{12}) = (1-B^{12})(1-B)$。但**实践中**，先做季节差分 $\nabla_{12}$ 再做普通差分 $\nabla$ 更直观——先去掉年际变化，再去掉月际变化。ACF 的解读也更清晰。

### 困惑 13：GARCH 的 $a_t$ 到底是不是白噪声？（Ch10.1）
**问题**：GARCH 的 $a_t$ 零均值、不相关，但不独立——这算白噪声吗？

**解答**：**弱白噪声**（weak white noise）但不是**强白噪声**（strict white noise）。
- 弱白噪声：均值为零 + 不相关 + 等方差 ✓
- 强白噪声：独立同分布 ✗

GARCH 的 $a_t$ 通过**条件方差 $h_t$ 相互关联**——$a_t^2$ 是自相关的！这就是为什么 Box Ch8 的 Ljung-Box 检验对 $\hat{a}_t$ 不显著，但对 $\hat{a}_t^2$ 显著 → 暗示 ARCH 效应存在。

**检测方法**：对残差平方 $\hat{a}_t^2$ 做 Ljung-Box 检验；如果显著 → 用 GARCH。

### 困惑 14：传递函数的"预白化"为什么必要？（Ch12.2）
**问题**：直接算 $X_t$ 和 $Y_t$ 的交叉相关不行吗？

**解答**：不行。如果 $X_t$ 自身有自相关（几乎总是如此），直接的交叉相关 $r_{XY}(k)$ 会被 $X_t$ 的**自相关结构"污染"**——你看到的不是纯粹的因果响应，而是因果响应和输入自相关的混合。

预白化的妙处：先把 $X_t$ 变成白噪声 $\alpha_t$（通过拟合 ARIMA），再看 $\alpha_t$ 和 $\beta_t$（$Y_t$ 同变换）的交叉相关 → 这才是纯粹的脉冲响应估计。

**与 NWP 的对应**：Warner Ch12 的资料同化中，"观测算子增量"也是一种预白化——把复杂的观测空间映射到模型空间后再做相关分析。

### 困惑 15：Granger 因果 = 真因果吗？（Ch14.3）
**问题**：如果 X Granger 因果 Y，是否意味着 X 真的"导致" Y？

**解答**：**不是。** Granger 因果只是**预测性因果**——"X 的过去帮助预测 Y"。可能的非因果解释：
- 共同隐藏因子 $Z$ 同时驱动 $X$ 和 $Y$（混淆）
- $X$ 比 $Y$ 更早反映同一个信号（领先-滞后关系）

Box 和 Granger 都清楚这个局限性。真正的因果推断需要实验设计或因果图（Pearl 的 do-calculus）。

**光伏中的例子**：电站 A 的功率 Granger 因果电站 B → 可能只是因为云系从 A 向 B 移动（物理上是云的因果，不是功率的因果）。但对预测来说，Granger 因果就够用了——我们只关心预测准确性。

### 三大教材的统一视角

**预测的本质**：
- Box：$\hat{z}_t(l) = E_t[z_{t+l}]$（条件期望，纯统计）
- Warner：$\hat{\mathbf{x}}_{t+l} = M(\mathbf{x}_t)$（动力方程积分，纯物理）
- Yang：物理模型 + 统计后处理 = 最终预测（混合）

**不确定性的来源**：
- Box：模型结构误差（$p,d,q$ 选错）+ 参数估计误差 + 固有随机性（$\sigma_a^2$）
- Warner：初始条件误差 + 模型误差 + 边界条件误差 → 通过集合预报量化
- Yang：天气预报误差 + 辐照度转功率误差 + 运维事件 → 通过概率预测量化

**三者的融合**（未来光伏预测系统）：
1. Warner 的 NWP 提供未来天气场（物理动力）
2. Yang 的 Model Chain 将天气场转化为功率（物理传递函数）
3. Box 的 ARIMA/GARCH 修正残差并提供不确定性区间（统计修正）
