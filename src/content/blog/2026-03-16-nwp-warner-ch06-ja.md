## 1. この章が重要な理由

最初の5章は NWP の物理・数学的枠組みを構築した：方程式（Ch2）→ 数値解法（Ch3）→ パラメータ化（Ch4）→ 地表面（Ch5）。しかしどれほど優れたモデルでも、初期条件が誤っていればすべて台無しになる。

Ch6 が解く問題は：**疎で、ノイズを含み、時空間的に不均一な観測から、真の大気にできるだけ近い初期状態をどのように定義するか？**

これは単純な補間ではない——初期条件は力学的制約（近似的な地衡風バランス）を満たさなければならない。さもなければモデル起動後に偽の慣性重力波が発生し、最初の12時間の予報が使い物にならなくなる。

## 2. 核心数学：スカラーからベクトルへ

### スカラー最適推定

誤差を持つ2つの温度推定：背景値 $T_b$（分散 $\sigma_b^2$）と観測値 $T_o$（分散 $\sigma_o^2$）。最適線形結合：

$$T_a = T_b + k(T_o - T_b), \quad k = \frac{\sigma_b^2}{\sigma_b^2 + \sigma_o^2}$$

- 背景誤差が大きい → $k$ が大きい → 観測をより信頼
- 観測誤差が大きい → $k$ が小さい → 背景をより信頼

**解析精度 = 観測精度 + 背景精度**（精度 = 分散の逆数）。これは解析が常に両入力より優れていることを意味する。

### ベクトルへの一般化

$$\mathbf{x}_a = \mathbf{x}_b + \mathbf{K}(\mathbf{y} - H(\mathbf{x}_b))$$

$$\mathbf{K} = \mathbf{B}\mathbf{H}^T(\mathbf{H}\mathbf{B}\mathbf{H}^T + \mathbf{R})^{-1}$$

**B 行列**（背景誤差共分散）は最も重要な鍵——1つの観測が周囲の領域に与える影響の範囲と形状を決定する。

## 3. 同化手法の進化

| 手法 | B 行列 | 時間 | 随伴 | 特徴 |
|------|--------|------|------|------|
| SC（Cressman/Barnes）| 単純距離重み | なし | なし | 最も単純、等方性 |
| OI | 事前計算・静的 | なし | なし | B を使うが近傍観測のみ選択 |
| 3DVAR | NMC 法・静的 | なし | なし | 全観測を使用、衛星放射を同化可 |
| 4DVAR | 暗黙的に発展 | あり | あり | ECMWF で運用中 |
| EnKF | アンサンブル推定・動的 | あり | なし | レジーム依存 |
| ハイブリッド | $(1-\alpha)\mathbf{P}_f + \alpha\mathbf{B}$ | あり | なし | 最適 $\alpha \in [0.1, 0.4]$ |

**重要な認識**：SC からハイブリッド手法まで、本質的にすべて同じ問題を解いている——**B 行列をいかに現実的にするか**。静的な B は天気パターンによって変化しない（晴天と嵐に同じ B を使う）、これが 3DVAR の根本的な欠陥である。EnKF はアンサンブル推定によって B を流れに合わせて変化させる。4DVAR は時間窓を通じて同様の効果を暗黙的に実現する。

## 4. 実践上の重要課題

### スピンアップ問題

モデルは前線の傾度、対流、雲を発展させるのに時間が必要——これらは初期条件に直接書き込めない。コールドスタート vs ウォームスタート vs ホットスタート、それぞれに適した場面がある。

### スペクトル・ナッジング

従来のナッジングはモデルが既に発展させた細かいスケールの特徴を壊してしまう。スペクトル・ナッジングの巧妙さ：大スケール（例：波数 < 3 の成分）のみを修正し、細かいスケールの特徴はモデルが自由に発展させる。

### 「バナナ関数」

曲がった流れの中（前線など）では、1つの観測の影響範囲は曲がった楕円であるべき——流れに沿って伸び、流れに垂直な方向には縮む。等方性の B 行列ではこれを実現できない。

### 適応観測

台風予報では随伴法/ETKF を用いて予報に最も敏感な領域を特定し、そこへ集中的にラジオゾンデを投入する。少数の精密に配置された観測 >> 多数のランダムに分布した観測。

## 5. 太陽光発電予測への示唆

NWP 日射量予報の品質 = 初期条件の品質 × モデルの品質。

- **雲場の初期化**は短期（<6h）日射量予報の鍵——初期条件の雲の位置が誤っていれば最初の数時間は無駄になる
- 衛星放射の直接同化（温度/湿度に逆変換せずに同化）は OI に対する 3DVAR の重要な進歩
- ERA5 の日射量データ = 同化システムの産物；日射量自体は通常同化されない → 系統的バイアスが存在する
