## 1. この章がなぜ重要か

Ch4 はパラメータ化のサイクルを確立した：**放射→地表→PBL→対流→微物理→雲→放射**。

Ch5 はこのサイクルの**出発点**——地表過程に焦点を当てる。地表エネルギーバランスは、どれだけのエネルギーが大気に入るか（顕熱 H）、どれだけの水蒸気が大気に入るか（潜熱 LE）、どれだけの熱が土壌に蓄えられるか（G）を決定する。これら3つの成分の比率が境界層発達と対流開始を直接制御する。

**太陽光発電への直接的な意義**：pvlib が必要とする入力の一つはアルベドであり、アルベドは定数ではない——土壌水分・植生タイプ・太陽高度角に依存する。NWP の地表モデルが誤ったアルベドを使用すると、日射量予測に系統的なバイアスが生じる。

## 2. 地表エネルギーバランス：すべての出発点

### 基本方程式

$$R = LE + H + G \quad (5.1)$$

- $R$ = 正味放射（すべての放射フラックスの合計）
- $LE$ = 潜熱フラックス（蒸発・蒸散が消費するエネルギー）
- $H$ = 顕熱フラックス（大気を加熱するエネルギー）
- $G$ = 地中熱フラックス（土壌に伝導するエネルギー）

### 正味放射の分解

$$R = (Q + q)(1 - \alpha) - I_{\uparrow} + I_{\downarrow} \quad (5.2)$$

- $Q$ = 直達太陽放射、$q$ = 散乱太陽放射
- $\alpha$ = 地表アルベド
- $I_{\uparrow}$ = 地表が放射する長波放射（$= \varepsilon\sigma T_g^4$）
- $I_{\downarrow}$ = 大気からの下向き長波放射

**重要な認識**：この方程式は Ch4 の放射パラメータ化（$Q, q, I_{\downarrow}$ を提供）と地表パラメータ化をつなぐ。放射スキームの出力は地表モデルの入力となる。

### 水収支方程式

$$\frac{\partial\theta}{\partial t} = P - ET - RO - D \quad (5.3)$$

$\theta$ は体積含水率、$P$ は降水入力、$ET$ は蒸発散損失、$RO$ は流出、$D$ は深層浸透。

## 3. 土壌熱伝導：温度波の伝播

### 熱拡散方程式

$$\frac{\partial T_s}{\partial t} = \frac{k_s}{C_s}\frac{\partial^2 T_s}{\partial z^2} \quad (5.6)$$

熱拡散率 $\kappa = k_s/C_s$ が温度波の伝播速度を決定する：
- $k_s$ = 熱伝導率（湿潤土壌 >> 乾燥土壌 >> 空気）
- $C_s$ = 熱容量（湿潤土壌 > 乾燥土壌）

### 深さとともに減衰する温度波

$$(\Delta T)_z = (\Delta T)_0 \exp\left(-z\sqrt{\frac{\pi}{\kappa P}}\right) \quad (5.7)$$

- 日変化波は約 0.5 m まで浸透
- 年変化波は約 7 m まで浸透（$\sqrt{365} \approx 19$ 倍の深さ）

**太陽光発電への意義**：土壌熱アドミタンス（thermal admittance $\mu_s = \sqrt{k_s C_s}$）が地表温度の日変化振幅を決定する。高い熱アドミタンス（湿潤土壌）→ 地表温度変化が小さい → 顕熱フラックスが小さい → 境界層発達が弱い → 対流が起きにくい。これが灌漑地域と乾燥地域の上空で雲が大きく異なる理由である。

## 4. 蒸発散：水循環とエネルギー循環をつなぐ

### 2 種類の地表フラックス表現

顕熱：$H = \rho C_p D_H (T_g - T_a) \quad (5.14)$

潜熱：$LE = \rho L D_W (q_{sat}(T_g) - q_a) \quad (5.15)$

$D_H, D_W$ は輸送係数であり、以下に依存する：
- **地表粗度**（森林 >> 草地 >> 水面）
- **風のシア**（風速が大きいほど → 乱流が強い → 輸送が速い）
- **安定度**（不安定 → 浮力による乱流 → 輸送が増加）

### 植生蒸散の特殊性

植生は根から深層土壌の水を吸収 → 木部を通じて葉まで輸送 → 気孔から水蒸気を放出する。このプロセスは：
- 消費される潜熱は**葉面**から（土壌表面ではない）
- 土壌含水量に制御される（萎凋点以下 → 蒸散が停止）
- 気孔の開閉に制御される（乾燥・高温 → 気孔が閉じる → 蒸散が減少 → 顕熱が増大）

**太陽光発電への連鎖反応**：乾燥 → 気孔が閉じる → 蒸散↓ → 顕熱↑ → 境界層が深くなる → 対流が強まる → 午後の積雲が増加 → 午後の GHI ランプダウンが激しくなる

## 5. 陸面モデル（LSM）：エンジニアリング実装

### 階層構造

23 の LSM が PILPS（陸面パラメータ化スキーム相互比較プロジェクト）に参加した。主な発見：**複雑な LSM が単純なものより常に優れているわけではない**——複雑なモデルは正確に取得しにくい大量のパラメータを必要とするため。

### Noah LSM（WRF デフォルト）

- 4 層土壌（10 cm、30 cm、60 cm、100 cm）
- 解く方程式：5.1（エネルギーバランス）+ 5.3（水収支）+ 5.6（熱拡散）+ 5.9（水分拡散）+ 5.14/5.15（地表フラックス）
- 皮面温度 $T_g$ を反復計算で求解

### 陸面データ同化システム（LDAS）

LSM を独立して実行し、（モデル出力ではなく）観測データで駆動することで、土壌温度・水分場を「スピンアップ」する。平衡状態に達するまで数ヶ月から数年のスピンアップが必要。

**エンジニアリングの教訓**：LDAS と大気モデルは**同じ LSM** を使用しなければならない。異なる LSM 間での土壌状態の変換はほぼ不可能——異なる前提、異なるパラメータ化、異なるグリッド。

## 6. その他の地表過程

### 海洋過程
- 短期予報（1〜2 週間未満）：SST は一定と仮定できる
- 例外：ハリケーン——強風が混合して SST を急降下させる；海洋モデルの結合が必要
- 長期・気候予測：海洋循環 + 海氷モデルの結合が必要

### 地形強制
- 平滑化された地形は山岳高度を低下させる → 山岳遮断と重力波抵抗を過小評価
- エンベロープ地形（envelope topography）：「グリッド平均高度」ではなく「山頂高度」を使用
- 解像度の影響は劇的：30 km グリッドと 3.3 km グリッドの地形差は一目瞭然

### 都市キャノピーモデル（UCM）
- 単純な方法：LSM パラメータを調整（粗度↑、アルベド↓、熱容量↑、透水面積↓）
- 複雑な方法：UCM が建築幾何形状をパラメータ化（街路峡谷での放射トラップ、影響効果）
- 都市ヒートアイランドが太陽光発電に与える影響：都市上空の境界層がより深い → 対流が強まる → 異なる雲パターン

## 8. Ch4 の仮説の検証

Ch4 で私が提起した仮説 1：**「地表パラメータ化の誤差寄与はどれほどか？」**

Warner の回答（部分的な検証）：
- 23 の LSM を比較した PILPS の結果、**複雑なモデルが必ずしも優れているわけではない**——パラメータの不確実性がモデルの複雑さよりも重要である可能性がある
- 土壌含水量が熱伝導率に与える影響は**2 桁のオーダー**（乾燥土壌 vs 湿潤土壌）——含水量誤差が増幅される
- LDAS には数ヶ月のスピンアップが必要であり、土壌状態の初期誤差の**減衰が非常に遅い**ことを示している

結論：地表は確かに過小評価された誤差源だが、雲ほど致命的ではない。地表過程のタイムスケール（時間〜日）は雲（分〜時間）よりもはるかに遅いため、誤差の伝播が遅い。
