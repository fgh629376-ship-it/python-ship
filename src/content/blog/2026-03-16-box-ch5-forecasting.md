# Box 时间序列分析 Ch5：最优预测——条件期望与预测函数

> **教材**：Box, Jenkins & Reinsel — *Time Series Analysis*, 4th Ed, Chapter 5

## 章节定位

Ch3 建模型，Ch4 处理非平稳，Ch5 回答核心问题：**已知模型，如何做最优预测？**

---

## 5.1 最小 MSE 预测 = 条件期望

### 基本结论

设我们在时刻 $t$ 预测 $z_{t+l}$，最小均方误差（MSE）预测为：

$$\hat{z}_t(l) = E_t[z_{t+l}] = E[z_{t+l} | z_t, z_{t-1}, \ldots]$$

**条件期望就是最优线性预测。** 这是 Wold/Kolmogorov/Wiener 预测理论的核心结果。

### 预测误差

$$e_t(l) = z_{t+l} - \hat{z}_t(l) = a_{t+l} + \psi_1 a_{t+l-1} + \cdots + \psi_{l-1} a_{t+1}$$

- **无偏**：$E_t[e_t(l)] = 0$
- **方差**：$V(l) = (1 + \psi_1^2 + \psi_2^2 + \cdots + \psi_{l-1}^2)\sigma_a^2$
- **概率区间**：$\hat{z}_t(l) \pm z_{\alpha/2}\sqrt{V(l)}$（假设 $a_t$ 正态）

### 重要推论

**一步预测误差 = 冲击**：$e_t(1) = a_{t+1}$。如果一步预测误差相关，说明预测不是最优的——可以进一步改进。

---

## 5.2 三种预测形式

### 形式 1：差分方程（实际计算最方便）

$$\hat{z}_t(l) = \varphi_1[z_{t+l-1}] + \cdots + \varphi_{p+d}[z_{t+l-p-d}] - \theta_1[a_{t+l-1}] - \cdots - \theta_q[a_{t+l-q}]$$

操作规则：
- 已知的 $z_{t-j}$ 不变
- 未来的 $z_{t+j}$ 替换为 $\hat{z}_t(j)$
- 已知的 $a_{t-j}$ 由 $z_{t-j} - \hat{z}_{t-j-1}(1)$ 计算
- 未来的 $a_{t+j}$ 替换为 0

### 形式 2：ψ 权重形式（计算预测误差方差）

$$\hat{z}_t(l) = \psi_l a_t + \psi_{l+1} a_{t-1} + \psi_{l+2} a_{t-2} + \cdots$$

### 形式 3：π 权重形式（加权移动平均形式）

$$\hat{z}_t(l) = \sum_{j=1}^{\infty} \pi_j [z_{t+l-j}]$$

当 $d \geq 1$ 时，$\sum \pi_j = 1$，预测是过去观测值的**加权平均**。

---

## 5.2.3 预测更新公式

$$\hat{z}_{t+1}(l) = \hat{z}_t(l+1) + \psi_l a_{t+1}$$

**这是实时预测系统的核心**——当新数据 $z_{t+1}$ 到达时：
1. 计算一步预测误差 $a_{t+1} = z_{t+1} - \hat{z}_t(1)$
2. 所有预测按 $\psi_l$ 比例更新

不需要重新计算所有预测，只需一次乘加。

---

## 5.3 预测函数的本质

### AR 算子决定预测函数的形式

对于 $l > q - p - d$，预测满足齐次差分方程：

$$\hat{z}_t(l) - \varphi_1 \hat{z}_t(l-1) - \cdots - \varphi_{p+d} \hat{z}_t(l-p-d) = 0$$

通解：

$$\hat{z}_t(l) = b_0^{(t)} + b_1^{(t)} l + \cdots + b_{d-1}^{(t)} l^{d-1} + \text{（衰减项）}$$

- **$\nabla^d$ 部分** → $d-1$ 阶多项式（主导长期趋势）
- **$\phi(B)$ 部分** → 衰减指数/正弦（过渡项，最终消失）

| 模型 | 最终预测函数 |
|------|-------------|
| IMA(0,1,1) | 常数（水平） |
| IMA(0,2,2) | 直线（水平 + 斜率） |
| ARIMA(1,1,0) | 水平 + 衰减指数 |
| ARIMA(1,1,1) | 水平 + 衰减指数 |

### MA 算子决定"如何拟合"

MA 参数影响预测函数如何"锚定"到数据上——即 $b_0^{(t)}, b_1^{(t)}, \ldots$ 的计算方式。

---

## 5.4 IMA 过程的预测 = 指数平滑

### IMA(0,1,1)

$$\hat{z}_t(l) = (1 - \theta_1) \sum_{j=0}^{\infty} \theta_1^j z_{t-j}$$

**这就是简单指数平滑（EWMA）！** $\alpha = 1 - \theta_1$ 是平滑参数。

### IMA(0,2,2)

预测沿直线：$\hat{z}_t(l) = b_0^{(t)} + b_1^{(t)} l$

水平 $b_0^{(t)}$ 和斜率 $b_1^{(t)}$ 持续更新。**这就是 Holt 线性趋势方法。**

### ARIMA(1,1,1)

预测 = $z_t$ 和 EWMA $\bar{z}_{t-1}(\theta)$ 的**线性插值**，权重随 lead time 变化。

---

## 5.5 状态空间表示

ARIMA 可以写成状态空间形式：$\mathbf{Y}_t = \mathbf{F}\mathbf{Y}_{t-1} + \boldsymbol{\psi} a_t$

这连接了 Box-Jenkins 框架与 Kalman 滤波器。状态空间形式：
- 处理**有限样本精确预测**
- 自然扩展到**缺失数据**和**多变量**
- 与 Kalman 滤波文献（航天工程、导航）互通

---

## 深度思考

### 1. "最优预测 = 条件期望"的深层含义

这不只是一个数学定理。它说明了：
- 预测的质量完全由**模型的准确性**决定
- 给定正确模型，不可能找到比条件期望更好的线性预测
- 预测的不确定性（$V(l)$）是**模型结构的固有性质**，不是方法论的缺陷

### 2. ψ 权重 = 预测的"记忆结构"

$V(l)$ 随 $l$ 增加而增加，增速取决于 $\psi_j$ 的衰减模式：
- AR 模型：$\psi_j$ 指数衰减 → $V(l)$ 快速趋于 $\gamma_0$（无条件方差）
- IMA(0,1,1)：$\psi_j \equiv 1-\theta_1$ → $V(l)$ 线性增长 → 非平稳！
- IMA(0,2,2)：$\psi_j \propto j$ → $V(l)$ 二次增长 → 更大的不确定性

### 3. 对光伏预测的直接指导

**更新公式是实时系统的关键**：
- 每当新的辐照度/功率数据到达，更新所有预测
- $\psi_l$ 衰减快 → 远期预测几乎不受近期冲击影响
- $\psi_l$ 衰减慢 → 近期冲击对远期预测影响大（如持续云遮蔽）

**概率区间 → 功率预测的置信区间**：
- 电网调度需要的不仅是点预测，还有上下界
- $\hat{z}_t(l) \pm z_{\alpha/2}\sqrt{V(l)}$ 直接给出

## 小结

Ch5 完成了"建模 → 预测"的闭环：

$$\boxed{\text{最优预测} = E_t[z_{t+l}], \quad \text{预测误差方差} = \sum_{j=0}^{l-1} \psi_j^2 \sigma_a^2}$$

- **最优预测**：$\hat{z}_t(l) = E_t[z_{t+l}]$
- **预测更新**：$\hat{z}_{t+1}(l) = \hat{z}_t(l+1) + \psi_l a_{t+1}$
- **预测区间**：$\hat{z}_t(l) \pm z_{\alpha/2}\sqrt{V(l)}$
- **IMA(0,1,1)**：= 简单指数平滑
- **IMA(0,2,2)**：= Holt 线性趋势
