# MIT 18.06 線形代数 Part 4：PCA・フーリエ変換・循環行列

> **コース**：MIT 18.06 Linear Algebra, Spring 2025
> **範囲**：講義28-36（PCA・複素空間・DFT・循環行列）

## 講義28：PCA——データサイエンスの第一ツール

### PCAとは何か？

**主成分分析（PCA）= データ行列にSVDを適用し、前 $k$ 個の右特異ベクトルを新しい座標軸とする。**

$X$ を $n \times p$ のデータ行列（$n$ 個のサンプル、$p$ 個の特徴量、中心化・標準化済み）とする：

$$X = U\Sigma V^T$$

- **主成分方向**：$V$ の列（$\mathbb{R}^p$ 内の方向）
- **主成分スコア**：$U\Sigma$ の列（各サンプルが主成分上に射影された値）
- **分散説明率**：$\sigma_i^2 / \sum \sigma_j^2$

### 2次元PCA

データの変動が最大になる2次元射影を選ぶ：$V$ の前2列 $\mathbf{u}, \mathbf{v}$ を取り、散布図 $(\mathbf{x}_i \cdot \mathbf{u}, \mathbf{x}_i \cdot \mathbf{v})$ を描く。

### PCA ≠ 最小二乗法

- **最小二乗法**：$y$ 方向の残差二乗和を最小化する（$x$ 軸に対して垂直）
- **PCA**：部分空間への**直交距離**の二乗和を最小化する（「垂直最小二乗法」）

**太陽光予測におけるPCA**：
- **気象特徴の次元削減**：温度/湿度/気圧/風速は高い相関を持つ → PCAで3〜5次元に削減し、分散の95%を保持
- **多地点発電パターン**：$n$ 箇所の発電所の発電量行列にPCA → 「晴天モード」「過雲モード」「曇天モード」を抽出
- **NWPアンサンブル解析**：Warner 第14章のEOF解析 = NWPアンサンブル予測行列にPCAを適用

## 講義29-30：複素ベクトル空間

### 実数から複素数へ

- **共役転置** $\bar{A}^T$（Hermitian転置）が $A^T$ の代わりになる
- **内積** $\langle \mathbf{v}, \mathbf{w} \rangle = \bar{\mathbf{v}}^T \mathbf{w}$ → $\|\mathbf{v}\|^2 = \langle \mathbf{v}, \mathbf{v} \rangle \geq 0$ が保証される

### 三種類の特殊な複素行列

- **ユニタリ行列**：$U\bar{U}^T = I$ → 長さを保つ変換（直交行列の一般化）
- **Hermitian行列**：$A = \bar{A}^T$ → 固有値がすべて実数（対称行列の一般化）
- **正規行列**：$A\bar{A}^T = \bar{A}^T A$ → 直交固有基底を持つ

### スペクトル定理の一般化

LaTeXを表のセルに入れないよう、箇条書き形式でまとめる：

- **実対称行列**：$A = EDE^T$、$E$ は直交行列、$D$ は実対角行列
- **Hermitian行列**：$A = UDU^*$、$U$ はユニタリ行列、$D$ は実対角行列
- **正規行列**：$A = UDU^*$、$U$ はユニタリ行列、$D$ は複素対角行列

## 講義31-33：循環行列とフーリエ基底

### 循環行列

巡回置換行列 $P$（$1 \to 2 \to 3 \to \cdots \to n \to 1$）の固有値 = $n$ 乗の単位根：

$$\omega_k = e^{2\pi i k/n}, \quad k = 0, 1, \ldots, n-1$$

**循環行列** = $P$ の冪の線形結合 = $c_0 I + c_1 P + c_2 P^2 + \cdots$

### フーリエ基底

$P$ の固有ベクトルが**フーリエ行列** $F$ を構成する——これが**離散フーリエ変換（DFT）の行列**である！

$$F_{jk} = \frac{1}{\sqrt{n}} \omega_n^{jk}$$

**核心的な性質**：フーリエ基底は**すべての循環行列の共通固有基底**である。

### 巡回畳み込み

2つのベクトルの巡回畳み込み = 対応する循環行列の積。

**重要定理**：DFTは畳み込みを成分ごとの積に変換する。

$$\text{DFT}(\mathbf{a} * \mathbf{b}) = \text{DFT}(\mathbf{a}) \odot \text{DFT}(\mathbf{b})$$

## 講義34-36：DFTの応用

### DFT = 基底変換

標準基底からフーリエ基底への座標変換：

$$\hat{\mathbf{f}} = F^* \mathbf{f}$$

逆変換：$\mathbf{f} = F\hat{\mathbf{f}}$

### 応用

- **信号圧縮**：DFT後に大きな係数だけを保持 → JPEG/MP3の数学的基盤
- **デノイズ**：DFT → 高周波成分を除去 → 逆DFT
- **畳み込みの高速化**：直接畳み込みは $O(n^2)$、DFT後の成分ごとの積は $O(n \log n)$（FFT）

### Box 時系列との深い関連

Box 第2章のパワースペクトル = 自己共分散関数のDFT：

$$p(f) = 2\left(\gamma_0 + 2\sum_{k=1}^{\infty} \gamma_k \cos 2\pi f k\right)$$

これは**まさに**フーリエ変換である！ACF $\gamma_k$ とスペクトル $p(f)$ はフーリエ双対の関係にある。

**線形フィルタの周波数領域での見方**：

ARMAモデル $\phi(B)\tilde{z}_t = \theta(B)a_t$ のスペクトル密度：

$$g(f) = \sigma_a^2 \frac{|\theta(e^{-2\pi if})|^2}{|\phi(e^{-2\pi if})|^2}$$

MA → 分子（零点）→ 特定の周波数でエネルギーを抑制
AR → 分母（極）→ 特定の周波数でエネルギーを増幅

**AR(2) のスペクトルピーク**（太陽黒点の約11年周期）= 複素固有値の偏角に対応する周波数。これは循環行列の固有値構造と完全に一致する！

### 太陽光予測におけるDFT

- **日周期の抽出**：発電量系列のDFT → 周波数 $1/24\text{h}$ のピーク = 日変化；$1/8760\text{h}$ のピーク = 年変化
- **デノイズ**：高周波ノイズ（センサーのジッタ）を低域通過フィルタで除去
- **特徴量エンジニアリング**：DFT係数をMLモデルの周波数領域特徴量として使用
- **SARIMAの周波数領域解釈**：季節差分 $\nabla_{24}$ は周波数 $k/24$ に零点を置くことと等価

## コース全体の大統一：線形代数の知識ネットワーク

```
ベクトル空間 → 基底 → 次元 → 次元定理
    ↓                              ↓
線形変換 → 行列 → 四つの部分空間 → 直交補空間
    ↓          ↓                   ↓
固有値 → 対角化 → スペクトル定理 → 正定値性
    ↓                               ↓
SVD ← 特異値 = √(M^TM の固有値)
 ↓         ↓                ↓
PCA    低ランク近似    擬似逆行列/最小二乗法
 ↓                           ↓
DFT  =  循環行列の固有値分解  =  Box のパワースペクトル
```

**MIT 18.06 全課程のコアメッセージ**：

> 線形代数は「連立方程式を解くためのツール」にとどまらない——それは**データの構造を理解するための言語**である。
> 四つの基本部分空間は「何が説明できるか」を教えてくれる；
> SVDは「どの順番で重要か」を教えてくれる；
> フーリエ変換は「どの周波数が支配的か」を教えてくれる。

## 太陽光発電予測プロジェクトへの行動指針

MIT 18.06 全課程の学習に基づき、将来の太陽光発電予測フレームワークへの具体的な提言：

1. **データ探索の第一歩**：気象特徴行列にSVDを適用し、階数と条件数を確認する
2. **特徴選択**：PCАで次元削減、または $V$ 行列でどの元の特徴が大きく寄与しているかを確認する
3. **多地点モデリング**：発電量行列の低ランク分解 → 空間パターンの抽出
4. **周波数領域分析**：DFTで周期的構造を確認 → SARIMAの $s$ を決定する
5. **数値安定性**：回帰にはQR分解を使用し、正規方程式を直接解かない
6. **正則化の判断**：条件数 $\kappa > 100$ → Ridge/Lassoが必要
7. **残差診断**：残差共分散行列の固有値分布 → モデルの十分性を検証する
