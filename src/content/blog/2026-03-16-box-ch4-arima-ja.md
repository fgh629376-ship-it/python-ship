# Box 時系列分析 Ch4：ARIMA——差分が非定常を定常に変える

> **教科書**：Box, Jenkins & Reinsel — *Time Series Analysis*, 4th Ed, Chapter 4

## 核心的アイデア

Ch3 の ARMA は定常系列のみを扱う。しかし実際の多くの系列（株価・売上・気温）は**固定した平均をもたない**。Ch4 の核心的な洞察：

> **$d$ 回の差分により「同次的非定常」系列を定常系列に変換できる。**

$$\phi(B)\nabla^d z_t = \theta(B)a_t$$

これが ARIMA(p,d,q) モデルであり、$\nabla = 1 - B$ が差分演算子である。

---

## 4.1 ARIMA 過程の定義

### なぜ「単位円内に根をもつ」非定常ではないのか？

Box はまず $\phi(B) = 0$ の根が単位円の**内側**にある場合を除外する——これは**爆発的な過程**（例：$z_t = 2z_{t-1} + a_t$、系列が指数的に爆発する）を生成するからだ。これは我々がモデル化したい種類ではない。

我々が求める非定常は**同次的（homogeneous）**なもの——固定された水準はないが、局所的な挙動はどこも似ている。数学的には、$\phi(B) = 0$ が単位円上に**ちょうど** $d$ 個の根（= 1）をもつ場合に対応する。

### ARIMA の二つの理解

1. **差分の視点**：$w_t = \nabla^d z_t$ は定常 ARMA 過程であり、$z_t$ は $w_t$ の $d$ 回累積和（「積分」）
2. **同次性の視点**：
   - $d = 1$：$z_t$ の水準は自由に漂流できるが、$\nabla z_t$ は定常 → **確率的水準**
   - $d = 2$：$z_t$ の水準と傾きが両方自由に変化できるが、$\nabla^2 z_t$ は定常 → **確率的水準 + 確率的トレンド**

### 定数項 $\theta_0$ の意味

$$\phi(B)\nabla^d z_t = \theta_0 + \theta(B)a_t$$

$d \geq 1$ のとき、$\theta_0 \neq 0$ は**決定論的トレンド**を意味する：
- $d = 1, \theta_0 \neq 0$：$z_t$ は傾き $\mu_w = \theta_0/(1 - \phi_1 - \cdots - \phi_p)$ の線形決定論的トレンドをもつ

**Box の推奨**：データや物理的理由が明確に支持しない限り、$\theta_0 = 0$ を仮定する（確率的トレンドの方が決定論的トレンドより柔軟）。

---

## 4.2 三つの明示的形式

### 形式 1：差分方程式形式

$$z_t = \varphi_1 z_{t-1} + \cdots + \varphi_{p+d} z_{t-p-d} + a_t - \theta_1 a_{t-1} - \cdots - \theta_q a_{t-q}$$

ここで $\varphi(B) = \phi(B)(1-B)^d$。**予測計算時に最も便利。**

### 形式 2：確率的ショック形式

$$z_t = a_t + \psi_1 a_{t-1} + \psi_2 a_{t-2} + \cdots$$

ψ 重みは $\varphi(B)\psi(B) = \theta(B)$ から逐次的に解く。**予測誤差分散の計算に必要。**

### 形式 3：反転形式（π 重み）

$$z_t = \pi_1 z_{t-1} + \pi_2 z_{t-2} + \cdots + a_t$$

π 重みは $\theta(B)\pi(B) = \varphi(B)$ から逐次的に解く。**無限 AR として表現する形式であり、指数加重移動平均（EWMA）はその特殊ケース。**

---

## 4.3 IMA 過程——指数平滑の理論的基盤

### IMA(0,1,1)

$$\nabla z_t = (1 - \theta_1 B)a_t$$

反転形式：$z_t = (1-\theta_1) z_{t-1} + \theta_1(1-\theta_1) z_{t-2} + \cdots + a_t$

$\lambda = 1 - \theta_1$ とおくと、これはまさに**指数加重移動平均（EWMA）**である！重みは $\theta_1^j$ の速度で指数的に減衰する。

**歴史的意義**：Holt/Brown/Winters の指数平滑法は Box-Jenkins 以前から広く使用されていた。Box はこれらの方法が**IMA(0,1,1) の最適予測とまさに一致する**ことを証明し、経験的手法に厳密な統計的基盤を与えた。

### IMA(0,2,2)

$$\nabla^2 z_t = (1 - \theta_1 B - \theta_2 B^2)a_t$$

最適予測は**直線**に沿って変化する（水準 + 傾きが継続的に更新される）。**Holt の二パラメータ指数平滑**に対応する。

## 深い考察

### 1. 差分 vs 去トレンド

- **差分 $\nabla^d$**：$d$ 次多項式トレンドを除去；トレンドは確率的と仮定；ほとんどの経済・産業系列に適用
- **回帰去トレンド**：$z_t = f(t) + \epsilon_t$；トレンドは決定論的と仮定；物理的トレンドをもつ系列に適用
- **物理モデル去トレンド**：$z_t - \text{clearsky}_t$；物理モデルが既知と仮定；太陽光発電の日射量に適用

**太陽光発電について**：まず快晴モデルで物理的に去トレンド（Yang Ch7）し、残差がまだ非定常なら一度差分をとる。

### 2. d の実際の選択

Box は繰り返し強調する：$d$ は通常 0、1、高くても 2。
- $d = 0$：定常系列
- $d = 1$：水準漂流（最も一般的）
- $d = 2$：トレンド漂流（より稀）
- $d \geq 3$：ほぼ不要

### 3. 単位根と非定常の現代的検定

Box の時代は主に ACF で差分が必要かを判断していた（ACF の緩やかな減衰 → 差分が必要かも）。現代的手法としては：
- **ADF 検定**（Augmented Dickey-Fuller）
- **KPSS 検定**
- **Phillips-Perron 検定**

これらは Ch6（モデル同定）で間接的に扱われる。

### 4. 過剰差分の危険性

元々定常な系列を $d$ 回差分すると、**不要な MA 単位根が導入され**、パラメータ推定が不安定になる。Box の助言：差分は少なめに抑え、ACF/PACF で判断する。

---

## まとめ

Ch4 = Ch3 の非定常への拡張。核心的な式は一つだけ：

$$\phi(B)(1-B)^d z_t = \theta_0 + \theta(B)a_t$$

しかしその力は絶大——非季節性の時系列のほぼすべてを $p, d, q \leq 2$ の ARIMA で記述できる。
