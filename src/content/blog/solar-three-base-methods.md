# NWP vs 卫星 vs 地面数据：光伏预测三大基础方法的物理本质

> 📖 本文基于教材 *Solar Irradiance and Photovoltaic Power Forecasting*（Yang & Kleissl, 2024）第七章，结合工程实践视角的深度解读。

光伏功率预测的"基础方法"是什么？就是那些**生成原始预测**的方法——还没经过后处理（Post-processing）优化的第一手结果。它们是整个预测流水线的地基。

三大基础方法按重要性排序：

| 方法 | 有效预测范围 | 核心物理 | 不可替代性 |
|------|-------------|---------|-----------|
| **NWP** | 日前 ~ 数日 | 求解大气动力方程 | ⭐⭐⭐ 唯一选择 |
| **卫星** | 15min ~ 4h | 云运动矢量平流 | ⭐⭐ 日内关键补充 |
| **地面数据** | 1min ~ 1h | 时序统计/空间相关 | ⭐ 超短期精修 |

---

## 一、NWP：唯一能"求解"大气的方法

### 1.1 七个方程，五条物理定律

数值天气预报的核心是**原始方程组（Primitive Equations）**——7 个方程求解 7 个变量（u, v, w, T, q, ρ, p）：

① 牛顿第二定律 → 3个动量方程（预测三维风速）

$$\frac{Du}{Dt} = -\frac{\nabla p}{\rho} + f \times u + F_{\text{friction}}$$

② 热力学第一定律 → 温度预测（$\theta$ = 位温，$Q$ = 加热率）

$$\frac{D\theta}{Dt} = \frac{\theta}{T} \cdot \frac{Q}{c_p}$$

③ 水汽守恒 → 水汽预测（$E$ = 蒸发，$C$ = 凝结）

$$\frac{Dq}{Dt} = E - C$$

④ 质量守恒 → 连续性方程

⑤ 理想气体状态方程 → 诊断关系（不含时间导数）

$$p = \rho R T$$

前 6 个是**预报方程**（含 D/Dt，即拉格朗日全导数），第 7 个是**诊断方程**。

> 💡 **关键洞察**：所有大气运动——从微风到台风——都遵循这 7 个方程。NWP 的本质就是在超级计算机上数值求解它们。

### 1.2 参数化：NWP 的阿喀琉斯之踵

原始方程只能描述网格尺度（>10km）的过程。更小尺度的**次网格过程**必须通过参数化方案"近似猜测"。

六种关键参数化，按对光伏预测的影响排序：

| 参数化方案 | 作用 | 对辐照度的影响 |
|-----------|------|---------------|
| **云微物理** | 凝结、碰并、降水过程 | ⭐⭐⭐ 决定云量——最大影响 |
| **对流** | 次网格垂直热量/水汽输送 | ⭐⭐⭐ 积云生消直接影响 GHI |
| **PBL** | 边界层垂直混合 | ⭐⭐ 影响低层水汽和云底高度 |
| **地表-大气** | 地表能量平衡 | ⭐⭐ 土壤湿度→蒸发→成云 |
| **辐射** | 辐射传输计算 | ⭐ 物理成熟，不是瓶颈 |
| **气溶胶** | 大气化学 | ⭐ 只在极端事件时重要 |

> ⚠️ **反直觉结论**：辐射参数化方案看似最相关，实际不是 NWP 辐照度预测的瓶颈。辐射传输物理已经很成熟——它只是把云水含量转换为地面辐照度。真正的瓶颈是**云微物理参数化**：云在微米尺度形成（CCN + 过饱和 → 凝结 → 碰并 → 降水），而 NWP 网格是千米级。这个尺度鸿沟是 NWP 辐照度预测误差的主要来源。

### 1.3 离散化与数据同化

**离散化**：把连续 PDE 变成计算机可解的代数方程。

- **有限差分法**：Du/Dt ≈ Δu/Δt，交错网格提高精度
- **谱方法**：球谐函数分解（ECMWF IFS 用 Tco1279 截断 ≈ 31.4km 分辨率）
- **CFL 条件**：Δt ≤ Δx/c，10km 网格 → 时间步长 < 100 秒

**数据同化（4D-Var）**：用观测修正模型预测的初始条件。

核心哲学：NWP 预测满足物理定律（作为先验），观测提供修正（作为证据），4D-Var 在时间窗内找"最优折衷"——不偏离物理太远，同时尊重观测。

## 二、卫星预测：4 小时的窗口

### 2.1 从卫星图像到辐照度

三类反演方法：

**① 物理方法（FARMS/NSRDB）**：
```python
# 核心公式
Gh = η * Gho + (1 - η) * Ghc  # η=云分比

# Ghc: REST2 晴空模型
# Gho: Beer-Lambert 定律 + 参数化云透射率
# 输入: GOES云属性 + MERRA-2大气 + MODIS气溶胶/反照率
```
FARMS 通过 LUT 预计算实现 1000 倍加速。

**② 半经验方法（Heliosat-2/SolarGIS/SolarAnywhere）**：
```python
# 核心公式——简单到令人不安
Gh = g(ν) * Ghc  # ν=云指数，g()=映射函数

# Heliosat-2 的 g(ν)：
# ν < -0.2  → κ = 1.2
# -0.2 ~ 0.8 → κ = 1 - ν（线性）
# 0.8 ~ 1.1  → 二次多项式
# ν > 1.1   → κ = 0.05
```
只用可见光单通道，分辨率可达 0.5km。商业太阳能资源评估的主力。

**③ 纯 ML 方法**：Random Forest/CNN 直接从卫星通道回归 GHI。FY-4A 案例：11 个特征（4 通道 + 太阳角 + 地理）→ 小时 RMSE ≈ 147 W/m²。精度不如前两类，但对缺乏成熟云属性产品的新卫星有价值。

### 2.2 云运动矢量（CMV）：预测的核心

卫星预测的本质：提取当前云运动，假设短期内云"冻结"（形态不变只做平移），将辐照度场向前推进。

**光流法（Lucas-Kanade）**：
亮度恒定约束：$I(x,y,t) = I(x+\Delta x, y+\Delta y, t+\Delta t)$

Taylor 展开得光流方程（一个方程两个未知数）：

$$I_x U + I_y V = -I_t$$

→ 邻域像素联立 → 最小二乘求解 $(U, V)$

**块匹配法（三步搜索）**：在连续帧中找最大互相关的像素块，块中心位移即 CMV。

> ⚠️ **4 小时极限**：所有 CMV 方法的共同假设——云场"冻结"——在 15-30 分钟后开始失效（云变形/消散/生成），4 小时后完全失效。卫星方法**不能生成新云**，只有 NWP 能模拟云的完整生命周期。

## 三、地面数据预测：时间维度的极限

### 3.1 参考预测（Baseline）

任何模型都必须打败这些 baseline：

```python
# 持续性（Persistence）
$\hat{\kappa}(t+h) = \kappa(t)$

# 智能持续性（Smart Persistence）
GHI_hat(t+h) = κ(t) × Ghc(t+h)  # κ持续，晴空值更新

# 气候学（Climatology）
$\hat{\kappa}(t+h) = \bar{\kappa}$

# CLIPER（最优凸组合）
$\hat{\kappa}(t+h) = \gamma_h \cdot \kappa(t) + (1 - \gamma_h) \cdot \bar{\kappa}$
# γ_h = autocorrelation at lag h
```

CLIPER 是最强的简单 baseline：当自相关高（短期），权重偏向持续性；当自相关低（长期），权重偏向气候学。

### 3.2 单站方法的信息论极限

无论用多复杂的模型（LSTM、Transformer），单站方法都受限于一个基本事实：

> **当预测范围 h 增大，κ(t) 和 κ(t+h) 的自相关 γ_h 趋近零。** 历史信息对未来没有预测力时，任何模型都无法从"没有信息的输入"中提取信息。

这就是为什么 > 4h 后单站方法退化为气候学。

### 3.3 时空方法：有限的改进

利用多站点的空间相关性：

- **STAR（时空自回归）**：用附近站点的滞后值加权预测
- **Kriging（克里金）**：地统计学，可预测网络外位置，但 O(n³) 计算
- **GNN（图神经网络）**：将站点网络建模为图，自然融合时空信息

但空间相关性同样随距离和时间快速衰减。 > 4h 后只剩 NWP 能提供新信息。

## 四、工程选择策略：三者融合

```
最优预测流水线:

超短期 (0-30min)  → 地面数据（实时修正）
日内    (30min-4h) → 卫星 CMV（捕捉云运动）
日前    (4h-3天)   → NWP（唯一选择）
多日    (3天+)     → NWP 集成预报

所有范围都需要:
  → 晴空模型归一化（κ = GHI/Ghc，去除太阳位置的确定性变化）
  → 后处理（MOS/ML，修正系统偏差）
  → 概率化（分位数回归/集成方法）
```

这不是"选哪个"的问题，而是"在哪个时间范围用哪个"的问题。三种方法互补，各守一段。

## 五、关键公式速查

| 公式 | 含义 | 来源 |
|------|------|------|
| `Du/Dt = -∇p/ρ + f×u + F` | NWP 动量方程 | 原始方程 |
| `Δt ≤ Δx/c` | CFL 稳定性条件 | 有限差分 |
| `Gh = η·Gho + (1-η)·Ghc` | FARMS 辐照度反演 | 物理方法 |
| `Gh = g(ν)·Ghc` | 半经验反演 | Heliosat-2 |
| `Ix·U + Iy·V = -It` | 光流方程 | Lucas-Kanade |
| `κ̂ = γ_h·κ(t) + (1-γ_h)·μ_κ` | CLIPER baseline | 最优凸组合 |

## 六、给光伏预测工程师的建议

1. **不要跳过 NWP**：它是日前预测的唯一信息来源。即使你只做日内预测，NWP 的云类型信息也能显著提升卫星 CMV 的精度。

2. **别高估 ML**：再强的 LSTM 也无法突破信息论极限。单站 > 4h，自相关归零，模型容量浪费在拟合噪声上。

3. **晴空归一化是免费午餐**：用 κ 而非 GHI 建模，等于免费获得太阳位置的物理知识，让模型专注于云的不确定性。

4. **卫星数据正在变好**：新一代静止卫星（FY-4A/Himawari-8/GOES-16）的时空分辨率大幅提升，5-10 分钟全盘扫描已成标配。

5. **最优策略是融合**：NWP 定框架 → 卫星填日内 → 地面精修超短期 → 后处理统一校正。
