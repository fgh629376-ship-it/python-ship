# MIT 18.06 线性代数 Part 1：向量空间的四大支柱

> **课程**：MIT 18.06 Linear Algebra, Spring 2025 — Prof. Nike Sun
> **教材**：Gilbert Strang, *Introduction to Linear Algebra*, 6th Ed.
> **范围**：Lecture 1-7（向量空间基础）

## 为什么学线性代数？

线性代数是**所有数据科学、机器学习、信号处理的数学语言**。对光伏预测来说：

- **最小二乘拟合**（Lec 12）= pvlib 温度模型的参数估计
- **SVD/PCA**（Lec 24-28）= 气象特征降维、多站点数据压缩
- **特征值分解**（Lec 20-23）= Box ARIMA 的特征方程分析
- **矩阵运算**= 深度学习前向/反向传播的本质

Strang 的名言："**线性代数的核心是四大子空间**。"

## Lec 1-2：向量与矩阵运算

### 线性组合——一切的起点

$\mathbb{R}^n$ 中的向量可以做两件事：**加法**和**标量乘法**。两者组合就是**线性组合**：

$$c_1\mathbf{v}_1 + c_2\mathbf{v}_2 + \cdots + c_k\mathbf{v}_k$$

一组向量的**张成（span）**= 所有可能的线性组合构成的集合。

### 矩阵乘向量 = 列的线性组合

$$A\mathbf{x} = x_1\mathbf{a}_1 + x_2\mathbf{a}_2 + \cdots + x_n\mathbf{a}_n$$

**这是理解矩阵最重要的视角**：$A\mathbf{x}$ 不是"行×列"的机械计算，而是 $A$ 的列向量的线性组合，系数由 $\mathbf{x}$ 给出。

### 高斯消元 = 矩阵乘法

消元的每一步都可以写成左乘一个初等矩阵 $G_i$：

$$G_k \cdots G_2 G_1 A\mathbf{x} = G_k \cdots G_2 G_1 \mathbf{b}$$

**这不只是计算技巧——它揭示了矩阵分解的本质**。LU 分解就是把所有 $G_i$ 收集起来。

**与光伏预测的联系**：数值天气预报（NWP）的隐式时间积分需要解大型稀疏线性方程组，高斯消元（LU 分解）是核心求解器。Warner NWP Ch3 的有限差分方程最终都归结为 $A\mathbf{x} = \mathbf{b}$。

## Lec 3-4：逆矩阵与线性无关

### Gauss-Jordan 消元求逆

将 $(A|I)$ 通过行变换化为 $(I|A^{-1})$。关键洞察：

> **不是所有方阵都可逆。** 可逆 $\Leftrightarrow$ 列线性无关 $\Leftrightarrow$ 零空间只有零向量。

### 列空间 $C(A)$

$m \times n$ 矩阵 $A$ 定义了一个映射 $\mathbb{R}^n \to \mathbb{R}^m$。**列空间** $C(A) = \{A\mathbf{x} : \mathbf{x} \in \mathbb{R}^n\}$ 就是这个映射的像。

$$A\mathbf{x} = \mathbf{b} \text{ 有解} \Leftrightarrow \mathbf{b} \in C(A)$$

### 零空间 $N(A)$

$$N(A) = \{\mathbf{x} : A\mathbf{x} = \mathbf{0}\}$$

可逆行变换不改变零空间：$N(A) = N(R)$（$R$ 是 RREF）。

**实际意义**：零空间的维度 = 系统的"自由度"。在光伏系统建模中，如果参数矩阵有非平凡零空间，说明模型**过参数化**——有些参数组合无法被数据区分（类似 Box Ch7 中 ARMA(1,1) 当 $\phi \approx \theta$ 时的脊形似然面）。

## Lec 5-6：通解的结构

### 齐次 + 特解 = 通解

对 $A\mathbf{x} = \mathbf{b}$：

1. 如果 $\mathbf{b} \notin C(A)$，无解
2. 如果有解，找一个**特解** $\mathbf{x}_0$（令自由变量 = 0）
3. **通解** = $\mathbf{x}_0 + N(A)$

这和微分方程的通解结构完全一样：特解 + 齐次解。Box 的 ARIMA 预测函数（Ch5）也是同样的结构——最终预测 = 特解（由 MA 部分锚定）+ 齐次解（由 AR 特征根决定的衰减/振荡）。

## Lec 7：基、维度与秩零度定理

### 基（Basis）

向量空间 $V$ 的基 = 一组**线性无关**且**张成** $V$ 的向量。

### 秩零度定理（Rank-Nullity Theorem）

$$n = \dim C(A) + \dim N(A) = \text{rank}(A) + \text{nullity}(A)$$

> **$n$ 列 = 主元列数 + 自由列数**

这是线性代数最基本的"守恒律"——信息不会凭空产生或消失，只是在列空间和零空间之间分配。

**深度联系**：
- **Box Ch3**：ARMA(p,q) 的可逆性/平稳性取决于特征方程的根——本质是 $\phi(B)$、$\theta(B)$ 作为算子的"零空间结构"
- **PCA 降维**：数据矩阵的秩 $r \ll \min(m,n)$ → $n-r$ 维零空间对应冗余特征 → 只需保留 $r$ 个主成分
- **NWP 数据同化**：观测矩阵 $H$ 的列空间 = 可观测状态，零空间 = 不可观测状态

## 模块小结：向量空间的几何直觉

Lecture 1-7 建立了线性代数的核心框架：

| 概念 | 几何意义 | 在光伏预测中的角色 |
|------|----------|-------------------|
| 线性组合 | 向量空间中的"移动方式" | 模型是基函数的线性组合 |
| 列空间 $C(A)$ | 矩阵能"到达"的空间 | 模型能表达的预测范围 |
| 零空间 $N(A)$ | 矩阵"看不到"的空间 | 模型无法区分的参数组合 |
| 秩 $r$ | 有效信息维度 | 数据的内在维度 |
| RREF | 最简化等价形式 | 特征选择后的精简模型 |
